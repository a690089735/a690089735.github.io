“展开选项”对话框

使用“展开选项”对话框上的控件设置“UVW 展开”修改器编辑器的首选项。

  • “编辑 UVW”窗口 “选项”菜单 “首选项”
  • “编辑 UVW”窗口上部工具栏 “选项”按钮

界面

“颜色”组

包含自定义 UVW 晶格的显示的色样。默认颜色在特定贴图中可能会变得难以识别。使用色样选择可以更好的显示指定贴图的颜色。

要改变颜色，请单击色样，然后使用“颜色选择器”来选择新颜色。

线条颜色

指定 UVW 晶格线条的颜色。默认设置为白色。

控制柄颜色

指定于面片控制柄的颜色。默认设置为黄色。

显示共享子对象

启用此选项后，与当前选择共享的未选中子对象以此种颜色高亮显示。大多数情况下，共享子对象为边。共享子对象带有单独顶点时即为顶点。默认设置为启用蓝色。

选择颜色

指定选中的 UVW 子对象的颜色。默认设置是红色。

Gizmo 颜色

指定给自由形式 Gizmo 的颜色。默认设置为桔黄色。

显示接缝

启用此项后，允许对视口中显示的坐标簇的边界指定一个容易区分的颜色。默认设置为启用绿色。

剥颜色

“剥模式”处于活动状态时指定给剥群集的颜色。默认设置是紫色。

背景色

指定于纹理贴图不显示的背景颜色。默认设置为暗灰色。

显示栅格

启用此项后，栅格线条为可见。默认设置为启用深蓝色。

还可以设置栅格大小。

[下拉列表]

允许对选中曲面指定一个填充图案。默认设置为“实体填充”。

“显示首选项”组

包含影响在视窗中显示贴图的控制器。

渲染宽度

指定在视窗中显示的图像宽度分辨率。此项不改变图像大小，仅改变分辨率。

渲染高度

指定高度分辨率。

平铺

纹理图像所重复的次数，在八个方向上向外（四个角和四个侧面）。

如果平铺值为 1，结果是一个 3 x 3 的晶格。如果平铺值为 2，结果是一个 5 x 5 晶格，以此类推。

可以通过“平铺位图”复选框切换平铺图案，如下描述。

平铺亮度

设置平铺位图的亮度。值为 1.0 时，亮度等于原始图像的值；值为 0.5 时亮度为一半；值为 0 时，为黑色。

提示禁用“影响中心平铺”（参见下文），可防止“亮度”设置影响中心平铺，因此能够更容易地确定是否调低了亮度。
使用自定义位图大小

启用该选项后，将位图纹理缩放到“宽度”和“高度”指定的值。可以调整这些设置来相对于纹理坐标对位图纹理进行缩放和再均衡。此缩放不影响材质中的位图，仅影响编辑器中的显示。

提示在处理大纹理时，为快速反馈而减小位图尺寸。在处理非均匀纹理时，在编辑器中将尺寸设置为彼此相邻可以使之易于操作。
显示单位为像素
启用后，以下微调器将显示为像素单位，而不是通用单位：
  • “变换/显示”工具栏上的 U 和 V 微调器。
  • “炸开”卷展栏“焊接”组中的“阈值”微调器。
  • 位于“排列元素”卷展栏和“紧缩 UV”对话框中的“填充”微调器。
  • “法线贴图”和“展平贴图”对话框中的“间距”微调器。
在视口中持续更新

在移动鼠标时影响视口中对 UVW 顶点的调整。默认设置为禁用（直到释放鼠标调整 UVW 顶点的效果才会显示于视口）。

显示图像 Alpha

在编辑器中显示背景图像的 alpha 通道（如果它存在的话）。

显示隐藏边

切换面边的显示。禁用该选项后，仅显示面。启用该选项后，将显示所有的网格几何体。

平铺位图

启用该选项后，可重复编辑器中的位图，从而显示材质中设置的平铺。

可以使用任意部分的平铺图像来设置纹理坐标。此操作在纹理图像段紧凑打包而且网格包含很多不同贴图区域时很有帮助。

影响中心平铺

当启用时，“平铺亮度”设置（参见上文）会均等地影响所有平铺。禁用此项后，中心或者“源点”会始终平铺保持在最高亮度，这样可以轻易的将原始平铺从副本中区别出来。

显示平铺栅格线

启用此选项后，显示平铺栅格线。

过滤器贴图

启用贴图平滑/过滤。

提示在处理低分辨率贴图时，禁用此选项对精确放置 UV 线非常有用。

混合平铺贴图到背景

影响对混合平铺的亮度设置低于 1.0 的颜色。禁用此选项后，平铺混合为黑色。启用此选项后，平铺混合为背景色。

其他“首选项”组

棋盘格平铺

在 UV 编辑器视口中更改棋盘格背景的大小。

栅格大小

设置水平和垂直栅格线条的位置。默认值为 0.1。

将栅格大小设置为 0 会有效的禁用栅格。为最高值 1.0 时，栅格与纹理大小相同。

应用位图比

应用参照位图的大小比率。您可以更改 \\maps\uvwunwrap 中的位图。

“选择首选项”组

软选择边距离

启用“软选择”后，将按照当前选择与受影响顶点之间的指定边数来限制衰减区域。影响范围使用一系列“边距”空间量度而不是用绝对距离。默认值为 16。

单击感应尺寸

设置从子对象中单击选中的距离。默认值为 4。范围从 1 到 10。

选定标记大小

设置编辑器窗口用于表示选定顶点方形框的大小。默认值为 2。范围从 1 到 10。

确定/取消/默认值

在对话框中单击“确定”接受更改，或单击“取消”取消更改。单击“默认值”可将此对话框的所有设置重置为默认值。

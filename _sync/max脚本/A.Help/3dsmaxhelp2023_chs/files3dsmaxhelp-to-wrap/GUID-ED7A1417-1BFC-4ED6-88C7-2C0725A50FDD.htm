图形朝向操作符

“图形朝向”操作符将每个粒子创建为矩形，这些矩形始终朝向某特定对象、摄影机或方向。对于诸如烟雾、火焰、水流、气泡或雪花的效果，对包含适当不透明度和漫反射贴图的材质使用“图形朝向”。

  • 粒子视图 单击事件中的“图形朝向”，或将“图形朝向”操作符添加至粒子系统，并选择。

默认情况下，粒子的顶和底面与水平面平行。“方向”设置可用于更改此默认对齐方式。

注意:要使“图形朝向”操作符能够计算粒子方向，那么它必须知道当前粒子位置。为此，始终将“图形朝向”操作符放置在事件中“位置”操作符的下面。

界面

“注视摄影机/对象”组

使用此控件可以定义粒子将要朝向的摄影机或对象。此对象名为注视对象。从技术上讲，朝向是通过保持每个粒子的局部 Z 轴指向“注视”对象来维持的，或当启用“使用平行方向”时，通过与“粒子源”图标和“注视”对象之间的向量对齐来维持。

名称
在拾取一个“注视”对象后，其名称会显示在此处。
拾取摄影机或对象
单击此按钮，然后选择场景中的摄影机或对象，用作“注视”对象。置于有效对象上时，鼠标光标变为交叉图形。
使用平行方向
当禁用时，所有粒子会继续旋转，以保持朝向“注视”对象的位置。每个粒子的方向不同，这是由于其位置不同于其他粒子。当启用时，所有粒子朝向同一个方向，其方向由粒子源 Gizmo 中心和“注视”对象之间的虚拟线定义。默认设置为禁用状态。

“大小/宽度”组

使用这些设置可以指定设置大小和大小参数的坐标系。此组中的数值设置不能设置动画。

  • 在世界空间中使用世界坐标系按系统单位设置绝对大小。

    单位使用“在世界空间中”选项，按系统单位设置粒子大小。范围从 0 到 1000000000。默认值为 1。

  • 在局部空间中在局部空间中设置相对于现有大小的粒子大小。“粒子流”使用现有图形的尺寸来确定“朝向矩形”的大小。

    继承 %设置正面粒子的大小，以现有大小的百分比表示。范围从 0 到 100。默认值为 100。

  • 在屏幕空间中设置正面粒子的大小，以屏幕宽度的百分比表示。每个粒子的实际大小会按整个动画过程的需要进行更改（这取决于其到摄影机的距离），使得从摄影机的视点来看粒子大小保持恒定。

    仅当“注视”对象为摄影机且“使用平行方向”禁用时此选项可用。

    比例 %设置粒子的大小，以屏幕宽度的百分比表示。默认值为 1。

    变化 %设置粒子大小可以变化的百分比。默认值为 0。

轴位置
指定当维持朝向方向时旋转所围绕的部分粒子。可以从下拉列表中选择的选项为“顶”、“中心”和“底”。默认设置为“中心”。

使用“顶”和“底”，则对应面的中心将用作旋转的粒子中心。

例如，当粒子位于曲面上并且每个粒子都为爆炸时，此选项很有用。在这种情况下，您可能希望整个粒子矩形出现在曲面上，因而会将“轴位置”设置为“底”。

W/H 比率
定义图形矩形的纵横（宽度与高度）比。

将此比率调整为虚拟的粒子贴图的比率。对于常用的 TV 纵横比 4:3，将 W/H 比率设置为 1.33。此参数不能设置动画。范围为 0.001 至 1000。默认值为 1（正方形粒子）。

此值不能设置动画。

“方向”组

使用此下拉列表来选择粒子在轴上旋转的方式，此轴并非由“注视”方向指定。

  • 对齐到地平线（默认设置。）使顶边与地平线（世界坐标 XY 平面）对齐。
  • 对齐速度跟随将每个粒子的右侧面用作主导边，从而使顶和底边与粒子移动的方向对齐。
  • 随机随机定向顶边。
  • 允许自旋若要自旋粒子，请选择此选项，并在相同事件中使用旋转操作符。在旋转操作符中，将“旋转轴”设置为“粒子空间”并使用默认轴的值：X=0，Y=0，Z=1。

“唯一性”组

“唯一性”设置能够更改大小/宽度随机变化。

种子
指定随机化值。
新增特性
使用随机化公式计算新种子。

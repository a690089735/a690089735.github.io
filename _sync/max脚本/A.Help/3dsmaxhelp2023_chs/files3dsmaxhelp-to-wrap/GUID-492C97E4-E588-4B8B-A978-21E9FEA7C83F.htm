RAM 播放器键盘快捷键

各种键盘快捷键适用于 RAM 播放器。

RAM 播放器功能 键盘快捷键
转到开始帧 主栅格
转到结束帧 末端
转到上一帧 左箭头键
转到下一帧 右箭头键
向后播放 上箭头键
向前播放 下箭头键
停止播放 Esc 或 Ctrl+C
切换播放或停止 空格键
切换通道 A 启用/禁用 A
切换通道 B 启用/禁用 B
打开通道 A 中的文件 Ctrl+A
打开通道 B 中的文件 Ctrl+B
在通道 A 中打开上次渲染图像 Alt+A
在通道 B 中打开上次渲染图像 Alt+B
移动动画 使用鼠标右键或使用 Ctrl 键 + 鼠标左键进行拖动。

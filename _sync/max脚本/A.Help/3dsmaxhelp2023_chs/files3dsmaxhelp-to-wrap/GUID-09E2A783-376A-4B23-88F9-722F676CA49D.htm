导入 Alembic 文件

  • “文件”菜单 “导入” “选择要导入的文件”对话框 “文件类型: Alembic (*.ABC)”

导入 Alembic 文件时，会自动导入以下内容的数据：

  • 节点变换
  • 节点可见性
  • 图形对象
    • Nurbs
    • Bezier
    • 直线
  • 网格/多边形
    • 面
    • 位置通道
    • 法线
    • UV 通道
    • 速度
    • 边顶点细分权重
    • OSD 支持
    注意:任何附加网格通道都添加到 3ds Max 贴图通道中。
  • 摄影机
    • 自由度光圈系数
    • 近距/远距：剪切距离
    • 视野
注意:不支持的任何节点都将添加为虚拟对象。

还有几个用于其他数据的选项，如果需要，您可以导入这些数据。

注意:3ds Max 不会导入 Alembic 文件中的粒子系统。

界面

检查 Alembic 文件
打开“检查”对话框，以便在导入文件之前查看文件内容。
提示可使用搜索栏快速过滤并查找内容。

“导入”组

坐标系
在 Y 轴向上或 Z 轴向上坐标系中导入 Alembic 几何体。
导入到场景根
将对象导入到场景的根级别，作为场景的子对象。否则，对象导入为名为 Alembic 的 Alembic 容器对象的子对象。默认设置为禁用。
删除图形后缀
删除“图形”后缀。

“动画”组

匹配时间范围
更改活动时间范围以匹配文件中的动画。默认设置为启用。
注意:如果文件不是动画，则活动时间范围不会更改。
将当前时间设置为开始帧
将场景的当前时间设置为已导入的动画的开始时间。

“导入数据”组

UV
对文件中的几何体应用 UV。 将检查面集和附加通道名称，如果名称以数字结尾，则该数字将用作 UV 通道编号的材质 ID。例如：如果面组名称为 group_7，则在导入时这些面的材质 ID 将为 7。
注意: 如果文件中有多个通过其他通道的 UV 通道，则也会将其导入。
法线
对文件中的几何体应用法线。禁用时，不以面状方式指定面法线。
顶点颜色
对文件中的几何体应用顶点颜色。
附加通道
导入所有其他通道数据。
速度
如果 Alembic 文件具有标准速度，则将每个顶点/粒子速度作为速率通道导入。
注意:每个渲染器必须正确配置为使用此通道才能达到所需效果。
材质 ID
导入存储为 Alembic 面集的材质 ID。
可见性
导入可见性数据。
对象属性
应用几何体的对象属性。
自定义属性
对节点应用自定义属性。
注意:只支持基本类型，例如字符串、整数和浮点数。阵列数据将会忽略。
注意:Alembic 格式规范不包含针对大多数此类数据的任何标准。

本节内容

  • Alembic 对象和 AlembicXform 控制器

    在 3ds Max中，每个从 Alembic ABC 文件导入的对象均显示为 Alembic 对象。已设置动画的 Alembic 对象具有 AlembicXform 控制器。

平移和缩放轨迹窗口

重要信息:在轨迹栏的时间标尺中而不是在轨迹窗口本身中执行这些操作。

要平移轨迹窗口，请执行以下操作：

  • 按住 Ctrl + Alt + 鼠标中键，然后在轨迹栏的时间标尺上拖动。

要缩放轨迹窗口，请执行以下操作：

  • 按住 Ctrl + Alt + 鼠标右键，然后在轨迹栏的时间标尺上拖动。

推拉摄影机和/或目标

当“摄影机”视口处于活动状态时，此弹出按钮上的按钮将代替“缩放”按钮。使用这些按钮可以沿着摄影机的主轴移动摄影机和/或其目标，移向或移离摄影机所指的方向。

  • 激活“摄影机”视口。 视口导航控件 （推拉摄影机）或（推拉目标）或 （推拉摄影机 + 目标）

推拉摄影机

自由摄影机沿着其深度轴，朝着镜头所指的方向移动。与目标摄影机不同，无论您推拉多远，自由摄影机的目标距离仍然保持固定。

注意:当目标摄影机视口处于活动状态时，“推拉摄影机”弹出按钮的三个按钮都可用。当自由摄影机视口处于活动状态时，按钮显示为弹出按钮，但对于此类型的摄影机只有“推拉摄影机”可用。如果激活目标摄影机视口，三个按钮将再次处于可用状态。

过程

要推拉摄影机，请执以下列操作：

  1. 激活“摄影机”视口。
  2. 单击 （推拉摄影机）或相同弹出按钮上其他按钮中的其中一个。
  3. 通过拖动来移动摄影机。
    • 向上拖动可沿着其视线向前移动摄影机。
    • 向下拖动可沿着其视线向后移动摄影机。
  4. 按下 Esc 键或者右键单击可关闭该按钮。

界面

“推拉摄影机”弹出按钮由以下几个按钮组成：

推拉摄影机

只将摄影机移向或移离其目标。如果移过目标，摄影机将翻转 180 度并且移离其目标。

推拉目标

只将目标移向和移离摄影机。在摄影机视口看不到变化，除非您将目标推拉到摄影机的另一侧，摄影机视图将在此翻转。然而，更改目标到摄影机的相对位置将影响其他调整，如环游摄影机，它将目标作为其旋转的轴点。

只有视口的摄影机是目标摄影机时此选项才可用。

推拉摄影机 + 目标

同时将目标和摄影机移向和移离摄影机。

只有视口的摄影机是目标摄影机时此选项才可用。

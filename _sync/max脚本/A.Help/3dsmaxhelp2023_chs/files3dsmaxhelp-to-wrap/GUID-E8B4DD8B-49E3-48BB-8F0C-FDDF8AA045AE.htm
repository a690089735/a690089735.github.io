材质修改器

“材质”修改器使您可以设置对象上材质 ID 的指定，或者简单的对其做出更改。如果材质 ID 设置了动画，那么对新材质 ID 的更改会从一帧到下一帧产生突变。

  • 选择对象。 “修改”面板 “修改器列表” “对象空间修改器” “材质”
  • 默认菜单：选择对象。 “修改器”菜单 “曲面” “材质”
  • Alt 菜单：选择对象。 “修改器”菜单 “UV、贴图和材质” “材质”

使用多维/子对象材质贴图的对象：

“材质 ID 1”用于监视器的外壳

“材质 ID 2”用于屏幕上的图像

提示如果想逐渐的从一种材质到另一种材质进行混合,那么可以试试在混合材质上设置“混合”参数的动画。

将该修改器与多维/子对象材质类型共同使用，为动画中不同的帧上的对象或面指定不同的材质，或者快速更改对象的材质 ID。

面片

该修改器不会将修改器堆栈中该修改器下面的面片对象转化为网格。输入到“材质”修改器的面片对象保留其面片标识。

过程

示例：要更改子对象选中项的材质 ID，请执行以下操作：

  1. 在“顶”视口中，创建一个球体。
  2. 在“材质编辑器”中，创建一个多维/子对象材质。

    将材质 ID 1 和 2 定义为不同的颜色。

  3. 将该多维/子对象材质指定给球体。
  4. 在“修改”面板上，从“修改器列表”中选择“网格选择”。
  5. 在“网格选择参数”卷展栏上，单击“多边形”。
  6. 在“前”视口中，使用区域选择球体的下半部分。

    选定的多边形变成红色。

  7. 当“多边形”仍然是活动子对象层级时（在堆栈显示中，一个正多边形图标出现在“网格选择”的右边），从“修改器”列表中选择“材质”。
  8. 在“材质”修改器“参数”卷展栏上，将“材质 ID”的值设置为 1 和 2 以切换颜色。

    在着色视口中，球体的下半部分更改为选中的材质 ID 的颜色。

界面

材质 ID

设置要被指定的材质 ID ：可以设置该参数的动画。如果在“面”、“多边形”或“元素子对象”层级进行输入选择，则 ID 只应用于选定的面；否则，将应用于整个对象。ID 编号表示在多维/子对象材质中的一个材质。

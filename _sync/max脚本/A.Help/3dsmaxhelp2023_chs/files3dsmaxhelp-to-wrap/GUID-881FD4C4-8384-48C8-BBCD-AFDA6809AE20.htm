编辑足迹位置

在创建足迹之后，有几种在场景中编辑足迹位置的方法。

  • 可以使用标准的 3ds Max 控件来选择、移动、旋转或删除足迹。
    注意:3ds Max 的缩放变换并不会改变足迹图标本身的大小，图标的大小是由 Biped 的脚部大小设置的。Biped 脚部大小最初根据 Biped 的高度来设置，接下来的大小变化是在“体形模式”中对脚部进行缩放引起的。请参见设置 Biped 的姿态。
  • 可以使用“足迹操作”卷展栏中的“弯曲”和“缩放”控件来编辑足迹。“弯曲”值使选定足迹外形曲线发生变化，而“缩放”则更改选定足迹间的距离。

选择足迹

要改变足迹的位置，必须先选中要更改的足迹。

启用 （足迹模式）时，可以按照选择 3ds Max 对象的方法选择足迹：

  • 按下 Ctrl 键单击足迹图标。
  • 单击并拖动选取边界框以选中要选择的足迹。
  • 按下 Alt 键并单击足迹可以取消选择足迹。

也可以在“轨迹视图”的“多谱表单”模式中选择足迹。请参阅编辑足迹计时。

Biped 关键点调整

当移动、旋转、删除、弯曲或缩放活动的足迹时，character studio 会自动调整 Biped 关键点以适应平衡和运动中所作的变化。这种变化称为调整。关于如何发生调整的信息，请参见编辑活动足迹。

变形扭曲

使用变形扭曲可以沿着对象的长度创建盘旋或扭曲的对象。扭曲将沿着路径指定旋转量。

  • 选择一个放样对象。 “修改”面板 “变形”卷展栏 “扭曲”

“扭曲变形曲线”对话框

使用扭曲来变形放样路线

以下是“扭曲”变形曲线的属性：

  • 单条红色曲线将沿着路径确定图形旋转。
  • 默认曲线值为旋转 0 度。
  • 当从路径开始处查看时，正值将生成逆时针旋转。
  • 负值将生成顺时针旋转。
  • 扭曲变形和倾斜将沿着路径生成旋转。在应用倾斜角之后将扭曲旋转应用到图形。可以使用“扭曲”变形来扩大或减少倾斜量。

有关对话框控件的特定信息，请参见“变形”对话框。

过程

要使用“扭曲”变形，请执行以下操作：

  1. 选择一个放样对象。
  2. 右键单击修改器堆栈显示中的放样。
  3. 单击“变形”卷展栏上的“扭曲”。
  4. 编辑单条变形曲线沿着路径指定旋转。

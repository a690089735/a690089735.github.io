“mParticles 粘合”测试：“刚性绑定/简化绑定”卷展栏

使用类型设置为“刚性”或“简化”的“mParticles 粘合”测试时，您可以指定绑定定位的放置以及绑定的可打破性（相应卷展栏为“刚性绑定”或“简化绑定”）。

  • “粒子视图” 单击或添加“mP 粘合”测试。 将绑定类型设置为“刚性”或“简化”。

两种类型的设置大致相同，但就两种方式（可以设置不同定位放置，并在它们之间切换）而言，分隔卷展栏的可用性取决于绑定类型。

界面

绑定锚定放置
选择软件确定绑定定位放置的方式：
  • 中心平均值将绑定的主定位放置在粒子轴的中心点，并将主定位轴定位为沿着粒子之间的向量。
  • 最近点平均值确定每对粒子碰撞图形的最近点，然后将主定位放置在这些点的中间。主定位轴在最近点处垂直于图形曲面。在这种情况下，如果主定位轴方向没有沿着粒子之间的向量，绑定会沿着接触曲面变得更脆弱且粘性减弱。例如，若您有要销毁的砖墙，请使用该设置。
  • 质量比例与粒子质量呈反向立方比例放置绑定的主定位点。此操作适用于类似以下情况：假设球形粒子的大小不同，密度相同。假设您为彼此接触的粒子创建绑定。在这种情况下，最好在接触点处创建主定位点（作为可能的断点）。在这种情况下，可以将接触点计算为粒子位置的加权平均值，其中权重系数为粒子质量的立方根。若要绑定大小不同、略呈球形的粒子时，请使用“质量比例”选项。

    有关显示“绑定定位放置”选项效果的视频，请参见绑定定位放置。

解算器因子
仅适用于刚性绑定。使用该设置可调整刚性绑定的刚性或柔性。该值的范围为 0 到 2，默认值为 1。增加该值会使绑定更具刚性（柔性降低），而潜在副作用是解决方案不稳定。为获得最佳效果，请将值保持在 2 以下，否则解决方案会有不稳定的风险。合理的值约为 1.1 或 1.2 或 1.4。

有关显示“解算器因子”值效果的视频，请参见解算器因子。

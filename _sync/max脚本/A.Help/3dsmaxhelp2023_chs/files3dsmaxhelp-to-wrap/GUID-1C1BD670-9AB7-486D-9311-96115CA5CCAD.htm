安装网络服务

本主题介绍如何安装作为 Windows 中网络服务的网络管理器和网络服务器。

通过执行以下操作来启用：

  1. 运行管理器和服务器的应用程序版本以正确配置应用程序并使您的渲染网络可操作。
  2. 生产阶段使用网络，以便您能确保该程序可靠运行。

    除非这些条件符合要求，否则不要执行操作。

安装作为 Windows 服务的网络管理器和渲染服务器支持渲染背景，而且非常方便，但这也意味着问题出现时，渲染服务器的屏幕上会显示很少的问题信息。这也是执行此操作之前需要平稳运行网络的原因。运行作为服务的管理器和服务器将不会更改显示在“队列监视器”中的信息。

在执行下列步骤时，管理器和服务器服务安装并注册在 Windows 下。此安装将替换应用程序模式（每当需要使用它们时即可手动运行该管理和服务器）的使用。每当启动计算机时可自动启动该服务，但也可设置为手动启动。应用程序模式的 \Network 子目录和初始化和 LOG 文件保持在原位，但该服务在 Windows 下运行而不是以独立的流程运行。

可以使用 -i 开关（作为服务安装）来直接运行命令提示窗口或“运行”对话框的作为服务的管理器和服务器。然后转至服务，并启动管理器和/或服务器或重新启动。

要移除已作为服务安装的管理器和服务器，您必须使用 -r 开关（移除服务）从命令提示窗口或“运行”对话框直接运行 3ds Max。

过程

要设置作为服务的管理器，请执行以下操作：

  1. 转至将要在其中安装作为服务的管理器的机器。
  2. 打开命令提示窗口，并将目录更改为 Backburner 根目录：例如，\Program Files\Autodesk\Backburner\。
  3. 输入 managersvc -i

    将显示下列消息：

    Backburner Manager ... Service Installed

  4. 转至 Windows“控制面板”“管理工具”“服务”，右键单击 Backburner Manager，并选择“启动”。

    如果通过右键单击菜单选择属性，您也可以设置用户、密码和其他参数。

要设置作为服务的服务器，请执行以下操作：

  1. 转至将要在其中安装作为服务的服务器的机器。
  2. 打开命令提示窗口，然后将目录更改为 Backburner 根目录。
  3. 输入 serversvc -i

    将显示下列消息：

    Backburner Server ... Service Installed

  4. 转至 Windows“控制面板”“管理工具”“服务”，右键单击 Backburner Server，并选择“启动”。

    如果通过右键单击菜单选择属性，您也可以设置用户、密码和其他参数。

  5. 在每台机器上重复这些步骤，在其中您可以设置作为服务的服务器。

群组快捷键

本主题概述了可用于群组动画的键盘快捷键。

要使用 character studio 键盘快捷键，请启用“键盘快捷键覆盖切换”。

当“运动”面板和“键盘快捷键覆盖”按钮处于活动状态时，所有 character studio 键盘快捷键也处于活动状态。

操作 快捷键 描述
求解 S 求解群组模拟。
分步求解   分步求解群组模拟。

  • 键盘快捷键
  • “键盘”面板
  • “自定义用户界面”对话框

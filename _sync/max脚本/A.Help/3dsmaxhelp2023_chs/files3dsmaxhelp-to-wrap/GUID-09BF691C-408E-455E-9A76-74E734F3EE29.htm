还原为启动布局

使用“还原为启动布局”选项可以自动加载 _startup.ui 文件，它可以将用户界面返回到启动设置。启动 3ds Max 时，将自动创建此临时系统文件。使用此选项可以将 UI 返回到启动设置。

  • 默认菜单：“自定义”菜单 “还原为启动 UI 布局”
  • Alt 菜单：“自定义”菜单 “自定义” “还原为启动 UI 布局”

有关如何在启动时加载不同用户界面方案的信息，请参见加载自定义用户界面方案。

当您退出 3ds Max 时，如果“首选项”对话框 “常规”面板 “退出时保存 UI 配置”开关处于启用状态，则会覆盖当前 UI 文件。

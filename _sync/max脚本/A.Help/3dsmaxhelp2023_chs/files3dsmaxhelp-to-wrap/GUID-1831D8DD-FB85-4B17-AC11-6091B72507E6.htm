关于 OpenSubdiv

OpenSubdiv 功能集有助于建模人员使用控制细分来创建基于多边形对象的具有平滑和折缝曲面的图形。您可以在多边形网格的顶点和边上指定曲面的总曲率和基准可变折缝。

OpenSubdiv 使用 Pixar 中的高级软件技术，用于在大规模并行 CPU 和 GPU 建筑上建模和制作细分曲面动画。有关技术背景信息，请参见 http://graphics.pixar.com/opensubdiv/

3ds Max 中的 OpenSubdiv 实施包含三个修改器：“OpenSubdiv”、“Crease”和“CreaseSet”。OpenSubdiv 执行细分、平滑和折缝，Crease 使您可在程序上选择对象的边和顶点，并将折缝值应用于它们；CreaseSet 提供增强功能，用于指定和管理顶点和边的集，以及它们的折缝设置。Crease 和 CreaseSet 还提供对“折缝资源管理器”的直接访问，该资源管理器的界面类似于电子表格，用于管理折缝的子对象。

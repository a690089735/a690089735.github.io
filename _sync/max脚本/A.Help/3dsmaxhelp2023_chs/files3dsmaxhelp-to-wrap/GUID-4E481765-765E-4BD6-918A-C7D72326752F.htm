Bezier 控制器

多功能 Bezier 控制器使用可调整样条线曲线在关键点间插值。

  • 指定控制器 Bezier

当想在关键点间完全可调整插补时，可以使用 Bezier 控制器。Bezier 是唯一一个支持下列操作的控制器：

  • 拖动切线控制柄
  • 为从一个关键点到下一个关键点的突变设置阶跃切线
  • 恒定速度控制

过程

要设定变换关键点的切线类型，请执行以下操作：

  1. 选择一个包含变换动画关键点的对象。
  2. 转到 “运动”面板，然后在“PRS 参数”展卷栏的底部单击变换关键点类型的按钮进行更改，例如“位置”。
  3. 在“关键点信息(基本)”卷展栏上，单击向左键和向右键按钮选择一个关键点。
  4. 从“内”或“外”切线弹出按钮中选择切线类型。

要规格化关键点的时间，请执行以下操作：

  1. 选择一个对象，然后选择一个要规格化的关键点。

    可以在“关键点信息（基本）”卷展栏中使用箭头来移动关键帧。

  2. 在 “运动”面板 “关键点信息(高级)”卷展栏上，单击“规格化时间”。

    该关键点在时间中移动，以平均通过关键点的速度。

界面

“运动”面板 >“参数”>“关键点信息（基本）”卷展栏

您可以通过在“轨迹视图”中右键单击关键点，或在轨迹栏上右键单击关键点，然后选择要编辑的关键点，打开具有相同参数的对话框。

[关键点编号]
当前关键点编号。要转到相邻的关键点，请单击向左键或向右键按钮。
时间
动画中出现关键点的时间。
L [时间锁定]
在“轨迹视图”编辑模式下水平拖动关键点的控件。启用后，仅可以垂直拖动关键点。
X/Y/Z
调整关键点的值；即“曲线编辑器”中的垂直位置。
注意:部分控制器仅适用于单个轴。
Bezier 缩放控制器（锁定 X）

使 X 值影响比例的全部三个轴。Y 和 Z 值被忽略，并且不显示它们的功能曲线。

当 X 锁定时，Y 和 Z 值不受 X 值改变的影响。当全部三个轴值相同时，如果锁定 X，并改变 X 值，然后解除对 X 的锁定，那么 Y 和 Z 值保持原来的值不变，而 X 保留其新值。

内/外 [关键点切线]
这两个弹出按钮可以设定关键点内切线和外切线的插值属性。

请参见切线类型。

切线复制

在当前关键点的切线间或前一个和下一个关键点的切线间复制切线类型。“关键点切线”弹出按钮两侧箭头按钮的用法如下：

  • “内”切线的左箭头复制到上一个关键点的“外”切线。
  • “内”切线的右箭头复制到当前关键点的“外”切线。
  • “外”切线的左箭头复制到当前关键点的“内”切线。
  • “外”切线的右箭头复制到下一个关键点的“内”切线。

“运动”面板 >“参数”>“关键点信息（高级）”卷展栏

卷展栏中的控件在以下三个方面影响速度：

  • 使用输入/输出值在关键点上控制绝对速度
  • 使用“规格化时间”在指定的时间量上平均速度
  • 使用“恒定速度”从一个组件关键点到下一个关键点间强制恒定速度
输入/输出

当参数到达关键点时，“输入”字段显示更改速率。当参数离开关键点时，“输出”字段显示更改速率。

只在关键点使用“样条线”切线类型时这些字段才被激活。

这些字段中的数字是以每“刻度”参数单位表示的更改速率。通过更改 X、Y 和 Z 的两个值，可以更改切线控制柄的长度和角度。

锁定按钮

启用该选项后，更改一条“样条线”切线会同时更改另一条，但是数量相反。例如，如果启用“锁定”按钮而“输入”值为 0.85，那么“输出”值为 -0.85。

规格化时间

平均时间中的关键点位置，并将它们应用于选定关键点的任何连续块。如果想平滑运动并且使对象加速减速，那么该选项很有用。

恒定速度

通过使对象以恒定的速度穿过曲线线段的方式，在一个关键点和下一个关键点间插补值。

自由控制柄

用于自动更新切线控制柄的长度。禁用时，切线长度是其相邻关键点相距固定百分比。在移动关键点时，控制柄会进行调整，以保持与相邻关键点的距离为相同百分比。启用时，控制柄的长度基于时间长度。

创建位置锁定关键点/创建旋转锁定关键点

将当前关键点的传入控制柄和上一关键点的传出控制柄都设置为线性插值。这样可以防止样条线插值过冲。

要使用这两个功能，请首先在“自定义”“自定义用户界面”中创建键盘快捷键。在操作列表中查找“创建位置锁定关键点”和“创建旋转锁定关键点”并指定热键。也可以将这些命令添加到工具栏或四元菜单（“四元菜单”面板）中。

“顶点链接指定”卷展栏

在“顶点链接指定”卷展栏中，您可以选择创建新的封套，并拾取链接之间混合的类型。

  • 选择已应用 Physique 修改器的对象。 “修改”面板 “Physique”卷展栏 单击 （附加到节点）。 在视口中选择某个根对象。 “Physique 初始化”对话框 “顶点 - 链接指定”卷展栏
  • 选择已应用 Physique 修改器的对象。 “修改”面板 “Physique”卷展栏 单击 （重新初始化）。 “Physique 初始化”对话框 “顶点 - 链接指定”卷展栏

界面

顶点 -“链接指定”组

  • 可变形（默认设置。）为链接创建可变形的封套。

    可变形封套根据 Physique 通过链接创建的变形样条线确定顶点链接指定。

  • 刚性

    为链接创建刚性封套。默认设置为不选择。

    刚性封套根据层次中的线性链接确定顶点链接指定。

在链接之间混合
该下拉列表列出选择链接最初混合的方式。默认值为 N 个链接。
提示通常应该保留默认设置 N 个链接。
  • N 个链接 顶点受所有重叠封套的影响。
  • 无混合 顶点仅受一个 (1) 链接的影响。从而使得应用在 character studio 1 中带有 Physique 修改器的网格能够与 character studio 3 一起使用。如果您正在为不支持混合的游戏引擎开发角色，或如果您要严格使用 character studio 1 样式的顶点链接指定，请选择“无混合”。
  • 2、3 或 4 链接 顶点受不只一个链接的影响，但是链接数量会限制在 2、3 或 4，具体情况取决于您的选择。如果正在开发具有有限混合支持的游戏引擎角色，则选择其中一个选项。

“径向衰减封套”组

创建封套

当启用时，将为链接创建封套。当关闭时，将不创建封套。默认设置为启用。

注意:无封套时，必须对顶点进行手动加权。如果要在模型没有封套时添加封套，请重新初始化 Physique。
  • 对象边界框封套尺寸基于层级中对象大小（如 Biped 四肢或骨骼）。
  • 链接长度每个封套有一个“径向缩放”，其长度为链接长度的三分之一。
重叠

设定封套重叠的范围。默认值为 0.1。

平滑

通过缩放外部边界，设置封套内外部边界之间的距离。范围从 0.0 至 10.0。默认值为 0.75。

衰减

设置封套内外部边界之间的衰减比率。衰减比率是一个 Bezier 函数。范围从 0 到 10。默认值为 0.5。

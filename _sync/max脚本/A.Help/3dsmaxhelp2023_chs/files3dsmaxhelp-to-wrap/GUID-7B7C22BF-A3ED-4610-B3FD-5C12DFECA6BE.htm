文件属性

使用“文件属性”可以输入有关场景的信息，以后可以通过 Windows 来读取这些信息，它们使用诸如 Windows 资源管理器这样的程序来查看场景文件的属性。

  • “文件”菜单 “属性” “文件属性”

此信息还可用于通过“文件查找程序”实用程序来查找场景文件。

“属性”对话框包含三个选项卡：

  • “摘要”面板显示常用的预定义字段。
  • “内容”面板显示保存场景之后可用的场景信息。
  • “自定义”面板用于创建新字段，文件查找器工具也可以使用这些字段。

“摘要”面板

摘要提供了一些字段，用于输入与所保存场景有关的信息。“摘要”面板提供了很多常用字段。如果您不想让某个字段显示在“摘要”选项卡中，则可以使用“自定义”选项卡为信息定义新的字段。

标题

在此输入场景或动画的标题。

主题

在此输入场景或动画的主题。

作者

在此输入场景或动画的作者。

经理

在此输入场景或动画的管理者。

公司

在此输入公司的名称。

类别

在此输入场景或动画的类别。

关键字

在此输入标识场景的关键字。

注释

在此输入个人注释。添加到“摘要信息”对话框上“描述”字段中的信息将显示在“注释”字段中，反之亦然。

“内容”面板

保存场景之后，将在“内容”选项卡中生成并显示场景中包含的所有内容的摘要列表。如果未保存场景，“内容”选项卡将保留为空白。

“自定义”面板

“自定义”面板用于创建自定义字段。如果有更多信息需要输入到场景的属性中，但是在“摘要”面板中没有任何字段或空间，则“自定义”非常有用。

名称

输入自定义字段的名称。您可以在“名称”字段中手动输入任何需要的内容。下拉列表提供了常用的字段类型。

类型

指定要输入到“值”字段中的信息类型：从文本、日期、数字、是或否中选择。指定要输入到“值”字段中的信息类型。

值

在此输入数据。

添加

将输入到“名称”、“类型”和“值”字段中的“自定义”属性添加到“属性”列表。

删除

将所选自定义属性从“属性”列表中删除。

属性

显示输入到场景的所有自定义属性。

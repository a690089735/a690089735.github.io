导入运动捕捉数据

在 Character Studio 中，可以同时导入旋转类型和位置类型的运动捕捉文件。

  • BioVision (BYH) 文件包含肢体和关节旋转数据。
  • character studio 标记文件 (CSM) 包含由运动捕捉装置生成的原始标记位置数据：在运动捕捉执行期间，将标记附加到行为对象身上。

    标志文件需要一些特别的校准。如果需要，首先要排列 Biped 大小，以适合标志，然后确定 Biped 肢体的方向以便与标志对齐。应该用每个帧上的关键点（没有步进提取）加载标志文件；这是校准所需的，并且可以启用校准控件。校准控件位于运动捕获展卷栏的第二排按钮上。

    在 character studio 中，CSM 标记文件格式支持单手或双手中的道具骨骼。请参见属性骨骼。

关键点减少、轨迹选择、足迹提取和剪辑循环是使用运动捕获导入过滤器时可用的一些选项。关键点减少易于对 Biped 进行操作和个性化导入的运动捕获数据。从运动捕获数据中提取足迹可以防止不恰当的滑动，这是动态捕获数据中的一个常见问题。

通常，您不会使用整个运动捕获剪辑，除非在 studio 中捕获自己产品的运动。您应该熟悉文件中的身体部位运动，然后使用运动流模式剪切各部分文件，以便创建动画。例如，在一个剪辑中获得伸展运动，并且将其与另一剪辑中的行走运动结合起来。然后您可以使用“Biped”卷展栏上的“保存分段”命令将编辑的脚本保存为 BIP 文件。加载此 BIP 文件以进行标准的运动编辑。这为您提供了一个良好的开端，您按照需求编辑各种效果。

所有运动捕获控件都位于运动捕获展卷栏上。

配置 TCP/IP

在上一步设置 TCP/IP中，有计算机名称和 IP 地址列表。有了这张列表，转到每个计算机并执行以下步骤。

在开放的网络中（如很多大规模公司中），不会 以任何方式更改 IP 地址、计算机名、工作组名称和域名。在这种情况下，为了避免潜在的灾难性后果，应该始终与系统管理员进行合作，以对 IP 地址进行更改。

下一步

设置渲染软件

过程

以下步骤有些用于 Windows XP 专业版；其他的用于 Windows 2000。在步骤标题末端找到相应的操作系统，并遵循下面的说明。

注意:在以下步骤的编写过程中，Windows XP 设为“经典 Windows”的显示界面。

要指定计算机名、工作组或域 (Windows 2000)，请执行以下操作：

  1. 从 Windows 任务栏上，打开“开始”菜单 “设置”“控制面板”“系统”。

    通过此操作显示“系统属性”对话框。

  2. 单击“网络标识”选项卡，然后单击“属性”按钮显示“标识更改”对话框。

    如果已经为计算机指定一个名称，该名称会出现在“网络标识”选项卡的“计算机名”字段中。根据列表检查此名称。

  3. 要更改此名称，在“计算机名”字段中输入新的名称即可。
    • 有效的计算机/主机名称的首字符不能为数字。
    • 不要在计算机/主机名称中加下划线或空格。
  4. 在“组成员”中，输入工作组或域名，这取决于如何设置您的网络。
    工作组

    组中的计算机组成一个单元，但是不属于域。如果为进行“网络渲染”设置简单网络，则可以使用“工作组”选项。

    域

    组织服务器的更复杂方式，在其中可共享安全策略和用户帐号数据库。域要求设置“主域”控制器。只有当网络管理员设置域并正常工作时，才可以选择“域”选项。

要指定计算机名、工作组或域 (Windows XP)，请执行以下操作：

  1. 从 Windows 任务栏上，打开“开始”菜单 “设置”“控制面板”“系统”。

    通过此操作显示“系统属性”对话框。

  2. 单击“计算机名”选项卡，然后单击“更改”按钮显示“计算机名更改”对话框。

    如果已经为计算机指定一个名称，该名称会出现在“计算机名称”选项卡的“计算机名”字段中。根据列表检查此名称。

  3. 要更改此名称，在“计算机名”字段中输入新的名称即可。
    • 有效的计算机/主机名称的首字符不能为数字。
    • 不要在计算机/主机名称中加下划线或空格。
  4. 在“组成员”中，输入工作组或域名，这取决于如何设置您的网络。
    工作组

    组中的计算机组成一个单元，但是不属于域。如果为进行“网络渲染”设置简单网络，则可以使用“工作组”选项。

要添加 TCP/IP 协议 (Windows 2000)，请执行以下操作：

  1. 从 Windows 任务栏上，打开“开始”菜单 “设置”“网络和拨号网络连接”“局域网连接”。

    通过此操作将打开“局域网连接状态”对话框。

  2. 单击“属性”按钮。

    通过此操作将打开“局域网连接属性”对话框。

  3. 查找“Internet 协议 (TCP/IP)”列表项。

    如果找到此列表，说明 TCP/IP 已安装到本机。转到本主题的以下步骤之一：

    • 以 DHCP 模式配置 TCP/IP
    • 用固定的 IP 地址配置 TCP/IP

      如果没有安装 TCP/IP，确保没有高亮列表中的任何一个条目（在列表中的空白区单击），然后接着完成以下步骤。

  4. 单击“安装”按钮。

    通过此操作显示将打开“选择网络组件类型”对话框。

  5. 在此列表中，单击协议，然后单击“添加”按钮。
  6. 在“选择网络协议”对话框中，选择“Internet 协议 (TCP/IP)”，然后单击“确定”。

    出现一条消息：“您要使用 DHCP 吗?”。向系统管理员咨询网络是否 DHCP 兼容，如果兼容，单击“是”。如果不确定，单击“否”并接着用固定 IP 地址设置工作站。

    TCP/IP 便添加到已安装的协议列表中。

  7. 单击“关闭”按钮。

要添加 TCP/IP 协议 (Windows XP)，请执行以下操作：

  1. 从 Windows 任务栏上，打开“开始”菜单 “设置”“网络连接”。

    通过此操作将打开“网络连接”对话框。

  2. 右键单击“局域网连接”并单击“属性”按钮。

    通过此操作将打开“局域网连接属性”对话框。

  4. 单击“安装”按钮。

    通过此操作显示将打开“选择网络组件类型”对话框。

  5. 在此列表中，单击协议，然后单击“添加”按钮。
  6. 在“选择网络协议”对话框中，选择“Internet 协议 (TCP/IP)”，然后单击“确定”。

以 DHCP 模式配置 TCP/IP (Windows 2000)：

  1. 从 Windows 任务栏上，打开“开始”菜单 “设置”“网络和拨号网络连接”“局域网连接”。

    通过此操作将打开“局域网连接状态”对话框。

  2. 单击“属性”按钮。

    通过此操作将打开“局域网连接属性”对话框。将会看到网络适配器卡在对话框顶部的“连接时使用”字段中列出。如果没有列出网络适配器，请参照 Windows 2000 帮助文档安装适配器。

  3. 在列表中，高亮显示 Internet 协议 (TCP/IP)，单击“属性”。

    将出现“Internet 协议 (TCP/IP) 属性”对话框。

  4. 确保选择“自动获取 IP 地址”选项。
  5. 单击“确定”依次关闭每个对话框。

    Windows 2000 完成配置连接。

  6. 重新启动计算机以完成配置。
  7. 在网络中的每台计算机上重复本页上的所有步骤。

  3. 在列表中，高亮显示 Internet 协议 (TCP/IP)，单击“属性”。

用固定的 IP 地址配置 TCP/IP (Windows 2000)：

  1. 从 Windows 任务栏上，打开“开始”菜单 “设置”“网络和拨号网络连接”“局域网连接”。

    通过此操作将打开“局域网连接状态”对话框。

  2. 单击“属性”按钮。

    通过此操作将打开“局域网连接属性”对话框。将会看到网络适配器卡在对话框顶部的“连接时使用”字段中列出。如果没有列出网络适配器，请参照 Windows 2000 帮助文档安装适配器。

  3. 在列表中，高亮显示 Internet 协议 (TCP/IP)，单击“属性”。

    将出现“Internet 协议 (TCP/IP) 属性”对话框。

  4. 选择“使用以下 IP 地址”。
  5. 在“IP 地址”框中，输入该计算机的地址。

    检查列表确保输入的地址正确无误。

  6. 在“子网掩码”框中，输入以下数字（这些数字对于所有计算机都是相同的）：

    255.255.255.0

    如果在一个大规模公司网络中，子网掩码可能会有所不同。在这种情况下，使用网络管理员指定的掩码。同样，在设置服务器连接到管理器时，以及在“网络作业分配”对话框中进行设置时，请使用与此处相匹配的子网掩码，以便“自动搜索”能顺利工作。

  7. 单击“确定”依次关闭每个对话框。

    Windows 2000 完成配置连接。

  8. 重新启动计算机以完成配置。
  9. 在网络中的每台计算机上重复本页上的所有步骤。

    注意：每台计算机必须有唯一的 IP 地址，否则会出现冲突。

应用 Max IK

借助“应用 Max IK”工具，您可以使用 3ds Max IK 系统设置 CATRig 动画。可以使用样条线控制很长的尾部，或使用数字 HI/IK 为角色的手指和脚趾提供 IK 控件。

  • 默认菜单：“动画”菜单 “CAT” “应用 Max IK”
  • Alt 菜单：“动画”菜单 “CAT 角色” “应用 Max IK”

3ds Max IK 系统每次只能对一个层产生影响。这样，您可以在动画中的不同时间对同一角色使用 IK 和 FK。或者，也可以将 3ds Max IK 系统指定到相对位于堆栈底部的某个层，以控制角色的所有层。

使用样条线 IK

指定“样条线 IK”时，工具需要使用样条线控制 IK 系统。该工具可以根据您选择的 CATBone 层次创建一个新样条线。

提示请始终确保先弯曲要在其中指定样条线 IK 系统的层次，再指定 IK 系统。向希望看到最大范围运动的方向弯曲层次。例如，在约束一只蜥蜴的尾部时，可以向其躯干周围弯曲尾部以帮助配置 IK 系统，从而获得最佳运动。如果在指定 IK 系统之前尾部是笔直的，那么稍后实现尾部动画时，尾部的骨骼可能会发生翻转。

单击“是”，系统将使用尾部形状作为导向创建一个样条线。另外，您也可以选择使用现有样条线控制角色。

系统将使用“应用 IK”对话框中指定的结数创建样条线。此时便可以实现点辅助对象的动画以驱动尾部动画了。

将 HI/IK 系统应用于蜥蜴的脚趾。

界面

拾取层次

选择要对其应用 IK 系统的骨骼。可以选择手臂、腿部、手指、脚趾或尾部。

拾取 IK 解算器

从下拉列表中选择要指定的 IK 系统。

有关解算器选项的详细信息，请参见 3ds Max 主帮助的“IK 解算器”部分。

拾取层

使用现有层容纳 IK 系统，或由工具创建一个新层。

结数

为样条线 IK 系统创建新样条线时，需要定义样条线的结数。

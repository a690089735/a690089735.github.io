“可编辑实体对象”卷展栏

导入一个 SAT 文件或将 3ds Max 中的一个对象转换为实体对象格式然后选择它时，“修改”面板上的第一个卷展栏为“可编辑实体对象”。利用这些控件，您可以在各种子对象级别上执行可用操作，包括翻转面、焊接和炸开（非焊接）边及更多操作。

  • 选择实体对象。 “修改”面板 “可编辑实体对象”卷展栏

实体对象没有子对象历史记录，它们的编辑方式与可编辑网格的编辑方式基本相同。导入的 SAT 文件产生实体对象。在将 3ds Max 对象或其他实体对象转换为实体对象后，该对象的构造历史记录将会丢失。

[子对象层级]
用于访问边、面和元素子对象级别的实体对象的按钮。只有修改器堆栈中才可使用操作对象级别。
[# 已选定的子对象]
子对象按钮之后的该只读字段显示了选定子对象的数目和类型。
面翻转
打开具有以下选项的“面翻转/可见性”卷展栏，这些选项可用于选择重复面、翻转面及设置面可见性。
删除选定项
删除选定的面或元素。您也可以通过按 Delete 键来删除选定项。
焊接选定项
通过将重合边或几乎重合的边合并为一个边来连接选定面或边。
焊接全部
将所有重合边合并在一起。
炸开选定项
将选定面分为可以分别操纵的单独元素。仅可在面和元素子对象级别上使用，至少选定一个子对象。
炸开全部
将所有面分为可以分别操纵的单独元素。
撤消修剪
移除对曲线修剪的面所做的修剪。
清理边
移除面内部的边且不对面进行修剪。
焊接
打开带有高级选项的“焊接”卷展栏。

[信息显示]

卷展栏的底部显示了与选定对象有关的信息。连接面或进行焊接时，使用这些值确定操作的结果。

# 条边
对象中的边数。
# 个顶点
对象中的顶点数。
# 条开放边
对象中的开放边数。

开放边即仅连接一个面的未缝合边。这是标识闭合实体的一个重要的指示器。如果 Open E 显示值大于 0，则体积不是闭合的。另请参见以下的“体积”显示。

注意:有时对象为曲面模型，因而具有开放边。
# 个面
对象中的面数。
# 个元素
对象中的元素数。
# 个体积
对象中闭合体积的数目。如果对象为有效的闭合实体，则它至少有一个体积。

本节内容

  • “焊接”卷展栏

    利用“正在焊接”卷展栏可以指定如何将边缝合在一起。在此过程中焊接是累积进行的，焊接之后将保留结果并在调整参数后再次使用。通常从“公差比例”值 1.0（默认值）开始并增加该值。

  • “面翻转/可见性”卷展栏

    使用这些控件可以控制实体对象中面的朝向。

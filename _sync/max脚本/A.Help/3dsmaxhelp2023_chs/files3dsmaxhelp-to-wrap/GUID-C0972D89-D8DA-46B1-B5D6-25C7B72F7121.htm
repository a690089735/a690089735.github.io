光域网

光域网是光源的光强度分布的三维表示。此平行光分布信息以 IES 格式（使用 IES LM-63-1991 标准文件格式）存储在光度学数据文件中，而对于光度学数据则采用 LTLI 或 CIBSE 格式。可以加载各个制造商所提供的光度学数据文件，将其作为 Web 参数。在视口中，灯光对象会更改为所选光度学 Web 的图形。

为了描述一个光源发射的灯光的方向分布，3ds Max 通过在光度学中心放置一个点光源来接近该光源。使用此近似，将仅分布描述为发出方向的功能。提供用于水平和垂直角度预设的光源的发光强度，并且 3ds Max 可按插值沿任意方向计算发光强度。

Web 图

通常，光度学数据使用图表进行描述。用于显示轴旋转的图标称为测角。

web 分布的测角图表

这种类型的图表可直观地表示灯源的发光强度如何随着竖直角度而发生变化。但是，水平角度是固定不变的，除非该分布关于轴对称， 要描述完整的分布可能需要多个测角图表。

在 3ds Max 用户界面上，Web 文件将显示为缩略图。

光度学 Web 文件的缩略图

鲜红的轮廓表示光束。在某些 Web 中，深红色的轮廓表示（不太明亮的）区域。

光域网

光域网是灯光分布的三维表示。它将测角图表延伸至三维，以便同时检查垂直和水平角度上的发光强度的依赖性。光域网的中心表示灯光对象的中心。

任何给出方向上的发光强度与 web 和光度学中心之间的距离成比例，在指定的方向上沿着与中心保持直线进行测量。

示例 1：统一球形分布

统一球形分布示例

以原点为中心的球体表示统一球形分布，也称为同向分布。图表中的所有点与中心是等距的，因此灯光在所有方向上都可均等地发射。

示例 2：椭圆形分布

椭圆形分布示例

在此示例中，负 Z 轴方向上的点与正 Z 轴方向上相应的点离原点的距离相同，因此相同的灯光量可上下照射。没有一个点有较大的 X 或 Y 分量，正或负，因此有极少的灯光从光源处横向投影。

自适应细分对话框

在使用 HSDS 修改器后用自适应细分来使细分平滑并编辑部分网格。或者，可以使用自适应细分从对象上移除一个细节层次。

  • 选择对象。 “修改”面板 “修改器列表” “对象空间修改器” “HSDS 修改器” “高级选项”卷展栏 “自适应细分”按钮
  • 选择对象。 “修改器”菜单 “细分曲面” “HSDS 修改器” “高级选项”卷展栏 “自适应细分”按钮

过程

要使用自适应细分，请执行以下操作：

  1. 用“HSDS 修改器”编辑一个对象。
  2. 选择“添加细节”或“移除细节”，这取决于要执行的操作。
  3. 将需要的细节量设为其中一种预设或指定为自定义长度和角度设置。
  4. 单击“确定”来执行所作的操作。

    添加或移除细节后，回到“HSDS 修改器”。根据移除或添加细节，细节的最高层次会减小或增加 1。

界面

“细节”组

添加/移除

确定是否单击“确定”按钮来增加或减小细节。

“参数”组

这些设置确定细节添加或移除的程度。长度和角度设置仅在选中“自定义”选项时可编辑。然而，它们默认设置显示为低、中、高选项。

低/中/高/自定义

选择其中一个预设项，或选择自定义来设置所需的长度和角度值。

最大 LOD

指定增加细节时 3ds Max 可以添加的最高细节级别。移除细节时为不可用。

长度

在添加或移除细节后任意边缘的最大允许长度。长度越小，允许的细化量越高。

角度

从一个顶点发出的两条对边之间的最大的允许角度。角度越小，允许的细化量越高。

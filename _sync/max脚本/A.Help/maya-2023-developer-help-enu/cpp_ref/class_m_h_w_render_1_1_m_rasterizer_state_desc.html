Detailed Description

Descriptor for a complete rasterizer state.

This class describes a complete rasterizer state, like fill mode, cull mode, depth bias, multisampling, point size, etc.

Examples:
cgFx/cgfxPassStateSetter.cpp, gpuCache/gpuCacheDrawOverride.cpp, rawfootPrintNode/rawfootPrintNode.cpp, and viewDX11DeviceAccess/viewDX11DeviceAccess.cpp.

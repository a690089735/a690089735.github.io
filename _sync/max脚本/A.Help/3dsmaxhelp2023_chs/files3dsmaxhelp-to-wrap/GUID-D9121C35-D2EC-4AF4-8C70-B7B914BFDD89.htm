平移摄影机

对于目标摄影机，通过拖动鼠标沿着平行于摄影机视图的方向移动摄影机及其目标。

当“摄影机”视口处于活动状态时，“平移”按钮将替换为此按钮。

过程

要平移摄影机，请执行以下操作：

  1. 激活“摄影机”视口。
  2. 单击 （平移摄影机）。

    该按钮处于启用状态时将高亮显示。

  3. 拖动可移动摄影机及其目标。

    摄影机及其目标将沿着平行于视图平面的方向移动，这将与摄影机的视线垂直。

  4. 按下 Esc 键或者右键单击可关闭该按钮。

要使用鼠标中键进行移动，请执行以下操作：

  • 按住鼠标中键，然后进行拖动。

要将平移约束到单个轴，请执行以下操作：

  • 按下 Shift 键。

    按下 Shift 键时，可将平移约束到移动的第一个轴。

要加速平移，请执行以下操作：

  • 按下 Ctrl 键。

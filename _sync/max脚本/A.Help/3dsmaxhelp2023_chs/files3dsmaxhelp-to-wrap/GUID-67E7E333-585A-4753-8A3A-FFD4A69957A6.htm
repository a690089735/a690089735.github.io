细分修改器（对象空间）

细分修改器提供多种用于生成输入几何体的精细三角化表示的算法。尤其是自适应算法和可变曲率算法，除了简单地细化网格外，这些算法通常还会生成高质量的三角剖分，其主要由大致等边的三角形组成。通过细分修改器重新划分网格有助于获取条件良好的输入网格，以便进行复杂的几何体处理操作（例如，适用于 3ds Max 的 Autodesk 重新拓扑工具提供的重新拓扑操作），或者使用光能传递渲染器进行渲染。此外，网格密度越高，在渲染期间就可以获得越精细的照明细节和精度。

注意: 处理密度较高的网格时，内存需求和计算时间通常会增加。
  • 选择对象 “修改”面板 “修改器列表” “对象空间修改器” “细分”
  • 默认菜单：进行选择。 “修改器”菜单 “光能传递” “细分”
  • Alt 菜单：进行选择。 “修改器”菜单 “细分曲面” “细分”

已应用“细分”修改器并选择“可变曲率”模式的对象

尽管这主要是为了增加光能传递解决方案的质量而开发的，但“细分”修改器也可以被需要形状良好的网格的任何应用程序所使用。例如，可以改善为地形复合对象而生成的不规则的网格元素。

该修改器有世界空间和对象空间变量。在世界空间修改器中，在变换成世界空间坐标系后，大小限制是在网格上。对象空间修改器在对象空间坐标中限制大小。

注意:通常，在处理光能传递解决方案时，就会自动地将细分修改器应用到场景中的对象上。可以在光能传递控件面板中在全局基础上对网格参数进行设置，也可以在“对象属性”对话框中为单独的对象设置网格参数。
提示当对一个对象上的细分设置感到满意时，可以将该修改器拖到其他对象上来进行传播。

界面

“细分模式”卷展栏

细分模式
选择要使用的细分算法：
  • 细分：高度约束的旧版解算器，可保持所有现有边。
  • Delaunay：在“保留网格数据”选项处于启用状态时，约束具有数据的边。此模式还包括用于松弛不受约束曲面的选项。
  • 自适应：尝试实现相对均匀的网格，像“细分”和“Delaunay”模式那样，但计算时间更长，结果通常也更好。由于对输入网格的依赖性较小，因此在该模式下，原始网格的不受约束边与输出网格的边之间通常没有对应关系。
  • 可变曲率：与“自适应”算法具有一样高的质量，并在网格的平面区域中提供类似的结果。不过，此模式将在高曲面曲率的区域中进一步优化，以便准确捕获形状。“可变曲率”模式对高曲率区域的优化程度由“可变密度”参数控制。此模式需要的计算时间最多，但通常可为复杂几何体提供最佳结果。

细分模式

大小
在输出网格上设置目标面长度比例。

Delaunay 模式

大小
在输出网格上设置目标面长度比例。这类似于其他模式下的边长。
松弛
在每个松弛迭代中设置松弛和未松弛顶点位置之间的混合。值为 1.0 将产生完全松弛的结果。
松弛迭代次数
设置要应用于输出网格的松弛迭代次数。
保留网格数据
启用后，在进行细分时，将保留网格数据（例如平滑组、材质 ID、布尔接缝、法线、指定的 UV 和顶点浮点数据通道）。这在“重新拓扑”等修改器中会有所帮助，在修改器中，可以将这些网格数据用作自动边约束。
禁用此选项后，细分结果中将没有网格数据。
UV 通道
仅当“保留网格数据”处于启用状态时，“UV 通道”才处于活动状态。将为指定的贴图通道创建索引并保留此贴图通道，为此，应将贴图坐标和接缝传播到输出网格。
保留锐边
允许将硬边传播到输出网格。检测到的边基于相邻面法线之间的角度。此设置影响此类边附近的细化程度，类似于“平滑修改器”使用边角度阈值来确定硬边和软边的方式。
角度
设置角度（以度为单位），超过该角度时将检测硬边。

自适应模式

边长
在输出网格上设置目标边长。如果网格上的边超过指定的边长量，则将细分该边及其关联面。边长量基于世界单位。值越小，处理时间越长。
规则性
控制在输出网格上施加的规则性程度。规则性越高，网格越均匀。例如，规则性为 1.0 时，输出网格上的大小相对均匀，三角形更接近等边。规则性越接近 0.0，大小变化越大，精度也越高。
阈值
为算法设置法线阈值。
保留网格数据

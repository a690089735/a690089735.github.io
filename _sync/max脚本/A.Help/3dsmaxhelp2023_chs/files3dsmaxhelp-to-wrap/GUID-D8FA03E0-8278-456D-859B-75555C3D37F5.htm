链接变换修改器

“链接变换”修改器将任何对象或子对象选择链接到另一对象中，这一对象称为控制对象。控制对象的运动、旋转和/或缩放变换将传递到对象或子对象选择上。

  • “修改”面板 选择对象或子对象。 “修改器列表” “对象空间修改器” “链接变换”
  • 选择对象或子对象。 “修改器”菜单 “动画” “链接变换”

使用链接变换

“链接变换”将从堆栈中接收的任何几何体链接到另一对象中，该对象称为控制对象。其单个控件只拾取控制对象。要使用此修改器，场景中必须至少有两个对象。

过程

要应用“链接变换”修改器。

  1. 选中对象堆栈的一个位置，并对其应用“修改器”列表中的“链接变换”。
  2. 在“参数”卷展栏上单击“拾取控制对象”。当设置动画时，在 0 帧做这些操作。
  3. 选择要成为控制对象的对象。

    这就完成了链接操作。控制对象的名称将显示在“参数”卷展栏中。

要在子对象层级上应用“链接变换”，请执行如下操作：

  1. 选择“可编辑网格”或已应用“网格选择”修改器的对象。
  2. 启用顶点子对象层级，并选择此对象上的部分顶点。
  3. 应用“链接变换”修改器。
  4. 在“参数”卷展栏上单击“拾取控制对象”。当设置动画时，在 0 帧做这些操作。
  5. 选择要控制子对象选择的另一个对象。

    这就完成了链接操作。控制对象的名称将显示在“参数”卷展栏中。

  6. 移动控制对象并查看它如何影响顶点。

界面

控制对象

与顶点链接的对象。完成变换后，顶点跟随变换。

拾取控制对象

单击此按钮，选择要设置为控制对象的对象。

回退变换

将使用“链接变换”修改器的对象与“控制对象”链接起来。通常情况下，移动“控制对象”将使链接对象移动两次，一次伴随“控制对象”，一次伴随链接。当启用切换时，“控制对象”上的任意变换仅应用到链接对象一次。此切换类似于“蒙皮”修改器的“回退变换顶点”的切换。

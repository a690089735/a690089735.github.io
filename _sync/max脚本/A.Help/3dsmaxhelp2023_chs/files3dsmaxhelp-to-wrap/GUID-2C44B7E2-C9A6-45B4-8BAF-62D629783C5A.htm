分割量测试

通过“分割量”测试，可以将特定数目的粒子发送给下一个事件，将所有剩余的粒子保留在当前事件中。可以按照特定数目或百分比或按照每 N 个粒子分割粒子流。对于特定数目的粒子，每个事件分割一次，但是可以通过设置百分比和“间隔”值的动画，随时间改变分割粒子的数量。

  • “粒子视图” 单击某个事件中的“分割量”，或将“分割量”添加到粒子系统中，然后选择它。

界面

该用户界面出现在“粒子视图”对话框右侧的“参数”面板中。

“为以下项测试真值”组

可以选择如何分割粒子系统。

  • 粒子比例（默认设置。）按“比率”值指定的百分比分割粒子系统。如果“比率”大于 0.0，是否分割特定粒子取决于随机化因子；使用“唯一性种子”值可以进行更改。

    比率 指定将测试为 True 的粒子的百分比。可设置动画。默认设置为 50.0%。

    仅适用于“粒子比例”选项。

  • 每 N 个粒子 分割定期的粒子采样。

    N 指定分割粒子的频率。可设置动画。默认设置为 3。

    例如，如果输入 8，则每 8 个粒子分割一次。

    仅适用于“每 N 个粒子”选项。

  • 前 N 个粒子

    从第一个进入事件的粒子开始分割指定为 N 值（下面）的粒子数，保留其他粒子。

  • 前 N 个粒子后所有粒子

    从 N 参数（下面）指定的 N 个粒子后的第一个粒子开始分割所有粒子。从第一个进入事件的粒子开始到第 N 个粒子的所有粒子保留在该事件中。

    N 对于“前 N 个粒子”，指定要分割的粒子数，对于“前 N 个粒子后所有粒子”，指定要保留在该事件中的粒子数。

    仅适用于“前 N 个粒子”或“前 N 个粒子后的所有粒子”选项。

每发射源
启用时，“粒子流”为每个发射源分别应用“前 N 个粒子”和“前 N 个粒子后的所有粒子”选项的 N 值。

对于聚集到单个“分割量”测试中的多个粒子流源，应使用此选项。

“唯一性”组

“唯一性”设置可以使“粒子比例”选项设置的粒子保留随机化。

种子
指定随机化值。
新增特性
使用随机化公式计算新种子。

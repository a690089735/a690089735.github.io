键盘快捷键

使用键盘快捷键启动操作，而无需使用鼠标。

键盘快捷键（也称为热键）提供最快、最高效的工作方式，因此被 3ds Max 美工人员广泛使用。例如，要在不使用菜单的情况下打开“从场景选择”对话框，只需按 H 键即可。要将活动视口更改为俯视图，可以按 T 键。

已为常见操作定义键盘快捷键，但是您可以更改这些指定热键，并将热键应用于未定义的操作。要修改或添加新的快捷键，请使用“热键编辑器”。自定义热键集保存在您的用户设置文件夹中，可以在使用 3ds Max 2020.1 及更高版本的系统之间复制。

多数情况下，可以使用用于打开对话框的相同命令将其关闭。这适用于输入方法的任何组合，包括菜单、工具栏按钮和键盘快捷键。有关详细信息，请参见“切换对话框”。

可以向多个命令指定相同的快捷键，只要这些命令出现在不同的上下文中即可。例如，在 Video Post 中，快捷键 CTRL+S 将指定给“添加场景事件”；然而，在主 UI 中，该快捷键将指定给“保存文件”。 使用键盘快捷键时，3ds Max 首先会寻找特定于上下文的快捷键。如果没有找到快捷键，则会在主 UI 快捷键中寻找相应的命令。

为使上下文特定的快捷键正常工作，必须启用键盘快捷键覆盖切换（默认设置）。如果禁用此选项，则只能使用主 UI 键盘快捷键。

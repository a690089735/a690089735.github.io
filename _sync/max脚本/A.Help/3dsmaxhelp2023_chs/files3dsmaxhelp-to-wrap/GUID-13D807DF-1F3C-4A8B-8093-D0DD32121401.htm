“分布(光度学文件)”卷展栏

当使用光域网分布创建或选择光度学灯光时，“修改”面板上将显示“分布(光度学文件)”卷展栏。使用这些参数选择光域网文件并调整 web 的方向。

  • “创建”面板 （灯光） “光度学” 创建光度学灯光。 “常规参数”卷展栏 将“光度学 Web 文件”选为分布类型。 “分布 (光度学文件)”卷展栏
  • 默认菜单：“创建”菜单 “灯光” “光度学灯光” 创建光度学灯光。 “常规参数”卷展栏 将“光度学 Web 文件”选为分布类型。 “分布 (光度学文件)”卷展栏
  • Alt 菜单：“对象”菜单 “灯光” 选择一个光度学灯光 创建灯光。 “常规参数”卷展栏 将“光度学 Web 文件”选为分布类型。 “分布 (光度学文件)”卷展栏

界面

Web 图

在选择光度学文件之后，该缩略图将显示灯光分布图案的示意图。

光度学 Web 文件的缩略图

鲜红的轮廓表示光束。在某些 Web 中，深红色的轮廓表示（不太明亮的）区域。

选择光度学文件

单击此按钮，可选择用作光度学 Web 的文件。该文件可采用 IES、LTLI 或 CIBSE 格式。一旦选择某个文件后，该按钮会显示文件名（不带 .ies、.ltli 或 .cibse 扩展名）。

当浏览光度学 Web 文件时，“文件”对话框还会显示高亮显示文件的缩略图。

X 轴旋转

沿着 x 轴旋转光域网。旋转中心是光域网的中心。范围为 -180 度至 180 度。

Y 轴旋转

沿着 y 轴旋转光域网。旋转中心是光域网的中心。范围为 -180 度至 180 度。

Z 轴旋转

沿着 z 轴旋转光域网。旋转中心是光域网的中心。范围为 -180 度至 180 度。

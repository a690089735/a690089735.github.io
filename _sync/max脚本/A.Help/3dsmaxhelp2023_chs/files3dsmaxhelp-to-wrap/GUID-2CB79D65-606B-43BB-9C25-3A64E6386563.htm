复制控制器

复制控制器将高亮显示的项目复制到“轨迹视图”剪贴板中。

  • “轨迹视图” 在“控制器”窗口中，高亮显示控制器轨迹。 “控制器”工具栏 （复制）
  • “轨迹视图” 在“控制器”窗口中，高亮显示控制器轨迹。 “编辑”菜单 “复制”
  • 键盘 Ctrl+C

可以使用“复制”和“粘贴”复制对象和修改器轨迹、控制器、对象和组件，这取决于您正在使用的对象。

“控制器”工具栏中也提供了“控制器”工具。在默认情况下，该工具栏是隐藏的；要取消隐藏，请右键单击该工具栏，然后选择“显示工具栏”“控制器：轨迹视图”。

并非可以复制和粘贴所有轨迹。在“轨迹视图”中使用“复制”和“粘贴”的基本限制是：

  • 只能复制选定的单个项。
  • 所复制的项只能粘贴到相同类型的另一个项中。
  • 仅在选择的多个项为相同类型时，才能将所复制的项粘贴到这些项中。

过程

要粘贴和复制修改器，请执行以下操作：

  1. 在“轨迹视图层次”中，高亮显示一个修改器轨迹。
  2. 从“编辑”菜单中选择“复制”。
  3. 在“轨迹视图层次”中单击一个对象的对象轨迹。
  4. 从“编辑”菜单中选择“粘贴”。

    将打开“粘贴”对话框。

  5. 设置“粘贴”对话框中的选项，然后单击“确定”。

    所粘贴的修改器插入选定对象轨迹的上方。它不替换目标对象中的任何轨迹。

要粘贴和复制控制器，请执行以下操作：

  1. 在“轨迹视图层次”中，高亮显示一个控制器轨迹。
  2. 在“轨迹视图编辑”菜单上，单击“复制”。
  3. 在“轨迹视图层次”中，高亮显示一个对象的控制器轨迹。

    确保选择与所复制变换相同类型的变换。

  4. 从“编辑”菜单中选择“粘贴”。

    将打开“粘贴”对话框。

  5. 设置“粘贴”选项，然后单击“确定”。

    控制器粘贴到选定对象上。

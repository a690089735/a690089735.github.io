轨迹视图工具栏右键单击菜单

可使用右键单击工具栏菜单上的命令，执行大多数“轨迹视图”中的 UI 的自定义任务。在“轨迹视图”工具栏右侧的空白区域中移动光标，然后单击右键。此快捷菜单上的自定义工具如下所示：

  • 停靠将工具栏放在窗口上下的水平位置，或沿窗口左右边缘垂直放置它。工具栏停靠在水平位置后，可以按照您的偏好，进行拖放以重排它们的顺序。还用于将窗口停靠在视口下方。
  • 浮动浮动选定工具栏。

    可以将工具栏浮动在喜欢的任意位置。还可以拖动工具栏左侧的双栏，以便在视口上方浮动工具栏。

  • 隐藏禁用选定用户界面元素的显示。隐藏某个元素后，可使用此菜单上的“显示 UI 元素”或“显示工具栏”来重新显示它。
  • 加载布局加载保存的已命名默认“功能曲线”、“摄影表”或轨迹栏布局。
  • 保存布局保存自定义布局，创建新的默认布局。在打开新的“轨迹视图”时，会显示保存的此布局。
    注意:要恢复默认的 3ds Max 布局，请参见要保存“轨迹视图”布局，请执行以下操作：。
  • 保存布局为使用您选择的名称保存自定义布局。自定义布局后，使用便于识别的名称保存它，然后可随时选择返回它。使用“加载布局”检索保存的布局。
  • 显示 UI 元素切换“轨迹视图”用户界面元素的显示。例如，可隐藏或显示菜单栏、滚动栏、控制器和关键点窗口以及时间标尺。
  • 添加工具栏用于显示可使用主菜单栏上的“自定义用户界面”下拉列表中的命令创建的其他工具栏。
  • 删除工具栏删除选定的工具栏。
  • 显示工具栏用于选择要显示或隐藏的工具栏。
    注意:在默认情况下，部分工具栏是隐藏的。

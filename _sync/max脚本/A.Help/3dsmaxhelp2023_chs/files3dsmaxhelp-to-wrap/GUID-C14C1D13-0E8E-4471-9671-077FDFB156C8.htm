键盘快捷键覆盖切换

可在仅使用主 UI 快捷键或使用所有快捷键之间切换。

  • 主工具栏 “键盘快捷键覆盖切换”

禁用“覆盖”切换时，只识别主 UI 快捷键。启用“覆盖”时，可以同时识别主 UI 快捷键和功能区域快捷键。如果指定给功能的快捷键与指定给主 UI 的快捷键之间存在冲突，则启用“覆盖”时，以功能快捷键为先。

您可以使用“热键编辑器”自定义键盘快捷键。该列表显示已将哪些快捷键指定给哪个命令或功能。

FBXImporterGetParam

getparam 脚本用于查询导入插件参数，其中 [arg] 为已识别的导入参数。

“FBXImporterGetParam”返回命名参数的值。如果参数无法识别或所提供的参数数目错误，则返回“unsupplied”值。

有关受支持的参数列表，请参见 FBXImporterGetParam [arg] [value]。

FBXImporterSetParam [arg] [value]

setparam 脚本 用于为导入设置特定的参数，其中 [arg] 为已识别的导入参数，[value] 是所支持参数对应的值。

  • 如果成功，“FBXImporterSetParam”将返回“OK”，如果向其发送的是无法识别的参数，将返回“unsupplied”。
  • 如果传递的参数数目错误，“FBXImporterSetParam”将返回“unsupplied”。

例如，要使用“添加并更新”场景元素模式和蒙皮修改器导入 FBX 文件，但无需加载动画 FCurve，则输入以下内容： 

	FBXImporterSetParam "Mode" #merge
	FBXImporterSetParam "Skin" true
	FBXImporterSetParam "Animation" false

"Cameras" True 或 False。有关此设置的说明，请参见摄影机。
"ConvertUnit" “mm”、“cm”、“dm”、“m”、“km”、“mi”、“in”、“ft”或“yd”。有关此设置的说明，请参见文件单位转化为。
"FillTimeline" True 或 False。有关此设置的说明，请参见填充时间表。
"FilterKeyReducer" True 或 False。
"FilterKeySync" True 或 False。
"GenerateLog" True 或 False。有关此设置的说明，请参见生成日志数据。
"ImportBoneAsDummy" True 或 False。有关此设置的说明，请参见骨骼创建。
"KeepFrameRate" True 或 False。
"Lights" True 或 False。有关此设置的说明，请参见灯光。
"LoadImportPresetFile" "x"，其中 x 表示预设文件的位置。有关此设置的说明，请参见导入预设。
"Markers" True 或 False。有关此设置的说明，请参见视觉标记。
"Mode" #create（添加）#exmerge（更新动画）#merge（添加并更新动画）。有关此设置的说明，请参见文件内容。
"PopSettings" 当您执行单击操作时，此设置将保留设置的当前状态。此设置加载 FBXImporterSetParam "PushSettings" 保存的所有设置。
"PushSettings" 当您执行单击操作时，此设置将保留设置的当前状态。此设置保存所有 I/O 设置
"PointCache" True 或 False。有关此设置的说明，请参见点缓存文件。
"Resampling" 浮点数。
"ResetImport" 还原为出厂默认设置。如果调用 ResetImport 时不使用以下软件的参数：
  • 3DS Max：加载 Autodesk Media & Entertainment 导入预设。
  • 3DS Design：加载 Autodesk Architectural (Revit) 导入预设。
将仅文件名用作参数：如果传递 "Autodesk Media & Entertainment.fbximportpreset" 或 "Autodesk Architectural (Revit) .fbximportpreset"，将搜索用于加载预设文件的出厂预设目录。

使用此命令加载出厂预设时无需知道目录。

注意:“单击”功能将使用该命令。
"ScaleConversion" True 或 False。有关此设置的说明，请参见单位。
"ScaleFactor" 浮点数。有关此设置的说明，请参见比例因子。
"Shape" True 或 False。有关此设置的说明，请参见变形。
"Skin" True 或 False。有关此设置的说明，请参见变形。
"SmoothingGroups" True 或 False。有关此设置的说明，请参见平滑组。
"TakeCount" 此参数仅适用于“get”函数。
"TakeIndex" 整数，范围从 1 到 FBX 中的应用的个数。此参数仅用于设置。指定应用时，用 0 指示无动画。（这是此属性的默认值。）用 1 指示 FBX 文件中的第一个应用，用 2 指示第二个应用，依此类推。有关此设置的说明，请参见动画应用。
"UpAxis" “Y”或“Z”。有关此设置的说明，请参见轴转化。

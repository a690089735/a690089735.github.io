操纵装备

通过 CAT 中的 FK/IK 系统，您只需将角色推送到所需姿势，而不必太过担心它们采用的是 IK 还是 FK。通常与 IK 关联的大多数控件也会在 FK 中提供，反之亦然。

当使用装备时，您将发现可以使用“移动”工具实现许多功能。当然，也可使用“旋转”实现，但它往往不是必需的。

FK/IK 系统包括三个主要功能：效果层次、FK 驱动 IK 和重定位。这些功能与 CAT 的程序脊椎、手指或足趾控件以及脚轴系统相辅相成。

CATRig 的功能之一是：如果移动层次中的骨骼（不管您是处于 IK 还是 FK 模式下），任意一侧的骨骼都会随之旋转以符合此移动操作。此系统使您可以快速、直观、精确地操纵装备。此系统称为“效果层次”。以下是三个效果层次示例。

弯曲肘部

往后移动肘部（前臂）会使手臂发生弯曲。

编辑之前的手臂位置

移动前臂之后的手臂位置

调整手指

移动手指部位同样非常直观。

伸开的手指

移动中指将使手指向上弯曲。

操纵脊椎

CATRig 具有两种类型的脊椎（即程序和 FK），这两种类型的脊椎的使用方式都大致相同。主要区别在于：FK 脊椎在每块脊椎骨骼上具有 PRS 控制器，这表示您可以单独变换这些骨骼。请注意，FK 脊椎也存在一定局限性。

脊椎在脊椎末端的连接部之间进行插补，因此，您只需根据需要变换连接部即可。

原始姿势：选择胸腔。

当移动胸腔时，脊椎将随之进行插补。

FK 脊椎的局限性

尽管 FK 脊椎使您能够单独设置每块脊椎骨骼的动画，但仍存在一些局限性：

  • FK 脊椎不适用于 CATMotion。
  • FK 脊椎不适用于 CAT 的重定位系统。
注意:使用程序脊椎创建的装备中的动画数据将无法正确加载到使用 FK 脊椎创建的装备中，反之亦然。

最后，FK 脊椎需要您管理更多关键帧。除非您确实需要附加控件，否则强烈建议您使用程序脊椎。

调整骨盆

有时，您可能需要在不影响胸腔（子连接部）的情况下调整骨盆（父连接部）。为此，CAT 提供了一个名为“锁定骨骼”的功能，通过此功能，您可以在不影响胸腔的情况下操纵骨盆。

选择骨盆

当变换骨盆时，胸腔将保持静止不动。

操纵手指（手指或足趾）

当您可以直接变换手指（手指或足趾）时，您可能更喜欢使用手指或足趾管理器。通过该管理器的控件，您能够编辑选定手指的展开、弯曲、滚动和卷曲程度。手指或足趾管理器还包含内置的手指或足趾姿势预设管理器。

选择手掌。

在“手指或足趾修改器”列表中，高亮显示要编辑的手指或足趾。

拖动“卷曲”微调器，手指将向内卷曲。

旋转脚部

在脚轴系统中，脚轴是脚部平台（脚部的 IK 目标）的子对象。您只需根据需要定位脚轴并旋转脚部即可。完全可以设置脚轴动画。

选择脚部平台。选择之后，您即可看到脚轴。

将轴移至脚部平台前方。

当旋转脚部平台时，此时它将从脚趾开始旋转。

当继续设置动画时，您只需根据需要在适当位置设置脚轴动画。

操纵连接部

当在动画模式下右键单击选定连接部时，可在“工具 2”四元菜单上找到“锁定骨骼”命令。例如，如果启用此控件，您可以编辑底部连接部（如骨盆），而不会影响末端连接部（如胸腔）的位置和旋转。请参见锁定连接部。

操纵尾部

要操纵尾部，请选择尾部中的若干骨骼并旋转。所有骨骼将一起旋转，从而使尾部自然卷曲。

选择多个尾部链接

当向上旋转时，尾部将平稳卷曲

操纵锁骨

通常，您应旋转锁骨，而不是移动它。此外，最好不要扭曲锁骨。

编辑锁骨旋转的最简单的方法是：选择上臂或肩膀骨骼并移动它。当移动上臂骨骼时，将根据您的移动旋转锁骨。或者，直接选择锁骨，并使用“移动”、“旋转”和“缩放”工具。

扫描修改器面板

应用扫描修改器后，其界面显示在“修改器”面板中，包含多个卷展栏。

  • “修改”面板 选择一个 2D 图形。 “修改器列表” “扫描”
  • 默认菜单：选择一个 2D 图形。 “修改器”菜单 “面片/样条线编辑” “扫描”
  • Alt 菜单：选择一个 2D 图形 “修改器”菜单 “样条线” “扫描”

本节内容

  • “截面类型”卷展栏（扫描修改器）

    使用“截面类型”卷展栏，可选择内置的备用截面或基于自定义图形的截面。

  • “插值”卷展栏（“扫描”修改器）

    “扫描”修改器的“插值”卷展栏中控件的工作方式同对任何其他样条线所执行的操作完全一样。但是，控件只影响选中的内置截面，而不影响截面扫描所沿的样条线。

  • 各种截面类型的“参数”卷展栏（扫描修改器）

    “参数”卷展栏是上下文相关的，并且会根据所选择的沿着样条线扫描的内置截面显示不同的设置。例如，较复杂的截面如“角度”截面有七个可以更改的设置，而“四分之一圆”截面则只有一个设置。

  • “扫描参数”卷展栏（扫描修改器）

    “扫描参数”卷展栏包含用来构建扫描几何体的各种控件。

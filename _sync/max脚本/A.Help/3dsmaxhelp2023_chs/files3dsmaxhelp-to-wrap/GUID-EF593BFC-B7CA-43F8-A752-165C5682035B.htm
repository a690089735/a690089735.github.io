重心坐标

假设点 A、B 和 C 围成一个三角形，这个三角形曲面上的每个 X 点都可以用角的加权总和表示：

X = a*A + b*B + c*C

其中 a、b 和 c 是 0 和 1 之间的数字，并且 a+b+c = 1。

这些数字称为点 X 的重心坐标。在三角形上，每个点都具有一组唯一的重心坐标。

示例

由重心坐标 (1/3, 1/3, 1/3) 指定的三角形的重心如下：X = 1/3 A + 1/3 B + 1/3 C = (A+B+C)/3。

如果其中一个重心坐标为零，那么点 X 必须位于对边上。例如：

如果 a=0，则 X = b*B + c*C

其中 b+c=1

这意味着 X 位于线段 BC 上。

另一方面，如果 a=1，则 b=c=0，那么 X 的确切位置必定在点 A 处。

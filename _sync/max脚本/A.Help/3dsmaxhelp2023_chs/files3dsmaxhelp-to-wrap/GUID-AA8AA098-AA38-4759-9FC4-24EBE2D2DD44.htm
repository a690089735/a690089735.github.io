编辑手指或足趾

可在手指或足趾级别以及手指或足趾中的每块骨骼级别来编辑手指或足趾。

  • 选择手掌或踝部骨骼。 “修改”面板 “手掌设置”/“踝部设置”卷展栏 将“手指或足趾数”设置为非零值。 选择手指或足趾骨骼。

界面

“手指或足趾设置”卷展栏

通过在视口中选择手指或足趾骨骼，可访问“手指或足趾设置”卷展栏。这些设置影响骨骼所属的整个手指或足趾。

名称
手指或足趾的名称。请参见使用 CAT 命名系统。
“复制”/“粘贴”/“镜像粘贴”
用于在其他手指或足趾中复制和粘贴此手指或足趾的设置的按钮。请参见复制和粘贴装备部位。
骨骼数量
手指或足趾所具有的骨骼数。范围介于 0 到 20 之间。
注意:在默认情况下，创建的每块新骨骼的维度将与手掌或足趾末端的骨骼的维度相同。
“宽度”/“深度”
手指或足趾在各维度上的大小（以 CATUnits 为单位测量）。

“手指或足趾骨骼设置”卷展栏

该卷展栏包含选定手指或足趾骨骼的设置。

名称
手指或足趾骨骼的名称。请参见使用 CAT 命名系统。
使用自定义网格
切换此手指或足趾骨骼的自定义网格。请参见在 CATBone 上自定义几何体。
长度
手指或足趾骨骼的长度（以 CATUnits 为单位测量）。
曲线权重
控制 CATMotion 中的 DigitCurl 值对手指或足趾骨骼的影响程度。值为 1.0 表示 CATMotion 具有 100% 的效果。值为 0.0 会对选定手指或足趾禁用 CATMotion。值为 -0.5 表示 DigitCurl 将在反方向上产生一半的效果。

“添加躯干部位”组

添加骨骼
为手指或足趾骨骼添加附加骨骼。
添加装备
为手指或足趾骨骼添加特殊装备对象，例如，小道具。有关详细信息，请参见添加装备对象。

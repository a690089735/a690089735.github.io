按名称选择（轨迹视图）

使用“按名称选择”，通过在可编辑字段中输入轨迹名称（包括可选通配符），可以高亮显示“控制器”窗口中的轨迹。如果“控制器类型”显示在“过滤器”选项中处于活动状态，则也可以按照控制器类型指定。

  • 轨迹视图 “轨迹选择”工具栏 “按名称选择”字段

使用通配符的“按名称选择”

注意

  • 名称选择不区分大小写，但是括号内的文本除外。
  • 只能选择已打开并显示的那些轨迹。
  • 您可以使用通配符字符。例如：

    Seat 将选择名称以 Seat 开头的所有对象。

    (Cone01)Position 只选择 Cone01 下的“位置”轨迹。

    (Cone*)Position 选择以“Cone”开头的所有对象下的“位置”轨迹

  • 如果用逗号隔开多个名称，则您可以在选择中包含它们，例如：Cone,Box,Sphere。

过程

要高亮显示场景中的所有轨迹，请执行以下操作：

  • 在“轨迹选择”工具栏的“按名称选择”字段中输入 *。

    一次将选中所有轨迹。

示例：要高亮显示三个对象的“位置”轨迹，请执行以下操作：

  1. 创建三个长方体。

    默认情况下，将这三个长方体命名为 Box01、Box02 和 Box03。

  2. 在主工具栏上，单击“曲线编辑器”。
  3. 在“轨迹视图层次”列表中，展开这三个长方体对象的层次。
  4. 在“按名称选择”字段中，键入 (Box*)p。

    将高亮显示所有位置轨迹。

  5. 在“轨迹视图”工具栏上，右键单击“过滤器”并选择“仅选定轨迹”。

    将在“层次”列表的顶部显示三个长方体的位置轨迹。

示例：要高亮显示使用逗号分隔的名称选择的多个轨迹，请执行以下操作：

  1. 创建长方体。
  2. 在主工具栏上，单击“曲线编辑器”。
  3. 在“轨迹视图层次”列表中，展开 Box01 层次以查看所有变换子轨迹。
  4. 在“按名称选择”字段中，键入 x pos,y rot,scale。

    高亮显示长方体的 X 位置、Y 旋转和缩放轨迹。放在每个名称之间的逗号作为 Boolean 和 ("OR") 操作符，允许一次选择多个名称。

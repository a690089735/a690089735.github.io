“参考几何体”卷展栏（投影修改器）

在“参考几何体”卷展栏上，可以创建已命名的子对象选择集，并且使其与高分辨率的几何体相关联。

  • 选择对象。 “修改”面板 “修改器列表” “对象空间修改器” “投影” “参考几何体”卷展栏

界面

名称

输入一组当前选定子对象的名称。

删除

单击可删除其名称在列表中高亮显示的子对象选择集。

添加

单击可将已命名的子对象选择集添加到列表中。

键盘快捷键：Enter。

选择

单击可选定其名称在列表中高亮显示的选择集中的子对象。

全部删除

单击可删除列表中所有已命名的子对象选择集。

参考几何体窗口

该窗口显示一个已命名的子对象选择集的列表，并且它们与高分辨率的源几何体相关联。如果在对象层级上拾取高分辨率对象，该窗口也显示“对象层级”后面是源对象的名称。

多项比例

对于子对象法线凹凸贴图，选中“渲染到纹理”的“要烘焙的对象”卷展栏“投影贴图”组中的“比例”后，该值增加法线凹凸贴图的默认大小。范围为 0.0 至 2.0。默认值为 1.0。

例如，如果子对象的比例渲染为以 16 x 16 像素渲染子对象选择，则更改“多项比例”为 2.0 将使法线凹凸贴图的大小更改为 32 x 32 像素。

除非子对象选择处于活动状态，否则该控件不可用。

拾取

要使高分辨率几何体和当前选择相关联，请单击“拾取”以将其启用，然后在视口中单击源对象。

拾取列表

要将高分辨率的几何体与当前选择关联，请单击“拾取列表”，然后使用“加入对象”对话框（与“从场景选择”对话框的功能类似）选择源对象。

“显示切换”组

想将高分辨率与您的低分辨率几何体时，该“显示切换”组非常有用。您可以在低分辨率和高分辨率几何体之间快速切换以对比版本。您可以显示选定或所有高分辨率几何体。

启用

启用之后，可能显示或隐藏参考几何体。默认设置为禁用状态。

隐藏参考几何体/隐藏工作几何体

启用“启用”之后，单击“隐藏参考几何体”可隐藏您的高分辨率几何体。相反，单击“隐藏工作几何体”可隐藏低分辨率几何体。您还可以选择显示和隐藏“拾取列表”中的高分辨率几何体。单击“禁用”可隐藏几何体。单击“隐藏”可显示几何体。

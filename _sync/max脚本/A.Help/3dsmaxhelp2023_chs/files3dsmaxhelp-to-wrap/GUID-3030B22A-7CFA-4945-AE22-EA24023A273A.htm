导入 3DS 文件

3DS 是 3D Studio® (DOS) 网格文件格式。您可以将 3DS 文件导入 3ds Max。

  • “文件”菜单 “导入” “选择要导入的文件”对话框 “文件类型” “3D Studio (*.3DS)”

导入 3DS 文件时，可以将导入的对象与当前场景合并，或完全替换当前场景。如果选择将对象与当前场景合并，则会询问您是否要将场景中动画的长度重置为所导入文件的长度（如果导入的文件包含动画）。

导入 3DS 文件时，将导入以下信息：

  • 背景（实心、渐变和位图）。
  • 雾、分层雾和距离线索。
  • 环境光级别。
  • 相减的透明度将转换为 3ds Max“过滤器”透明度，且过滤器颜色设置为漫反射颜色。
  • 透明度衰减设置。
  • 将启用所有贴图通道。在 3DS 文件中关闭的贴图通道不会导入到 3ds Max。
  • 所有贴图参数，包括 UV 变换、负值、镜像和旋转。有些贴图参数（如“模糊”、“亮度”及 RGB 和 Alpha）与在3ds Max 中的工作方式截然不同。这些值将转换为产生类似影响的新值。
  • 遮罩位图将作为 3ds Max 遮罩纹理导入。
  • 导入带有纹理 1 和纹理 2 的材质时，将创建合成纹理，并将添加到标准材质的“漫反射”通道中。
  • 反射贴图、自动生成立方体和镜像。
  • 自动反射贴图第 N 帧和“贴图尺寸”设置。
  • 对“大理石”和“噪波”材质进行 SXP 平移。
  • 3DS/DOS R4 IK 关节参数。
  • 3D Surfer 面片数据。

导入 PRJ 文件时，您将导入上述所有条目，以及“形状”。

导入 3DS 文件时，将不导入以下信息：

  • 变形关键点。
  • 关键帧产生器实例。
  • 已关闭的贴图通道。
  • 自定义的 .cub 格式立方体贴图。
  • 使用贴图左上方像素的 RGB 颜色的贴图图案透明度。

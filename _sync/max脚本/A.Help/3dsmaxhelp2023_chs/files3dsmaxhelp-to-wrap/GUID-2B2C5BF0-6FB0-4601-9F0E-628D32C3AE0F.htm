压碎边界

ProOptimizer 在优化对象时不考虑边缘或面是否位于边界上。

“压碎边界”处于启用状态

顶点 % = 50.0

顶点数 = 278

面数减少但不保留对象边界。

保护边界

（默认设置。）ProOptimizer 将保护那些边缘位于对象边界上的面。不过，高优化级别仍然可能导致边界面被移除。如果对多个相连对象进行优化，则这些对象之间可能出现间隙。

“保护边界”处于启用状态

顶点 % = 50.0

顶点数：557

保留了边界。

“保护边界”处于启用状态

顶点 % = 1.0

顶点数：5

减少“顶点 %”或“顶点数”可能会导致边界扭曲。

排除边界

ProOptimizer 从不移除带边界边缘的面。这会减少能够从模型移除的面数，但可确保在优化多个互连对象时不会出现间隙。

“排除边界”处于启用状态

顶点 % = 10.0

顶点数：65

“顶点 %”值较低时，“排除边界”可能会干扰优化。

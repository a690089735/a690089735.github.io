复制、粘贴和镜像关节参数（HD 解算器）

使用“对象参数”卷展栏中的“复制”和“粘贴”按钮，在 IK 控制器之间复制和粘贴关节设置。还可以将关节设置从非 IK 控制器复制到 IK 复制器，但是不能从 IK 复制器复制到非 IK 控制器。利用 HD IK 解算器层次，可以粘贴到选择的多个关节。

  • 进行选择。 “层次”面板 “IK “对象参数”卷展栏 复制、粘贴和镜像关节参数

使用“镜像粘贴”，可在粘贴操作中镜像有关 X、Y 或 Z 轴的 IK 关节设置。其中包含四个按钮：每个按钮分别针对 X、Y 和 Z 轴，以及禁止一起镜像的“无”选项。

还可以使用主工具栏上的“镜像”工具，镜像 IK 关节。

过程

要复制和粘贴关节参数，请执行以下操作：

  1. 在反向运动学链中选择一个对象。
  2. 设置滑动或转动关节参数。
  3. 在“对象参数”卷展栏上的“滑动关节”或“转动关节”组中，单击“复制”。
  4. 在反向运动学链中，选择不同对象。
  5. 在“镜像粘贴”组中，选择要镜像的轴。
  6. 在“对象参数”卷展栏上的“滑动关节”或“转动关节”组中，单击“粘贴”。

    “滑动关节”或“转动关节”都包含单独的剪贴板。所复制的关节参数保存在剪贴板中，直到您使用新复制的参数替换它们。

界面

“滑动关节”组

使用下列按钮可在对象之间复制滑动关节参数。这些按钮不可用于路径关节。

“转动关节”组

使用下列按钮在对象之间复制转动关节参数。

“镜像粘贴”组

用于在“粘贴”操作期间关于 X、Y 或 Z 轴镜像 IK 关节设置。

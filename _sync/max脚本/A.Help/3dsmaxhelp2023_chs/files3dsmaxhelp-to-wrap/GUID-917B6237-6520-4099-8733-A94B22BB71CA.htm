SHP 文件

SHP 是 3D Studio (DOS) 图形文件格式。您可以将这些文件导入到 3ds Max。

  • “文件”菜单 “导入” “选择要导入的文件”对话框 “文件类型” “3D Studio 图形(*.SHP)”

SHP 文件包含使用 3D Studio 中的 2D Shaper 创建的多边形。

导入包含多个图形的 SHP 文件时，3ds Max 为您提供了以下选择：要么将所有图形合并为一个对象，要么生成多个传入对象。

图形导入器可以查看传入样条线上的向量，如果这些向量在一组百分比点中位于同一条直线上，图形导入器会将角度更改为平滑的 Bezier（否则，将为 Bezier 角）。

本节内容

  • “图形导入”对话框

    导入 SHP 文件时，首先会显示“3DS 导入”对话框。这与导入 3DS 文件时看到的对话框相同。设置选项之后，如果在“3DS 导入”对话框中单击“确定”，然后会显示“图形导入”对话框。

“样条线 IK 解算器”对话框

如果已选择样条线 IK 解算器作为 IK 解算器并启用“指定给子级”，将显示“样条线 IK 解算器”对话框。

  • “创建”菜单 （系统） “骨骼” 选择“SplineIKSolver”作为“IK 解算器”，并启用“指定给子对象”。 创建骨骼结构。

使用此对话框可以为将创建用于控制骨骼结构的样条线设置参数。反过来，样条线的形状由位于每个结（控制点）的辅助对象的位置确定。

界面

“样条线选项”组

IK 名称
设置 IK 链对象的名称。单击“确定”后，此对象显示为链端点处的一组蓝色十字线。
自动创建样条线
基于本对话框中的参数自动创建用于样条线 IK 解算器的样条线。如果禁用此选项，将不会创建样条线，您必须手动创建样条线并将其指定给样条线 IK 解算器，才能使用解算器。有关此步骤的说明，请参见样条线 IK。默认设置为启用。
曲线类型
选择曲线类型，以作为样条线 IK 控制：
  • Bezier使用“创建”“图形”“线”工具创建一条与样条线类似的 Bezier 样条线。这是默认的曲线类型。
  • NURBS 点创建一条 NURBS 点曲线，所有结（控制节点）都在该曲线上。
  • NURBS CV创建一条 NURBS CV 曲线，所有结（控制节点）未必都在该曲线上。
样条线结数量
样条线上的顶点或 NURBS 曲线上的控制点和 CV。您可以通过移动节并设置节的动画来改变样条线或曲线。结数量设置曲线上的细化程度，并设置可以选择并移动以更改曲线形状的点数。

默认值是样条线 IK 链中的骨骼数量。由于样条线 IK 解算器的一个用途是为了减少需要设置动画的对象数量，因此，您可能要使用一个较低的值，例如骨骼数的一半。

“辅助对象选项”组

创建辅助对象
启用后，3ds Max 将一个或多个点辅助对象放置在样条线或曲线的每个结上，从而使其可以更轻松地移动结并设置结的动画。

您可以启用多个辅助对象类型。在您移动一个辅助对象时，位于相同位置的其他辅助对象也将移动，就像曲线节移动一样。

中心标记
在每个结处放置一个小的 X 形点辅助对象。
三轴架
在每个结处放置一个小的三轴架形点辅助对象。
交叉
在每个结处放置一个小的交叉形点辅助对象。
长方体
在每个结处放置一个小的长方体形状的点辅助对象。
大小
设置辅助对象的大小。
恒定屏幕大小
无论视口放大或缩小了多少，都保持辅助对象的大小不变。
在顶端绘制
在场景中所有其他对象的上面显示辅助对象，以便改进复杂场景的可见性。
提示要更改创建后的辅助对象显示，请选择每一个辅助对象，并更改“修改”面板上的选择。

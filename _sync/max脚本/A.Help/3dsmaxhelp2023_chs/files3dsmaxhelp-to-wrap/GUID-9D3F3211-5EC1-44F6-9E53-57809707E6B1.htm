使用运动场

使用运动场可影响液体和泡沫的速度。

将运动场添加到流体模拟：

  1. 从“创建”面板 >“空间扭曲”，单击“运动场”并在视口中拖动。

    或者，您也可以使用菜单创建运动场对象：

    • 默认菜单：“动画”菜单 “流体” “运动场”
    • Alt 菜单：“动画”菜单 “流体” “运动场”

  2. 选择要添加运动场的液体对象。
  3. 在“设置”卷展栏上，单击“模拟视图”以打开“模拟视图”窗口。
  4. 在“液体属性”>“运动场”>“添加运动场”下，单击“拾取”并选择运动场。

    运动场现在与流体模拟进行交互。要调整参数，请在场景中再次选择运动场对象，然后使用“修改”面板上的控件。

    提示要在流体模拟上可视化运动场的效果，请启用“修改”面板的“显示”卷展栏下的“速度栅格”。

将运动场控件应用于几何体：

  1. 将运动场添加到场景。
  2. 在“组控件”下，启用“几何体”。
  3. 拾取希望其继承运动场控件的对象。
  4. 将该对象作为碰撞对象添加到模拟。
  5. 将运动场添加到模拟。

    选定的碰撞对象现在继承所有的运动场控件，然后您可以进行修改以调整结果。

常见技术

可以启用和调整以下参数，以使用运动场创建这些常见的流体效果。

阻尼

  • 要减慢模拟，只需在“运动场”属性的“组控件”区域中启用“阻力”选项即可。

风

  • 将“方向”控件与“运动场”属性的“组控件”区域中的“阻力”选项结合使用，并根据需要启用“湍流”和“噪波”选项。

涡流

  • 启用“湍流”和“噪波”以查看“湍流”和“噪波”卷展栏。在该处，调整“噪波幅值”以添加一些随机运动。在“速度”卷展栏中，启用“钳制速度”以保持流体涡流，其中“最大速度”设置为较高的值，“最小速度”使用从 0.0 增大的值设置动画。

漩涡

  • 在场景中创建一个圆柱体。在“运动场”属性的“组控件”区域中，启用“几何体”以查看“几何体”卷展栏。单击“拾取”按钮，然后选择一个圆柱体。启用“衰减”。在“方向”卷展栏中，确保“方向幅值”为 0.0，并调整“绕轴”选项。如果速度变得太高，也可以增加阻力。

波浪

  • 要创建自发形成然后消散的波浪，请像对上面的涡流那样设置柱形运动场。将圆柱体旋转到一侧，使之与液体曲面的一部分相交，然后为其设置动画向前移动。可以添加一点方向以帮助流体向前移动以摆脱重力，并对“远离轴”使用负值以阻止液体飞离。

断开瀑布

  • 为了帮助将瀑布或类似流分解成水滴，请启用“阻力”并使用“最大深度”较低的“法向阻力”。对于喷射，将泡沫添加到模拟，并将单独的场直接应用于已启用“阻力”和“湍流”的泡沫。

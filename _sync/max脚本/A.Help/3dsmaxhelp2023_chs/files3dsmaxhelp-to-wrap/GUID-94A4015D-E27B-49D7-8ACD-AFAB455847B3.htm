2D 视图

使用“2D 视图”可以将贴图绘制到叠加至视口的平面、矩形的“2D 绘制”窗口。

  • 默认菜单：“工具”菜单 “视口画布” 开始绘制。 单击“2D 视图”。
  • Alt 菜单：“材质”菜单 “创建/编辑贴图” “绘制位图(视口画布)” 开始绘制。 单击“2D 视图”。

此窗口仅在活动视口中可见，因此可在多视口显示中截断。为获得使用“2D 视图”时的最佳结果，可最大化活动视口（按 Alt+W 切换视口最大化）。

“2D 绘制”窗口中的绘制方式与 3D 曲面上的常规绘制方式完全相同。可以正常使用所有绘制工具和层工具。在 2D 窗口中绘制时，对象曲面上的贴图会实时更新。甚至在“2D 绘制”窗口打开时，还可以直接绘制到对象。

3ds Max 视口导航工具不影响 2D 窗口，但基于鼠标的控件会影响：通过滚动鼠标滚轮可放大和缩小窗口，通过按住滚轮按钮（或中间按钮）进行拖动可平移窗口。

笔刷图像扭曲

将矩形纹理贴图应用于圆形的 3D 对象时，在“聚集”点（例如球体的顶部和底部）可能会发生扭曲，因为使图像适应更小的曲面区域需要压缩。使用“视口画布”直接绘制到 3D 曲面时，软件通过在接近聚集点时自动展开图像可以对这种扭曲进行补偿。但是，如果在“2D 绘制”窗口绘制这种区域，图像在窗口中显示正常，但在对象曲面上发生扭曲。

直接应用于顶部附近球体曲面的图像（左侧）在“2D 绘制”窗口中发生扭曲。

应用在“2D 绘制”窗口的图像（右侧）在球体曲面上发生扭曲。

界面

要移动“2D 绘制”窗口，可拖动其标题栏，要重新调整窗口大小，可拖动侧面和底部边缘以及底部角点。要退出“2D 绘制”，可单击“2D 视图”按钮标题栏的“关闭”按钮 (X)。

此外，窗口标题栏中提供了如下工具：

切换 UV 线框

在“2D 绘制”窗口启用和禁用纹理坐标的线框显示（显示为栅格）。

使纹理适合视图

缩放视图以适应窗口中的整个纹理贴图。

实际大小

进行缩放，以使纹理贴图中的每个像素与屏幕像素同样大小。然后纹理贴图以其实际大小显示。

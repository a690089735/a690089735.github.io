导出到 3D DWF 文件

  • “文件”菜单 “导出” “导出到 DWF” “选择 DWF 文件”对话框 “另存为类型”下拉列表 选择“设计 Web 格式(*.dwf)”
  • “文件”菜单 “导出” “导出” 从“另存为类型”下拉列表中选择“发布到 DWF”。

使用 3D DWF 发布，可以导出三维模型的 Design Web Format™ （.DWF™）文件，这样导出的效果几乎与渲染的场景有相同的视觉精度。DWF 文件是高度压缩文件。您可以与没有 3ds Max 的团队成员轻松共享这些文件。

  1. 如您希望在 Autodesk Design Review 中显示的那样，在活动视口中设置场景。

    如果要发布特定的对象或层，请选中这些对象。隐藏不希望发布的对象或层。

    注意:也可以在查看器中分离希望隐藏或使之透明的对象。
    提示如果场景中有摄影机并且希望导出此视图，那么请确保在发布 DWF 文件时“摄影机视口”处于活动状态。
  2. 选择“文件”菜单 “导出”“导出到 DWF”。

    也可以只需选择“导出”，然后从文件选择器对话框的“另存为类型”列表中选择“发布到 DWF (*.DWF)”。

  3. 指定要导出的文件名。
  4. 单击“保存”。
  5. 使用“DWF 发布选项”对话框来进行想要的设置，然后单击“确定”。
  6. 如果启用“保存日志”文件但是不想覆盖现有的日志文件，请输入新的名称或指定其他的文件夹。

本节内容

  • “DWF 发布选项”对话框

    如果发布为 DWF 格式，可以有很多不同的选择。

Physique 快捷键

本主题概述了 Physique 的键盘快捷键。

打开键盘快捷键覆盖切换来启用 character studio 的键盘快捷键。

当“运动”面板和“键盘快捷键覆盖”按钮处于活动状态时，所有character studio 键盘快捷键也处于活动状态。

操作 快捷键 说明
复制封套 Ctrl+C  
删除 Ctrl+D  
下一步 PageDown  
下一选择级别 I  
粘贴封套 Ctrl+V  
上一步 Page Up  
上一选择级别 Shift+I  
重置封套 Ctrl+E 将所选链接的封套重置为默认值。

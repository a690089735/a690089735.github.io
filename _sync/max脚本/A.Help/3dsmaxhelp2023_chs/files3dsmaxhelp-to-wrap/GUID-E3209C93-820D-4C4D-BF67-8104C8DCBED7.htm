装载目录

您可以将目录装载到驱动器盘符，作为使用 UNC 名称的替代方式。例如，在混合的 UNIX/XP/2000 网络中，您可能需要装载输出目录。

对于网络渲染，您需要在网络中的所有计算机上装载（或映射）目录。从而使所有渲染服务器都能够访问共享目录。

在开始此设置之前，请为所有服务器选择要装载的公用驱动器盘符。如果已安装其他驱动器，则可能需要针对此装载将指定切换到自由驱动器盘符。

如果贴图或目标目录位于渲染服务器上，则像其他所有目录一样，在此计算机上装载该目录，即使目录位于本地磁盘也是如此。

使用装载的目录时，请确保要装载的目录正确共享。在指定位图时，请始终使用带有公用驱动器盘符的路径。

以下是一些常规步骤。有关详细信息，请参见 Windows XP 或 2000 文档。

注意:在以下步骤的编写过程中，Windows XP 设为“经典 Windows”的显示界面。

过程

要将目录映射到驱动器盘符（Windows 2000 或 XP），请执行以下操作：

  1. 在 Windows 资源管理器中，选择“工具”“映射网络驱动器”，以显示“映射网络驱动器”对话框。
  2. 将“驱动器”下拉菜单值设置为所选的公用驱动器盘符。
  3. 在“文件夹”中，使用 UNC 约定输入输出目录的确切位置。

    通过使用“映射网络驱动器”对话框中的“浏览”按钮选择计算机和共享目录，也可以将目录映射到一个驱动器盘符。

  4. 单击“完成”以完成装载。
    注意:如果将驱动器映射到大型公司网络中的服务器，则可能需要输入用户名和密码才能进行访问。

“材质和 UV”组

控制 ProOptimizer 处理带有材质或纹理贴图的对象的方式。

  • 创建或导入一个对象。 “修改”面板 “修改器列表” “ProOptimizer”修改器 “优化选项”卷展栏 “材质和 UV”组

界面

保持材质边界

启用时，ProOptimizer 将保留材质之间的边界。属于具有不同材质的面的点将被冻结，并且在优化过程中不会被移除。默认设置为启用。

禁用时，ProOptimizer 可沿材质之间的边界移除面。

原始对象

以红色点显示材质之间边界的原始对象

“保持材质边界”处于启用状态时的优化对象（红色点显示边界）

关闭“保持材质边界”时的优化对象

保持纹理

启用时，ProOptimizer 将保留纹理贴图坐标。禁用时，ProOptimizer 将移除纹理坐标。默认设置为禁用状态。

如果对象没有纹理坐标，将禁用此选项。

注意:如果在优化带贴图的对象时禁用“保持纹理”，则当您尝试渲染该对象时，3ds Max 将显示“缺少贴图坐标”对话框。
保持 UV 边界

仅当启用“保持纹理”时，此控件才可用。启用时，ProOptimizer 将保留 UV 贴图值之间的边界。禁用时，ProOptimizer 将忽略 UV 贴图值。默认设置为禁用状态。

  • 公差仅当启用“保持 UV 边界”时才可用。“公差”值控制 UV 边界的处理方式。值为 0.0（默认值）时，将保留所有具有与相邻面不同的 UV 贴图值的面。值为 1.0（默认值）时，允许移除任何面，即使其相邻面具有不同的 UV 值。
    提示将“公差”设置为 0.1 时，将保留大多数贴图边界，但同时允许合理程度的优化。

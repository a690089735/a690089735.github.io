“调整颜色”对话框（顶点绘制修改器）

使用“调整颜色”对话框可以调整当前选定顶点的颜色。如果没有活动的顶点子对象选择，它将均等地影响所有顶点。

  • 选择对象。 “修改”面板 “修改器列表” “顶点绘制” “顶点绘制颜料盒” 单击“调整颜色”按钮。
  • 选择对象。 “工具”面板 “更多” “指定顶点颜色” 单击“指定给选定对象” “修改”面板 “修改器列表” “顶点绘制” “顶点绘制颜料盒” 单击 （调整颜色）。
  • 默认菜单：选择对象。 “修改器”菜单 “网格编辑” “顶点绘制” “顶点绘制颜料盒” 单击 （“调整颜色”）。
  • Alt 菜单：选择对象。 “修改器”菜单 “几何体(转换为网格)” “顶点绘制” “顶点绘制颜料盒” 单击 （调整颜色）。

界面

  • HSV（默认设置。）选中该选项后，前三个滑块标签为 HSV，并且调整颜色的色调、饱和度和值。
  • RGB选中该选项后，前三个滑块标签为 RGB，并且调整颜色的红色、蓝色和绿色组件。

请参见红、绿、蓝/色调、饱和度、亮度。

预览

启用此选项后，在着色的视口（只要在颜料盒上，顶点颜色显示 - 未着色或顶点颜色显示 - 着色处于活动状态）中交互地预览顶点颜色调整。默认设置为启用。

对比度滑块

调整顶点颜色的对比度。

直方图和输入级别微调器

直方图

以图形方式表示顶点选择中颜色的分布，以及当前阴影、gamma 和高亮显示输入级别。

阴影、gamma 和高亮显示输入级别有助于调整 3ds Max 视口颜色，以更好地匹配目标硬件显示（如游戏引擎）。

阴影级别

调整显示阴影的级别。

Gamma 级别

调整 gamma 显示。该值是一个 gamma 校正值。

高亮显示级别

调整高亮显示的级别。

当调整级别时，相应的箭头在直方图上移动，以表示当前设置。（但是，不能以图形方式拖动箭头。）

应用

单击该选项可将当前设置应用于顶点颜色，而无需关闭对话框。

重置

单击该选项可将对话框设置恢复为其默认设置。

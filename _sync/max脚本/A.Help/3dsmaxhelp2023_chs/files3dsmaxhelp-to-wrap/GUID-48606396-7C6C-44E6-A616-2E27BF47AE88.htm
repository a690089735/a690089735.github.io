MAXScript 界面

MAXScript 是 3ds Max 的内置脚本语言。它的主要界面 MAXScript 菜单包含用于创建和处理脚本的各种命令。此外，状态栏包含一个 MAXScript 迷你侦听器而工具面板还提供 MAXScript 功能。

  • 默认菜单： “脚本”菜单
  • Alt 菜单： “脚本”菜单 “脚本”
  • “实用程序”面板 “MAXScript”

有关 MAXScript 的详细信息，请打开“MAXScript 帮助”（位于“帮助”菜单 “MAXScript 帮助”中）。

本节内容

  • 新建脚本

    使用“新建脚本”可打开一个新的“MAXScript 编辑器”窗口。使用此窗口来编写新脚本。

  • 打开脚本

    使用“打开脚本”选项可以打开选择现有脚本的标准文件对话框。然后新的“MAXScript 编辑器”窗口将显示选定的脚本。

  • 运行脚本

    使用“运行脚本”可打开一个常用文件的打开对话框，用以选择一个现有脚本。然后 MAXScript 将读取和执行所选脚本。所有输出都将打印到“侦听器”窗口。

  • MAXScript 侦听器

    “MAXScript 侦听器”窗口主要是 MAXScript 语言的交互翻译器。它与 DOS 命令提示窗口类似。

  • 宏录制器

    “MAXScript 宏录制器”捕获许多用户执行的操作，并生成与这些操作相对应的 MAXScript 命令。

  • Visual MAXScript 工具（参见 MAXScript 帮助）

    Visual MAXScript 是 3ds Max 脚本语言的强大接口，它使 MAXScript 功能更容易学习和使用。使用 Visual MAXScript，您可以迅速创建脚本的 UI 元素和布局。

  • MAXScript 调试器

    MAXScript 调试器实现了前半部分的脚本开发和调试环境。

  • 从命令行运行脚本

    3ds Max 可以直接在命令行中输入脚本命令。从 DOS 命令行启动 3ds Max 后，可以运行指定的 MAXScript 或 Python 启动脚本。这对自动批处理渲染这样的任务来说非常有用。

2D 平移缩放模式

在 2D 平移缩放模式下，您可以平移或缩放视口，而无需更改渲染帧。

  • “摄影机”、“透视”或“聚光灯”视口 常规（“[+]”）视口标签菜单 “2D 平移缩放模式”
  • “摄影机”、“透视”或“聚光灯”视口 “平移”/“2D 平移”/“穿行”弹出按钮 “2D 平移缩放模式”
注意:2D 平移缩放模式不适用于旧版视口（Direct3D 或 OpenGL）。但是，OpenGL 虚拟视口功能提供相似的功能。

此模式使您能够检查和编辑场景中渲染帧外的部分。当您将“透视”或“摄影机”视口与“透视匹配”结合使用时，此模式尤其有用。2D 平移缩放模式处于活动状态时，可以平移（平移）或缩放（推拉）视口，而无需更改渲染帧。对于“摄影机”视图，这意味着摄影机的位置不会更改。

在 2D 平移缩放模式处于活动状态时，会在其视口标签菜单的右侧显示一个附加的视口标签菜单。

该菜单显示 2D 平移缩放模式处于活动状态，如果已进行缩放，它会显示缩放级别（默认值为 100%）。此外，您还可以用其退出 2D 平移缩放模式。

2D 平移缩放模式不适用于正交视口。

如果放大，以便图像中的像素大于屏幕上的像素，则 3ds Max 将会禁用过滤，以便视图显示单个像素。对于某些图像和模型，此方法可以帮助您进行对齐。

视口背景详细信息

使用 2D 平移缩放进行放大时会显示单个像素。

注意: 在“2D 平移缩放模式”处于活动状态时，按 Z 来最大化显示（如果使用 Autodesk Maya 快捷方式，则按 F）不会测试场景范围，而只是 2D 平移缩放到其原状态，100% 在视口中可见。

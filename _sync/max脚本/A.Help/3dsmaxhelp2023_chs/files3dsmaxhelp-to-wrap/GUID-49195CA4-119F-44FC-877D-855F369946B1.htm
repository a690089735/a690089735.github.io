创建 VUE 文件

  1. 使用“渲染设置”对话框 “指定渲染器”卷展栏可将“VUE 文件渲染器”指定为产品级渲染器。

    您不能将 VUE 文件渲染器指定为 ActiveShade 渲染器。

  2. 激活摄影机视口。
    注意:您必须从摄影机视口进行渲染，才能包含摄影机自身的坐标。
  3. 使用“VUE 文件渲染器”卷展栏可指定一个文件名。
  4. 渲染该场景。

    VUE 文件将写入到磁盘中。将显示“渲染帧窗口”，但不显示图像。

倒角剖面修改器

倒角剖面修改器使用一个图形作为路径或“倒角剖面”来挤出另一个图形。

有两种方法可以创建倒角剖面。经典方法利用创建对象的传统方法（例如，将样条线用作剖面）。改进的方法利用倒角剖面编辑器，后者还与加强型文本结合使用。

注意:尽管此修改器似乎与包含不同缩放设置的放样对象相似，但却是不同的，因为它使用不同的轮廓值而不是缩放值来作为线段之间的距离。此调整图形大小的方法更复杂，从而会导致一些层级比其他的层级包含或多或少的顶点，因此，它通常更适合于处理文本。
注意:如果删除原始倒角剖面，则倒角剖面失效。与提供图形的放样对象不同，倒角剖面只是一个简单的修改器。

操作步骤

经典方法

若要使用经典方法创建倒角剖面，请执行以下操作：

  1. 创建要倒角的图形。
    提示为获得最佳结果，请使用“顶”视口。
  2. 在“前”视口中，创建一个图形用作倒角剖面。
  3. 选择要应用倒角的图形。
  4. 在“命令”面板上，选择“修改”选项卡，然后从“修改器”列表中选择“倒角剖面”。
  5. 在“参数”卷展栏上，选择“经典”。
  6. 在“经典”卷展栏上，单击“拾取剖面”，然后单击剖面图形。

改进方法

若要使用改进的方法创建倒角剖面，请执行以下操作：

  1. 创建要倒角的图形。
    提示为获得最佳结果，请使用“顶”视口。
  2. 在“命令”面板上，选择“修改”选项卡，然后从“修改器”列表中选择“倒角剖面”。
  3. 在“参数”卷展栏上，选择“改进”。
  4. 在“挤出”字段中，将挤出深度设置为大于 0 的数字。
  5. 在“倒角”组中，从列表中选择倒角剖面预设，或选择“自定义”。
  6. 如果要修改预设或创建一个新剖面，请执行以下操作：
    1. 单击倒角剖面编辑器。
    2. 若要在使用倒角剖面编辑器时查看图形中的剖面，单击“自动更新”。
      注意:也可以在“改进”卷展栏中调整设置。
    3. 在“倒角剖面编辑器”窗口中，单击顶点或曲线，然后拖动以重新定位它。
    4. 根据需要（平移、缩放、镜像、翻转等）使用工具。
    5. 当对剖面满意时，请执行以下操作之一：
      • 如果要关闭而不将剖面另存为预设，请单击“取消”。
      • 如果修改了预设，并且希望覆盖它，请单击“保存”。
      • 如果要将剖面另存为新预设，请单击“另存为”，然后输入一个名称并单击“确定”。

“平移和 2D 平移缩放”弹出按钮

在聚光灯视口中，使用此弹出按钮可在平移导航或 2D 平移缩放模式之间选择。

  • 平移（针对聚光灯视口）
  • 2D 平移缩放模式

    2D 平移缩放模式可用于平移或缩放视口，而无需更改渲染帧。（仅限于 Nitrous 视口。）

透视视口或者摄影机视口不显示此弹出按钮。这些视口显示“平移和穿行”弹出按钮。正交视口仅显示“平移”按钮。

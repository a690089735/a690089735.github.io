“显示数据”操作符

在包含“显示数据”操作符的事件中，“显示数据”操作符将在视口中显示“数据”操作符或测试针对某些或所有粒子创建的数字数据，或者如果操作符在全局事件中，则将显示流中的粒子。

  • 粒子视图 添加或选择“显示数据”。

界面

显示粒子 ID
在显示的数据之前显示每个粒子的 ID 编号。粒子是按它们的出生顺序编号的，第一个出生的粒子编号为 1，依此类推。在 ID 编号后面添加 > 符号以区别于数据。
[色样]
显示用于显示数据的颜色。软件为每一个添加到系统的“显示数据”操作符随机选择不同的颜色。若要更改颜色，请单击色样并使用“颜色选择器”对话框来选择新颜色。
注意:如果发射器已选定，数据将显示为白色。
添加前缀
您可以添加在文本字段中输入的自定义前缀。不需要在文字后面附加空间；操作符会自动从数据偏移前缀。在 ID（如果显示）和数据之间显示该前缀。

“显示数据通道”组

[“数据通道”按钮]
单击此按钮可为要显示的数据选择通道。在“选择数据通道”对话框中选择通道。
数据类型
此只读字段显示正在显示的数据的类型。
显示为
显示数据的显示方式。如果只有一个可用选项，此字段为只读。如果有多个可用选项，例如就实数数据而言，可以通过下拉列表选择一个选项。
自旋速率值/速度值
如果将“显示为”设置为“角度”，该选项将为“自旋速率值”。如果将“显示为”设置为“世界单位”，该选项将为“速度值”。启用以启用相应选项。
单位每
当“自旋速率值”/“速度值”处于启用状态时，使用该设置可为自旋速率或速度值选择时间帧。

“屏幕偏移”组

使用这些值可相对于粒子位置调整脚本数据的定位（以像素表示）。通过使用不同的偏移值以及多个“脚本”和“显示脚本”操作符，您可以一次显示多个数据。

显示 ID 和数据的位置
用于为数据显示过滤粒子。这些选项包括：
  • 所有粒子在视口中显示所有粒子的数据。
  • 仅选定粒子仅显示选定粒子的数据。若要选择粒子，请使用“修改”面板，接着使用“选择”卷展栏控件，然后使用标准视口选择方式来拾取粒子以查看其数据（请参见“粒子流”文档获取更多信息）。
  • 仅选定组仅显示已选定组的粒子的数据。使用组选择操作符按钮可指定适用的操作符。
组选择操作符
如果您已将“显示 ID 和数据的位置”设置为“仅组选择”，并且粒子系统至少包括一个“组选择”操作符，请单击此按钮以指定“组选择”操作符或“摄影机消隐”操作符（应将“显示数据”操作符应用到其成员粒子）。
精度
对于浮点值，设置显示的小数位数。范围从 0 到 8（其中 0 表示没有小数位）。
执行顺序
为数据显示设置计时：
  • 前事件中的任何操作符执行之前显示粒子数据值。
  • 正常按操作符的当前执行顺序显示数据。您可以使用两个“显示数据”操作符（位于涉及的“数据”操作符的上方和下方）以查看“数据”操作符更改某些数据的方式。
  • 后事件中所有操作符执行之后显示数据值。

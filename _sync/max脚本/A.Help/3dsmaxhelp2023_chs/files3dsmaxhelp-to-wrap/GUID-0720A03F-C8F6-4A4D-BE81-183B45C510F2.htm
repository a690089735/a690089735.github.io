Interactive 明暗器的“标准基础”节点

“标准基础”节点是 Interactive 明暗器树的根，该节点提供明暗器的 DirectX 输出。

注意:在 3ds Max 2018.4 Update 中，Stingray 基于物理的明暗器被重命名为 Interactive。

Interactive“标准基础”节点接受不同贴图类型的输入。

  • 位置偏移：该材质的 UV 偏移。
  • 基础颜色：反射或漫反射贴图。
  • 不透明度
  • 不透明度阈值

    不透明度小于指定值的所有曲面点将被视为不存在。例如，值为 0.05 将移除透明度为 99.5% 或更高的所有曲面点。

  • 法线
  • 金属

    金属贴图是确定对象中哪些为金属区域以及哪些不是金属区域的遮罩。贴图中的值应为 0 或 1，非金属为 0，金属为 1。贴图值应表示材质的顶层；因此，绘制的金属不应设置为 1，因为绘制不是金属。

  • 粗糙度

    粗糙度贴图确定曲面的粗糙程度，及其反射率。粗糙度为 0 表示平滑曲面，即为完美的反射。如果增加粗糙度值，则反射变得更分散和更模糊。

  • 发射：这是辉光贴图。
  • 环境光阻挡
  • 折射
  • 密度

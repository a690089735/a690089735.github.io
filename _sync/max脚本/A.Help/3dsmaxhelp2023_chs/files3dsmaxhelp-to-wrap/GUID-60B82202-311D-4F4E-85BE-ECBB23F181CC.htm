“扭曲姿势”卷展栏

该卷展栏工具组能够使您创建并编辑 Biped 肢体的扭曲姿势。

  • 选择 Biped。 “运动”面板 “扭曲姿势”卷展栏

您可以使用卷展栏的预设姿势（您可以对其进行修改和重命名），或创建您自己的预设姿势。添加一个新姿势后，会在选定肢体的相对方向和卷展栏的 0 扭曲值（这表示没有扭曲）之间建立一个参考。当为一个旋转肢体设置了多个参考姿势时，要根据最近的姿势计算肢体的扭曲。

提示建议您为所有主要的肢体旋转姿势添加扭曲。这能够保证旋转 Biped 的肢体产生一个经过充分校准的扭曲。
注意:从技术上说，您没必要启用扭曲链接，来使用“扭曲姿势”卷展栏工具组。然而，如果 Biped 不包含扭曲链接，则不使用“扭曲和偏移”设置。

“扭曲姿势”卷展栏工具只影响有三种自由度（DOF）的肢体，如上臂和大腿，因为您可以通过旋转它们控制其扭曲链接。有两种自由度的肢体（如前臂和小腿）是不同的，因为如果您旋转它们的子肢体，只能控制其扭曲链接。

当前臂需要腕部扭曲它时，上臂可以扭曲它自身。

注意:如果您选择了有两种自由度的肢体，唯一可用的设置是“偏移”值；您可以改变它来调整扭曲链接上的旋转分布。

过程

示例：要“添加和编辑”扭曲姿势，请执行以下操作：

该步骤考虑了向 Biped 链接中添加扭曲链接的概念，并集中于使用“扭曲姿势”卷展栏上的工具来设置基本姿势。

  1. 提供一个在每个上臂上有五个扭曲链接的 Biped。
    提示您可以使用 Biped 肢体上的“透明”来更好地区分内部的扭曲链接。
  2. 退出 （“体形模式”），然后展开“扭曲姿势”卷展栏。
  3. 选择右上臂。

    该卷展栏控件被启用，因为您选择了有三种自由度的肢体。

  4. 使用 （“上一关键点”）和 （“下一关键点”）循环浏览不同的预设姿势。
    提示通常您可以使用这些预设作为设置肢体姿势的起点。
  5. 从下拉列表中选择姿势 6。

    Biped 的手臂向上伸展，并显示一些扭曲。

  6. 单击“设置关键点”。

    这将扭曲值 0 指定给了上臂的扭曲链接。

    注意:扭曲变更总是反映在两侧上（在这种情况下是两个上臂）。
  7. 围绕其 Y 轴局部 旋转上臂，使其向该侧伸展。
  8. 单击“添加”创建一个新的姿势。将其重新命名为侧臂。

    这个新姿势重设了扭曲链接的当前“扭曲”值。

  9. 围绕其 X 轴局部 旋转上臂。

    根据肢体保存姿势的接近度计算上臂中的扭曲。

    旋转上臂来验证扭曲链接。

界面

上一个/下一个关键点

滚动扭曲姿势列表并从中进行选择。

扭曲姿势列表

可以使您选择一个预设或保存姿势，应用到 Biped 选定的肢体中。默认情况下，有五个扭曲姿势可用于每个有三种自由度的肢体：上、前、侧、下和后。您还可以重命名当前的扭曲姿势。

扭曲

将所应用的扭曲旋转的数量（以度计算）设置给链接到选定肢体的扭曲链接。这样就影响了来自相反一侧的扭曲链接。默认值为 0。范围为 –180 至 180。

注意:改变“扭曲”值将把当前肢体的方向自动重设为活动的扭曲姿势。
偏移

沿扭曲链接设置旋转分布。设置为 1.0 将使扭曲向顶部链接集中，设置为 0.0 将使扭曲向底部链接集中。默认设置是 0.5，这时旋转均匀地分布在链接中。这样就影响了来自相反一侧的扭曲链接。

注意:改变“偏移”值将把当前肢体的方向自动重设为活动的扭曲姿势。
注意:您还可以为有两种自由度的肢体设置“偏移”值。
添加

根据选定肢体的方向创建一个新的扭曲姿势，并将“扭曲”和“偏移”重设为其默认值。

注意:为一个肢体添加一个新的姿势将自动使其用于相反一侧的肢体。
设置

用当前的“扭曲”和“偏移”值更新活动扭曲姿势。

删除

移除当前的扭曲姿势。

默认

用五个默认的预设姿势替换所有具有三种自由度的肢体的所有扭曲姿势。

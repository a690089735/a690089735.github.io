用户和系统路径

设置各种文件类型的系统路径。当为 3ds Max 编写脚本时，此对话框特别有用。

  • 默认菜单：“自定义”菜单 “配置用户和系统路径” “配置用户和系统路径”对话框 “用户和系统”面板
  • Alt 菜单：“自定义”菜单 “自定义” “配置用户和系统路径” “配置用户和系统路径”对话框 “用户和系统”面板

启动 3ds Max 时，它会检查这些文件夹，然后运行或使用相应的文件：

文件类型 功能
其他图标 其他图标的路径
其他宏 其他宏的路径
其他脚本 其他脚本的路径
其他启动脚本 其他启动脚本的路径
硬件明暗器缓存 硬件明暗器的缓存
页面文件 自动位图分页的路径
临时 临时文件的路径
提示“页面文件”条目默认为您的用户设置（\users\<用户名>\appdata\local\autodesk\3dsmax\temp ）中的子文件夹 \temp。 但是，如果计划进行大量渲染工作，最好是将此路径重新分配给可用作位图缓存的暂存磁盘的专用、高性能驱动器。

要启用或禁用位图页面设置，请使用“资源追踪”对话框 “位图性能和内存”菜单，或“位图代理的全局设置和默认设置”对话框。

过程

要修改路径，请执行以下操作：

  1. 单击路径项以高亮显示它。
  2. 单击“修改”。
  3. 使用“选择目录”对话框执行以下操作之一：
    • 在“路径”字段输入路径。
    • 进行浏览，以查找路径。
  4. 单击“使用路径”。

    新的路径将立即生效。

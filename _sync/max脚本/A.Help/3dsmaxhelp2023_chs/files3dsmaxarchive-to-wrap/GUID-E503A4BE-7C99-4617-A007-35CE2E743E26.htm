曲面到达行为

“曲面获得”行为与“查找”行为相类似，它可用给代理指定一个或多个静态对象或移动对象。主要区别是“曲面获得”可以使代理到达目标时停止运动。在某种程度上可以指定代理在对象的何处停止运动，及代理在停止之前如何到达目标。

  • “创建”面板 （辅助对象） “对象类型”卷展栏 “群组” “设置”卷展栏 “行为”组 单击“新建”按钮。 “选择行为类型”对话框 “曲面获得行为”
  • 选择群组对象。 “修改”面板 “设置”卷展栏 “行为”组 单击“新建”按钮。 “选择行为类型”对话框 “曲面到达行为”
  • 选择群组对象。 “修改”面板 “设置”卷展栏 “行为”组 从下拉列表中选择“曲面到达”。

一个使用此行为的例子是，一群鸟飞过一排电线杆，然后每一只鸟降落在不同的电线杆的顶部。

界面

[目标按钮]
指定单个目标。单击该按钮，然后在视口中点击目标对象。目标名称会出现在按钮上。在选择目标之前，标签显示为“无”。

如果使用“多个选择”选择了多个目标（请参见下一条），按钮上会出现“多个”字样。要查看哪些对象被指定为目标，单击多个选择按钮。

多个选择
打开选择对话框来指定多个目标。

当拥有多个目标时，可以设置代理移向组中与之最近的目标，或计算目标的平均位置。

“到达”组

指定“曲面到达”行为的三个方面：速率、速度和距离。

到达后禁用
此选项打开时，在代理到达曲面后关闭“曲面获得”行为。默认设置为启用。
速率
多个代理的最大加速度设置用来指定代理所能达到的加速度。值为 1.0 意味着使用代理的完整加速度。默认值为 0.5。
偏离
添加随机变化到“速率”设置中。

“偏离”设置值乘以 －1 与 ＋1 之间的一个随机数，然后用此结果乘以“速度”设置值，就得出实际偏离值。默认值为 0.0。

速度
即到达速度，与目标速度相关。默认值为 0.0。
偏离
添加随机变化到“速度”设置中。默认值为 0.0。

“偏离”设置值乘以 －1 与 ＋1 之间的一个随机数，然后用此结果乘以“速度”设置值，就得出实际偏离值。

距离
离开目标的最大半径距离，行为在此目标上处于活动状态。默认值为 9999999.0。

只要代理在此半径范围内，行为不受影响。

偏离
添加随机变化到“距离”设置中。

“偏离”设置值乘以 －1 与 ＋1 之间的一个随机数，然后用此结果乘以“距离”设置值，就得出实际偏离值。默认值为 0.0。

“位置”组

偏移
基于曲面法线指定离计算到达点的合适距离，此距离为代理所使用。默认值为 0.0。
正面
此选项打开时，代理仅试图到达面向此代理曲面上的三角形上的点。默认设置为禁用状态。
  • 随机（默认设置。）character studio 在目标曲面上随机选择一个点作为到达点。

    当使用“随机”选项时，character studio 仅在模拟开始时给代理选择一次到达点。

  • 最近character studio 选择目标曲面上最近的点作为到达点。

    如果“最近”被选中但“每个帧”处于禁用状态，那么 character studio 仅在模拟开始时为代理选择一次到达点。

    每帧启用该选项之后，character studio 在每个帧为代理选择到达点。它在“最近”被选中时可用。默认设置为禁用状态。

    目标对象在动画期间进行旋转时，“每个帧”都是有用的，但需要较多的计算时间。

显示偏移
当打开该选项时，用从每个曲面对象的顶点发出的线段显示偏移距离，这些线段都垂直于曲面。

“处理方法”组

同时使用“高度”和“下降”为代理指定运动路径。它允许使用范围很广的行为，从软下降、渐进着地到直接直升机型下降。

在这两种情况下，“偏离”设置值乘以 －1 与 ＋1 之间的一个随机数，然后用此结果乘以相关值，就得出实际偏离值。

高度
指定沿着面法线距到达点的距离。

此点是代理在下降到到达点之前第一个到达的点。

偏离
添加随机变化到“高度”设置中。

“偏离”设置值乘以 －1 与 ＋1 之间的一个随机数，然后用此结果乘以“高度”设置值，就得出实际偏离值。

下降起点
指定代理与到达点之间的距离，代理在此点开始下降。
注意:注意“下降起点”值应当设置得足够高，以使代理在下降过程中不会飞过目标，这是因为相对于代理开始下降的时刻，它的速度太大并且减速度太小。
偏离
添加随机变化到“下降起点”设置中。

“偏离”设置值乘以 －1 与 ＋1 之间的一个随机数，然后用此结果乘以“下降起点”设置值，就得出实际偏离值。

偏移此法线
此选项打开时，可以设置一个方向向量来指定产生最终方向向量的角。默认设置为禁用状态。

X/Y/Z使用这些微调器可在世界坐标中指定最终方向向量。例如，由默认设置 X=0、Y=0、Z=1 指定的向量意味着代理沿世界纵轴到达目标。

种子

影响用来计算“偏离”设置值的随机数。

为在不同的“曲面获得”行为中获得相似随机选择，请使用同一“种子”值。

[目标图标色样]

显示用来描绘目标图标的颜色。默认情况下为深蓝色。

显示目标启用显示目标图标，在求解过程中当为代理计算一个新过渡目标时显示此图标。在使用“处理方法”设置时创建了过渡目标。默认设置为启用。

目标缩放
指定目标图标的全部尺寸。默认值为 5.0。

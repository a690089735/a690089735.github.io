从 Vault 打开

使用“从 Vault 打开”命令可以直接从 Autodesk Vault（3ds Max 附带的数据管理提供程序）打开 MAX 文件。

  • “文件”菜单 “打开” “从 Vault 打开”

此选项可以对创建数字内容时使用的资源进行安全控制和版本管理，而无需使用 Vault 客户端。选择用于存储场景的 Vault 版本。

如果您没有设置工作文件夹，3ds Max 会在您尝试从 Vault 打开文件时显示一条警告。解除警告后，将打开“浏览文件夹”对话框，您可以使用它指定一个工作文件夹。

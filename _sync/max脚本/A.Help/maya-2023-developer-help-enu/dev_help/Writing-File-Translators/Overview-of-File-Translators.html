Overview of File Translators

File Translators can be created with a proxy class provided by the Maya API. Translators allow many types of information to flow in and out of Maya, which can create a flexible production pipeline.

With a translator, you can set your own file extension. This increases the flexibility of Maya to be used with third party applications and file formats.

With a registered translator plug-in, the new file type appears in the list of file types when saving, opening, importing and exporting files. Moreover, you can create an option box with MEL scripts to customize options for the translator.

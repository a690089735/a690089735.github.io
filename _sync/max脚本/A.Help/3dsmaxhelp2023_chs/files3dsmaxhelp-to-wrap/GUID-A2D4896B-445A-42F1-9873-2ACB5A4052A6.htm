使用 AutoCAD、Revit 和 AutoCAD Architecture

3ds Max 提供了旨在简化设计可视化工作流程的大量功能。文件链接管理器使您可以创建到图形（DWG、DXF）、Revit (RVT) 或 FBX 文件的活动链接。然后您可以在 3ds Max 中应用材质和动画，但如果原始文件中有任何更改，仍然会更新几何体。

几何体文件格式

使用 Max Interactive 实时引擎

通过与 3ds Max Interactive 的实时链接，专业和初级开发人员可以使用与 3ds Max 的直接连接更加轻松地将内容导入到 Interactive 实时引擎中。有关详细信息，请参见与 3ds Max Interactive 链接。

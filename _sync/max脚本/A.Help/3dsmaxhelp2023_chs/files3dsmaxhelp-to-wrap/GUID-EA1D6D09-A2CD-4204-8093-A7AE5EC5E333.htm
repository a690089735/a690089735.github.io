CAT (Character Animation Toolkit)

CAT (Character Animation Toolkit) 是 3ds Max 的角色动画插件。CAT 有助于角色装备、非线性动画、动画分层、运动捕捉导入和肌肉模拟。

本节内容

  • 入门：使用 CATRig 装备

    CATRig 是定义 CAT 骨骼动画系统的层次。它是一个快速、复杂而又灵活多变的角色装备，旨在用于创建您需要的角色，而不必编写脚本。

  • 使用 CAT 设置动画

    CAT 的 FK/IK 装备操纵系统使您只需执行简单的推拉操作即可将装备部位置于所需姿势，而不管它们是采用 IK 还是 FK。对于行走循环序列，CATMotion 允许您创建完全自定义的行走循环，并围绕场景指挥角色，而无需安排每个足迹。足迹冲突检测起到了锦上添花之效。

  • 使用姿势和动画

    CAT 提供了用于处理运动数据的工具，包括导入、贴图、重定位和编辑。可以保存单个动画层甚至整个层堆栈供重用，包括用于其他角色。

  • CATMotion

    CATMotion 是 CAT 的程序运动循环生成系统。使用 CATMotion 可以创建与装备的速度和方向相适应的运动循环而不会出现滑动足迹，即使是在不平坦的表面上移动时也是如此。

  • 使用运动捕捉数据

    将运动捕捉数据加载到层管理器中的方法与加载任意其他关键帧数据的方法大致相同。加载运动捕捉数据之后，可以使用相同的关键帧编辑工具对运动数据进行转换、编辑、复制和调整时间。然后可以使用所有分层工具以非破坏性的方法来编辑运动捕捉数据。

  • 处理肌肉

    在 CAT 中，肌肉属于非渲染辅助对象，可用于在拉伸和变形时创建看似逼真的皮肤。

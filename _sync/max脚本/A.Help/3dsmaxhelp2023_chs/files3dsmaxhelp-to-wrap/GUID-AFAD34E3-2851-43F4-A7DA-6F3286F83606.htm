摄影表简介

“摄影表”编辑器使用“轨迹视图”来在水平图形上显示随时间变化的动画关键点。这以图形的方式显示调整动画计时的简化操作，因为可以在一个类似电子表格中看到所有的关键点。“轨迹视图”的另外一种模式是曲线编辑器，在曲线编辑器中，除了可以处理关键点以外，还可以处理插值曲线。

  • “轨迹视图” “编辑器”菜单 “摄影表”
  • 右键单击在视口 “摄影表”中选中的对象。
  • 默认菜单：“图形编辑器”菜单 “轨迹视图 - 摄影表”
  • Alt 菜单：“动画”菜单 “运动编辑器” “轨迹视图 - 摄影表”

经典动画技术使用曝光表，也称为“X”表或摄影表。此垂直图表作为摄影机操作人员的说明进行提供。在代表每一快照的编号列表中指示对话和摄影机动作，从而成为动画影片的单个照片帧。经典曝光表也包含用于在背景上合成动画角色的 cel 绘画的说明。此设备作为 3ds Max 中“摄影表”工具中的奇妙工具。

通常使用“摄影表”来交错角色的肢体移动，以便它们不会同时移动。如果拥有群体角色，可以使用“摄影表”来切换移动，防止它们一致移动。

注意:如何在“关键点”窗口中进行缩放和平移取决于处于活动状态的交互模式（3ds Max 或 Maya）。

“摄影表”菜单和工具

“摄影表”编辑器与经典的 X 表相类似。它仅用水平图（而不是垂直图）显示超时帧。您可以在一个类似电子表格的界面中看到所有关键点。可以选择场景中任意或所有的关键点，进行缩放、移动、复制和粘贴，以及以其他方式直接在此工作，而无需在视口中工作。可以选择为子对象或子树选择关键点，因此可以产生简单的变化同时影响很多对象和它们的关键点。

“摄影表”允许软选择关键点，这在使用在每一帧上具有关键点的运动捕获数据时相当有用。

“摄影表”提供用于直接使用时间的工具。使用“时间”菜单上的工具可以选择、剪切、复制、粘贴和插入和颠倒时间。

如曲线编辑器，“摄影表”提供了快速访问工具的菜单栏和工具栏。

“摄影表”菜单栏

与“曲线编辑器”不同，摄影表有两种模式：“编辑关键点”和“编辑范围”。这些模式可影响“关键点”窗口中的显示。

编辑关键点模式

“编辑关键点”处于活动状态时，关键帧以矩形内的长方体显示在栅格上。代码带有颜色的关键点显示哪一帧被设置为关键帧（位置是红色、比例是黄色、旋转是绿色，等等。）

带有颜色的关键点

“摄影表”关键点现在以框内的矩形显示，所以可以容易的快照子帧关键点，关键点在两帧之间。充满框的关键点在帧上，小矩形的关键点是子帧。

子帧关键点

与“曲线编辑器”类似，“摄影表”允许使用关键点之上的软选择。这在处理大量关键点时极其有用。例如在运动捕获数据文件中。将它与缩放关键点结合来操纵运动数据。

“摄影表”关键点的软选择。

编辑范围模式

“编辑范围”处于活动状态时，动画轨迹会以范围栏显示出来，且单独的关键点均不可见。当您仅要更改操作执行时间或操作开始和结束的时间，而不是动画范围内的特定关键点时，请使用此模式。

编辑范围模式

修改子树与修改子关键点

在“摄影表”模式下工作时，您可以切换修改子树和修改子关键点。这用来分别自动移动子树的轨迹和子对象的关键点。如果在使用“摄影表”时体验一下减速，请尝试禁用这些选项并手动移动关键点。在“摄影表”中，“修改子树”默认为启用，但“修改子对象关键点”则为禁用。

时间编辑

“摄影表”提供一系列直接使用时间的工具。可以选择一段时间，这段时间包含该时间段内的所有关键点，此后在这段时间内执行不同的操作。可以复制和粘贴时间到循环动画中，或颠倒时间使动画反向播放。可以插入时间来添加一段空白到动画中，或删除时间来缩短动画。

注意:为获得“摄影表”程序，请查看链接下的独立工具和菜单选项。

摄影表显示优化

仅计算在视图的当前域中显示的轨迹，从而优化“摄影表”编辑器的响应性。

“摄影表”暂时保持关键点缓冲区不变，所以用以计算的轨迹作为要计算的父对象的结果，而不必重新计算。使用缓冲区来代替。

“摄影表”编辑器的默认自动导航设置只能自动扩展到当前选定对象的节点轨迹。这样不仅能够需要显示其关键点的轨迹数量，而且还有助于将“摄影表”编辑器设计为强制执行从上到下的工作流。

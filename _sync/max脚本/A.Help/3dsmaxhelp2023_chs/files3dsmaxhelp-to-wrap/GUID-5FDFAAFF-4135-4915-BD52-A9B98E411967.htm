标记文件

来自运动捕捉设备的文件。Character Studio 可读取两种标记文件格式：

  • CSM（character studio 标记）

    这是 character studio 的原生标记文件格式。

    也可以将 TRC（运动分析运动捕捉）文件导入到场景，然后使用 MACUtilities 工具将其转换为 CSM 格式。

  • BVH（BioVision 层次数据格式）

    这是行业标准格式。

生成网格

任何液体模拟的表面都可以表示为标准多边形网格。只要创建模拟就会自动添加到场景，在对当前帧解算或自动生成组件之前，网格没有多边形。解算网格通常是渲染或导出之前的最后一步。

提供了以下两种网格：

  • “缓存网格”是整个模拟的已解算、已缓存的网格。如果要将模拟数据保存到 Alembic (.abc) 文件，必须首先生成缓存网格。
  • “动态网格”在当前帧上自动生成。动态网格对于预览您的工作很有用，可以查看在任何一帧上网格中显示细节的程度，而不必计算整个模拟的完整网格。此外，动态网格与任何渲染器都兼容。

在“模拟视图”窗口 >“显示设置”>“液体设置”>“显示类型”中选择网格类型。也可以选择显示点、平面、球体或自定义对象。

注意:将多个复杂对象用作网格可创建极大的文件，这可能会导致系统不稳定。

操作步骤

要生成缓存网格，请执行以下操作：

  1. 创建液体模拟。
  2. 在“模拟视图”窗口 >“管理系统区域”中，解算液体 和/或泡沫 组件。
  3. 在“模拟视图”窗口 >“显示设置”>“液体设置”>“显示类型”中，选择“Bifrost 缓存网格”。
  4. 在“模拟视图”窗口 >“管理系统区域”中，对已缓存的液体和泡沫组件禁用解算，并启用网格组件 。
  5. 运行结算器 。

    缓存网格将生成并可用于导出。

要生成动态网格，请执行以下操作：

  1. 创建液体模拟。
  2. 在“模拟视图”窗口 >“管理系统区域”中，解算液体 和/或泡沫 组件。
  3. 在“模拟视图”窗口 >“显示设置”>“液体设置”>“显示类型”中，选择“Bifrost 动态网格”。
  4. 在视口中，转到模拟的任何帧。

    将为该帧自动生成网格。

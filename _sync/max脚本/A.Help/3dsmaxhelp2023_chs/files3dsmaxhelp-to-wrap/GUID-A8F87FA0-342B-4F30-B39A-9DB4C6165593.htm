“从气候数据中选一时间周期”对话框

“从气候数据中选一时间周期”对话框可与“配置气候数据”对话框一起运行。使用它，您可以从气候数据 (EPW) 文件中选择一个时间周期。

  • 创建日光系统。 “创建”面板或 “修改”面板 “日光参数”卷展栏 “位置”组 选择“气候数据文件”。 单击“设置”。 “配置气候数据”对话框 单击“更改时间周期”。 “从气候数据中选一时间周期”对话框
  • 创建日光系统。 “运动”面板 “控制参数”卷展栏 选择“气候数据文件”。 单击“设置”按钮。 “配置气候数据”对话框 单击“更改时间周期”。 “从气候数据中选一时间周期”对话框

界面

选定的时间周期
显示当前选定的时间周期。默认值为气候数据文件中的第一个周期。

从气候数据文件中选择时间周期的范围

周期选择器
拖动此滑块，可浏览文件中可用的时间周期。
增量周期选择器
使用月、日、小时或分钟微调器控件，可按照相应的时间单位移动周期选择器。如果文件不包含该时间级别上的增量，则微调器不会生效。

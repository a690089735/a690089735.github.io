STL 检查修改器

“STL 检查”修改器用以检查对象，以查看是否能正确地导出到 STL（stereolithography）文件格式。Stereolithography 文件为特殊的机器所使用，以基于 STL 文件中的数据来产生原型物理模型。

  • 选择对象。 “修改”面板 “修改器列表” “STL 检查”
  • 默认菜单：选择对象。 “修改器”菜单 “网格编辑” “STL 检查”
  • Alt 菜单：选择对象。 “修改器”菜单 “几何体(转换为网格)” “STL 检查”

要创建物理模型，STL 文件必须要有一个完全且闭合的曲面。如果该文件用来创建物理模型，那么可以在导出之前用“STL 检查”来测试几何体，这样可以节省时间和金钱。

“STL 检查”错误：

1. 开放边

2. 双面

3. 钉形

4. 多个边

过程

要检查对象的 STL 兼容性，请执行以下操作：

  1. 选择对象，然后在“修改”面板 “修改器列表”上，选择“网格编辑”“STL 检查”。
  2. 启用“检查”。

    如果发生了错误，“状态”组中的消息就会进行显示。“STL 检查”可以通过选择问题几何体，也可以通过给它指定一个特殊材质 ID，或同时使用这两种方法来指出错误。

界面

“错误”组

选择这些选项中的其中一个，可以选择特定于该选择的错误几何体，并根据在“选择”组中所选的选项进行选择。

  • 开放边检查开放边（洞）。
  • 双面检查共享相同 3D 空间的面。
  • 钉形检查钉形，它是只与对象共享一个边的孤立面。
  • 多重边检查共享多个边的面。
  • 所有内容检查上述所有项。
提示当检查全部耗费的时间最长时，如果要使用 STL 文件来生成物理模型，推荐使用该选项。

“选择”组

这些选项基于“错误”组中的设置，指定了选定错误几何体的级别。

  • 不选择如果选择该选项，则“STL 检查”不会选择对象中任何错误的部分。
  • 选择边如果选择该选项，则“STL 检查”将通过选择面中的错误边来对该边进行标记。错误边的选择在视口中不可见。
  • 选择面如果选择该选项，则“STL 检查”通过选择对象中的错误面来对该面进行标记。错误面的选择在视口中不可见。
更改材质 ID

启用（默认情况下）此选项后，STL 检查还会标记错误面，方法是为这些面指定唯一的 ID。使用微调器可选择 STL 检查使用的材质 ID 的值。

检查

启用该选项可以执行 STL 检查。对于复杂的模型，希望能在启用此选项时可以暂停，并能在“状态”组中显示看到报告错误的时间。默认设置为禁用状态。

“状态”组

在启用“检查”后，显示错误数。

提示如果禁用了“选择边”，可以在“面”子对象层级上应用编辑网格修改器并按材质 ID 进行选择，来查看错误的面。也可以将多维/子对象材质应用到对象以查看发生错误的地方。

“液体参数”卷展栏

“预设”组

预设
加载、保存和删除预设液体参数。列表包括以下预设：啤酒、血液、焦糖、奶油、蜂蜜、番茄酱、乳胶漆、枫糖浆、水银、牛奶、机油、橄榄油、橙汁、油灰、硅树脂、牙膏、传输流体和水。

单击 可加载下拉列表中当前选定的预设。单击 可采用新的预设名称保存当前设置，单击 可删除选定的预设。

注意:不能删除提供的任何预设。

“水滴”组

水滴是模拟中与液体主体分离的粒子。水滴有几种用途：

  • 断开飞溅和成片的流体，以获得更像喷射的效果。
  • 降低与主液体实体分离的粒子导致的内存使用量和计算时间。水滴需要更简单的计算（涉及弹道运动而不是液体动力学）。
  • 更正液体曲面中由于粒子过早地重新加入主体而导致的凹凸和凹痕等问题。
阈值
设置粒子转化为水滴时的阈值。值越小，创建的水滴越多，因而会更快地分解图纸和卷须。要防止产生水滴，请将阈值设置为 1.0 或更大值。
并回深度
设置在重新加入液体并参与流体动力学计算之前水滴必须达到的液体曲面深度（以体素宽度为单位）。此设置可以减少液体曲面中由于粒子过早地重新加入主体而导致的凹凸和凹痕。

“粒子分布”组

此组控制曲面上和液体内部的每体素粒子数，以及曲面层的深度。通常，粒子数越多，细节就越多，内存需求也就越高且计算越慢。通常，内部体素需要的粒子数比曲面处少，因为要对曲面进行渲染或网格化。

曲面带宽
设置液体曲面的宽度，以体素为单位。
内部粒子密度
设置液体整个内部体积中的粒子密度。
曲面粒子密度
设置液体曲面上的粒子密度。您可以使用此值生成密度较大的粒子分布，同时在液体曲面上显示更多细节，而对细节没有要求的内部无需提供额外粒子。请勿使用小于 1.0 的值。
提示如果遇到过量的体积，特别是在非常易溅的模拟中，请将“内部粒子密度”和“曲面粒子密度”都设置为 1.0。

“漩涡”组

启用
启用漩涡通道的计算。这是体素中旋转幅值的累积。漩涡可用于模拟涡流。
衰退
设置从每一帧累积漩涡中减去的值。
倍增
设置当前帧卷曲幅值在与累积漩涡相加之前的倍增。
最大值
设置总漩涡的钳制值。

“曲面张力”组

启用
启用曲面张力。
曲面张力
增加液体粒子之间的吸引力，这会增强成束效果。

“粘度”组

粘度
控制流体的厚度。例如，水的粘度低（约 0.001），而蜂蜜的粘度高。使用值为 0.0 的水可以避免不必要的计算。
比例
将模拟的速度与邻近区域的平均值混合，从而平滑和抑制液体流。这在物理上是不正确的，但与使用“粘度”设置相比，可以用于以很少的计算成本来模拟粘度。在模拟期间，大于 2.0 的值可能会产生无法预测的结果。

“腐蚀”组

以下选项控制流体边界收缩回到粒子位置的接近程度，这有助于解决静止、有轻微涟漪或有剧烈湍流的流体实体上不希望得到的效果。值小于 0.0 或大于 1.0 时将不会产生任何效果。

因子
控制流体曲面的腐蚀量。0.0 值为无腐蚀，1.0 将腐蚀整个粒子半径。

较低的值（0.0 到 0.2）有助于贴附模拟中的曲面，例如水面平静的池塘。较高的值（0.7 到 0.9）有助于优化波浪和船尾迹的外观，并避免偏离正道的粒子在整个流体曲面中“漫游”。

接近实体的因子
确定流体曲面是否基于碰撞对象曲面的法线，在接近碰撞对象的区域中腐蚀。请注意，无论该曲面在何处腐蚀，它始终以完整因子数量腐蚀。

“接近实体的因子”设置将影响腐蚀情况，如下所示：

  1. 设为 0.0 时，碰撞对象附近不发生腐蚀。
  2. 设为 0.5 时，仅曲面法线向下倾斜的碰撞对象附近的流体曲面会腐蚀。这有助于缓解液体粘滞到碰撞对象底面的问题，同时仍允许碰撞对象顶部存在法线向上倾斜的细流。
  3. 设为 1.0 时，流体曲面各处均腐蚀。

    可以根据需要在 0.0 到 1.0 之间更改此值以微调行为。较低的值通常用于改进会穿过碰撞对象的浅水流的外观，并缓解碰撞对象壁附近存在的排斥、成束或攀爬等问题。而较高的值用于避免阻尼飞溅。

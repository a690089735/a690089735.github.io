Autodesk 金属

此材质具有金属的外观。

  • 材质/贴图浏览器 材质 Autodesk Autodesk 金属
注意:仅当活动渲染器支持时，该材质才会显示在“材质/贴图浏览器”中。

应用于器具且类型为不锈钢的 Autodesk 金属，以及应用于椅子构架且类型为黄铜的 Autodesk 金属

界面

“金属”卷展栏

类型

设置材质的基础颜色和纹理，并控制使用哪些属性。

  • 铝（默认设置。）对于铝，您可以更改抛光。
  • 阳极氧化铝对于阳极氧化铝，您可以更改颜色。
  • 铬对于铬，您可以更改抛光。
  • 铜对于铜，您可以更改绿锈和抛光。
  • 黄铜对于黄铜，您可以更改抛光。
  • 青铜对于青铜，您可以更改绿锈和抛光。
  • 不锈钢对于不锈钢，您可以更改抛光。
  • 锌对于锌，系统未提供进一步的选项。
颜色（仅对于“阳极氧化铝”才显示）

用于设置阳极氧化铝的自定义颜色。

绿锈（仅对于“铜”和“青铜”才显示）

用于设置铜或青铜的绿锈。值越大，绿锈的量就越多。范围在 0 到 1.0。默认值为 0.0。

磨光（对于“阳极氧化铝”或“锌”不显示）

用于设置铝、铬、铜、黄铜、青铜或不锈钢的曲面抛光。

  • 抛光（默认设置。）
  • 半抛光
  • 缎纹
  • 拂刷

“浮雕图案”卷展栏

启用

启用后，系统会对材质应用浮雕图案。默认设置为禁用。

类型

用于选择预定义的浮雕图案，或创建自定义图案。

  • 滚花（默认设置。）
  • 花纹板
  • 网纹板
  • 自定义 - 图像可选择用于控制浮雕图案的位图。
图像（仅对于“自定义 - 图像”才显示）

当“自定义 - 图像”为活动“类型”时，单击可为浮雕图案指定位图。

数量

控制浮雕的高度。范围在 0.0 到 2.0 之间。默认值为 0.3。

缩放

调整浮雕图案的比例。范围在 0.0 到 5.0。默认设置为 1.0。

“裁切”卷展栏

启用

启用时，将为该材质启用裁切贴图。默认设置为禁用。

类型

用于选择预定义的裁切图案，或创建自定义图案。

对于可调整尺寸的裁切图案，值是以英寸为单位进行设置的。

  • 交错排列的圆形（默认设置。）对于“交错排列的圆形”图案，您可以设置“直径”及“中心间距”距离。
  • 直线排列的圆形对于“直线排列的圆形”图案，您可以设置“直径”及“中心间距”距离。
  • 正方形对于“正方形”图案，您可以设置“大小”及“中心间距”距离。
  • 希腊式对于“希腊式”图案，没有其他的设置。

    “希腊式”图案是 0.625" 的方形，而这些方形由 0.125" 分隔。

  • 四叶式交叉对于“四叶式交叉”图案，没有其他的设置。

    “四叶式交叉”高度（在每叶的中间凹入处）为 0.25"，宽度（从一叶的顶点到相对叶顶点之间的距离）为 0.5"

  • 六边形对于“六边形”图案，没有其他的设置。

    “六边形”高度（长尺寸）为 0.3125"，宽度（窄尺寸）为 0.25"。

  • 自定义对于“自定义”图案，利用“图像”按钮可选择用于生成裁切的贴图。默认的贴图类型是 Autodesk 位图，但您可以使用标准材质编辑器方法选择不同的类型。

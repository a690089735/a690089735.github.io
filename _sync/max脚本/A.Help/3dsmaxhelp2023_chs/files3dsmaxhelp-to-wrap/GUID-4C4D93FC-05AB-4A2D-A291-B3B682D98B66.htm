Biped 键盘快捷键

本主题中的表格显示了 character studio 的默认键盘快捷键。

使用主工具栏上的键盘快捷键覆盖切换来启用 character studio 键盘快捷键。

当“运动”面板和“键盘快捷键覆盖”按钮处于活动状态时，所有 character studio 键盘快捷键也处于活动状态。

要创建或编辑快捷键，请使用“自定义用户界面”对话框。

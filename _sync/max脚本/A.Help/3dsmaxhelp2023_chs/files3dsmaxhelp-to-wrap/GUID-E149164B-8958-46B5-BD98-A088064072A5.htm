Visual MAXScript 工具（参见 MAXScript 帮助）

Visual MAXScript 是 3ds Max 脚本语言的强大接口，它使 MAXScript 功能更容易学习和使用。使用 Visual MAXScript，您可以迅速创建脚本的 UI 元素和布局。

  • 默认菜单：“脚本”菜单 “Visual MAXScript 编辑器”
  • Alt 菜单：“脚本”菜单 “脚本” “Visual MAXScript 编辑器”
  • “实用程序”面板 “实用程序”卷展栏 “更多”按钮 Visual MAXScript

有关 Visual MAXScript 的详细信息，请打开“MAXScript 帮助”（位于“帮助”菜单 “MAXScript 帮助”中）。

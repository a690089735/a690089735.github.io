附加功能和限制（Quicksilver 渲染器）

Quicksilver 如何处理 3ds Max 的某些其他功能。

功能 功能和限制
渲染元素 支持以下 Alpha 和 Z 深度元素：
  • Alpha
  • 材质 ID
  • 对象 ID
  • Z 深度
渲染到纹理 不受支持
对象属性设置 Quicksilver 渲染器会忽略这些设置，但 3ds Max 会将“可渲染”和“可见性”属性传递给渲染器，因此这些设置可以按预期工作。

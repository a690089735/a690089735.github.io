正交捕捉模式

在绘制线形样条线或墙对象时，正交捕捉模式将线的创建限制在相对活动栅格水平或垂直的方向。也就是说，启用“正交”模式后，只能绘制与活动栅格线平行的线。

  • 默认情况下，正交捕捉模式不能在 3ds Max 用户界面中直接访问。要将它添加为“捕捉”工具栏上的按钮，请参见下面的“步骤”部分。

对于绘制所有线条必须相互垂直的平面图（譬如房屋平面图），正交捕捉模式非常有用。若要将线条绘制约束到其他角度增量，请使用极轴捕捉模式。

以下规定适用于使用正交捕捉模式：

  • 使用正交捕捉模式与使用极轴捕捉模式相互排斥；每次只能有一种模式处于活动状态。
  • 正交捕捉模式主要适用于线形样条线和墙对象的创建。它也可用于其他对象的创建，但是结果不是很理想。

正交模式将显示指南针，可以提供输入相对于局部 X 轴正向的当前角度的读数。

注意:如果您在正交模式下按住 Alt 键，下一个点将同时受到当前对象的上一个点和第一个点的限制。这使您能够准确地关闭样条线。在此模式下，显示两个指南针，第一个点和 上一个点上分别显示一个。

过程

要将正交捕捉模式添加到“捕捉”工具栏，请执行以下步骤：

  1. 如有必要，打开“捕捉”工具栏。要执行此操作，右键单击主工具栏的空白部分（如下拉列表的正下方部分），并选择“捕捉”。
  2. 向右拖动工具栏的右边缘，以便为新按钮腾出空间。
  3. 选择“自定义”菜单 “自定义用户界面”。

    此操作会打开“自定义用户界面”对话框。

  4. 在对话框上，单击“工具栏”选项卡。
  5. 将对话框中的“操作”列表下滚至“正交捕捉模式”项。通过单击列表中的任一项并在键盘上按下 O 键，可以直接跳转到首字母为 O 的部分。
  6. 将“正交捕捉模式”项从列表中拖到“捕捉”工具栏的空白部分。此操作将在工具栏中添加按钮。
  7. 关闭“自定义用户界面”对话框。

    3ds Max 将自动以其修改后的状态保存该工具栏，并使新按钮成为用户界面的永久部分。

为使用正交捕捉模式，请执行以下操作：

  1. 启用 （“正射捕捉模式”）。
  2. 开始绘制一条线形样条线。

    在放置 第一个点的位置处会出现橘黄色的指南针，上面显示的红色数字表示当前线段相对于局部 X 轴正向的角度。

  3. 在视口中移动鼠标光标。

    线条在局部 X 轴上跳转至 90 度角度增量，同时，指南针上的读数也随之更新，以显示当前角度。

  4. 单击以放置下一个顶点。
  5. 重复步骤 3 和 4，直至您做好完成图形的准备。
  6. 执行下列任一操作：
    • 在任意处单击右键，从而无需关闭样条线即可在最近的顶点完成图形。
    • 要通过关闭样条线完成图形，可将鼠标光标放至接近第一个点处然后单击。会打开一个小对话框，询问您是否希望关闭样条线，单击“是”。

转换场景

使用“场景转换器”快速更改灯光、材质和渲染器功能。

  • 默认菜单： 菜单栏 “渲染” “场景转换器”
  • Alt 菜单： 菜单栏 “渲染” “工具(渲染设置)”￼ “场景转换器”
  • Alt 菜单： 菜单栏 “场景” “工具(场景设置)” “场景转换器”

首次使用场景转换器时，它会使用称为 ART 的转换预设文件打开。它包含使用 Autodesk ART 渲染器所需的大多数转换规则，并转换基于物理的工作流所需的功能。

在将来，场景转换器会记住上次使用的转换预设，并自动将其加载为默认设置。

过程

转换场景

要转换一个或多个场景，请执行以下操作：

  1. 执行下列操作之一：
    • 要创建新的转换预设，请单击“创建新的预设文件”。
    • 要使用现有的转换预设，请执行以下操作：
      1. 单击“打开预设文件”。
        注意:可以在 C:ProgramData\Autodesk\ApplicationPlugins\MAXtoA\Scene Converter 中找到 Arnold 渲染器的预设。
      2. 浏览并选择先前定义的预设文件。
      3. 单击“打开”。
      4. 根据需要修改预设。
  2. 根据需要定义转换规则。
  3. 在“选项”面板中，定义设置。
  4. 单击“转换场景”。

    如果要使用渲染预设转换场景，则会显示一个对话框，提示您选择要合并的类别。右下角将显示进度条。将创建当前文件的备份，预设中的转换规则将应用到场景。

    注意:处理时间取决于场景的复杂性和大小。

定义转换规则

要修改转换预设，请执行以下操作：

  1. 在场景转换器中打开预设文件。
  2. 在“转换规则”组中，执行以下操作之一：
    • 选择“展开全部”。
    • 手动展开树视图。
  3. 按 Ctrl 键并单击要更改的转换规则。
  4. 执行下列操作之一：
    • 要仅使用选定的规则并忽略预设中的所有其他规则，请单击“仅应用选定项”。
    • 要从预设中移除规则，请单击“删除选定项”。

要定义渲染器，请执行以下操作：

  1. 选择“编辑器”面板。
  2. 从“将渲染器更改为”列表中选择渲染器。
    注意:只能选择已安装的渲染器，并且只能对已安装的功能运行转换。
  3. 从“使用渲染预设”列表中选择一个渲染预设。
    注意:可以在“渲染设置”对话框中创建渲染预设。对于要使用的当前项目，预设文件必须位于 RenderPresets 文件夹中。

要添加规则，请执行以下操作：

  1. 选择“编辑器”面板。
  2. 在“源”组中，执行以下操作之一：
    • 选择“展开全部”。
    • 手动展开树视图。
  3. 选择特定的源（要转换的内容），然后在“目标”组中选择要将源转换到的内容。
  4. 单击“添加规则”。

从场景中移除缺少的旧版资源

要从一个或多个场景中移除缺少的旧版资源，请执行以下操作：

  1. 在场景转换器中，单击“选项”选项卡。

    在“当前场景状态”下，将有缺少的旧版资源列表。

  2. 选择要从场景中移除的资源。
    注意:缺少的资源之间的从属关系显示在工具提示中。移除缺少的资源将移除嵌入的所有从属关系。
  3. 单击“从场景中移除选定元素”。
  4. 如果要在将来打开文件时自动移除任何缺少的旧版资源，请选择“文件打开时自动移除缺少的旧版资源”。
    重要信息:如果选择此选项，将在不通知您的情况下移除缺少的旧版资源，且场景转换器不会打开。
  5. 单击“转换场景”。

    注意:处理时间取决于场景的复杂性和大小。

编辑曲线 CV 子对象

本主题描述了 CV 子对象的控件，这些子对象处在曲线上。标记为 CV 卷展栏包含支持 NURBS 模型的 CV 子对象控件。除了此处描述的 CV 卷展栏，“曲线 CV”子对象层级还能显示“软选择”卷展栏。

  • “修改”面板 选择 NURBS 对象或子对象。 “修改器”堆栈显示 “曲线 CV”子对象层级 选择 CV 子对象。
  • “修改”面板 选择 NURBS 对象或子对象。 右键单击。 “工具 1”（左上角）象限 “子对象” “曲线 CV” 选择 CV 子对象。

在此处，可以在曲面上的 CV 曲线编辑 CV，与编辑其它种类的曲线 CV 一样。您可以变换曲面上 CV 曲线中的 CV，但是不能将 CV 移离曲面。使用“曲线 CV”子对象层级是编辑这些 CV 的一种方法，另一种方法是使用“在曲面上编辑曲线”对话框。

过程

要变换曲线 CV 子对象，请执行以下操作：

  1. 在“曲线 CV”子对象层级， 选择一个或多个 CV。

    子对象选择工具与子对象的其他类型工具相同。当键盘快捷键覆盖切换处于启用状态时，还可以使用 H 键。请参见子对象选择。

    本主题稍后介绍的“界面”下方的“选择”组框可以提供选择 CV 子对象的其他选项。

  2. 激活 （选择并移动）或其他变换，然后在视口中拖动，以变换选定内容。

    交互式变换 CV 时，将会更改模型的形状。

    只有当选择了多个 CV 时，“旋转”和“缩放”才起作用。

提示

  • 变换 NURBS CV 子对象时，“锁定选择集”按钮很有用。可以在一个视口中进行选择，单击“选择锁定切换”（或按 Ctrl+Shift+N），然后在其他视口中变换选定内容。
  • 当移动 CV 子对象时，请尽可能系统地移动它们，以避免“迷路”。

要使用键盘来选择曲线 CV 子对象，请执行以下操作：

您可以使用 Ctrl 键和箭头键选择曲线 CV 子对象。使用方向键时，可以按照子对象的创建顺序对其进行遍历。为此，请执行下列步骤：

  1. 启用 （键盘快捷键覆盖切换）。
  2. 通过单击或拖动来选择 CV。
  3. 按住 Ctrl 并使用方向键可以在 CV 子对象间移动。

    对于曲线上的 CV，方向键会沿着曲线，遍历选择 CV。方向键并不会在曲线子对象间移动。

还可使用 H 键盘快捷键（当启用“键盘快捷键覆盖切换”按钮时）显示对话框，然后按名称选择 CV。Ctrl+H 只在鼠标光标的正下方显示 CV 的名称。

要从曲线中移除 CV，请执行以下操作：

  1. 选择 CV。
  2. 单击“删除”。

    键盘快捷键：删除

    这就删除了 CV，然后会更新曲线形状。

    注意:CV 曲线的 CV 数目，至少要比曲线的度数多一。

要向曲线中添加 CV，请执行以下操作：

  1. 启用“优化”。
  2. 在曲线上需要添加 CV 的位置单击。

    CV 会添加到所单击的位置。为了保持原来的曲率，附近的 CV 会远离新的 CV。

要添加 CV 并延伸曲线的长度，请执行以下操作：

  1. 单击以启用“延伸”。
  2. 将鼠标移动到 CV 曲线之上。曲线会高亮显示为蓝色，曲线的一端会出现一个框，显示出曲线扩展的位置。
  3. 从高亮的端点 CV 拖动，然后释放鼠标按钮。

    新的 CV 会添加到曲线的原始长度之外。

要熔合两个 CV，请执行以下操作：

  1. 启用“熔合”。
  2. 无需释放鼠标按钮即可单击 CV。拖动到另一个 CV 上，然后释放鼠标按钮。

    选择的第一个 CV 得到了第二个 CV 的位置，并且附着到它之上。如果第一个 CV 有动画控制器，那么会丢弃该控制器。如果第二个 CV 有动画控制器，那么第一个 CV 也会得到它。

    默认情况下，熔合的 CV 以紫色显示。

要将熔合的 CV 取消熔合，请执行以下操作：

  1. 选择熔合的 CV。
  2. 单击“取消熔合”。

    现在，可以独立移动和编辑两个 CV。

要变换区域，请执行以下操作：

  1. 使用子对象选择，然后选择一个或多个 CV，作为变换中心。
  2. 启用“软选择”。
  3. 变换 CV。

    选定的 CV 周围的区域也会相应变换。

移动是最普遍使用的变换。对非局部变换中心可以使用“旋转”和“缩放”。

提示如果“软选择”不起作用，那可能是对于曲面大小来说“衰减”值太小。请在“软选择”卷展栏上，增加“衰减”值，使它可以覆盖其它的点或 CV。

界面

除了此处描述的 CV 卷展栏，“曲线 CV”子对象层级还能显示“软选择”卷展栏。

“选择”组

曲线子对象选择控件

单个 CV

（默认设置。）启用此选项后，可以通过单击选择单个 CV，或者通过拖动区域，来选择一组 CV。

所有 CV

启用此选项后，单击或拖动会选中曲线上的所有 CV。

名称

显示“未选择任何 CV”、“选定多个 CV”，或者“曲线名（索引）”其中之一，其中“曲线名称”是 CV 父曲线的名称，“索引”是 CV 在曲线上的 U 位置。无法编辑“名称”字段，自定义 CV 的名称。

如果 CV 是熔合的，那么“名称”字段显示的是第一个 CV 的名称。

权重

调整所选定 CV 的权重。可以使用 CV 的权重 来调整曲线上 CV 的效果。增加权重将曲线拉向 CV。减少权重将释放远离 CV 的曲线。

增加权重是使曲线变硬的一种方法；就是说，在特定位置使曲率尖锐。

默认情况下，在“创建”面板或 NURBS 子对象创建卷展栏上创建时，CV 或 NURBS 对象的权重是 1.0。几何上 CV 的权重，转化为 NURBS 时，可能会不同，取决于对象原来的形状。

当多个 CV 被选中时，可以更改权重。当多个 CV 被选中时，使用“权重”字段或微调器，可以为它们全部指定所选的值。因为权重是相对的（比例的），所以对所有选中的 CV 使用“权重”控件时，没有可见的效果。

隐藏

单击，可以隐藏当前选定的 CV。

全部取消隐藏

单击，可以将所有隐藏的 CV 取消隐藏。

熔合

将一个 CV 熔合到另一个 CV 上。（不可以将 CV 熔合到点上，反之亦然。）这是连接两条曲线的一种方法。这也是改变曲线形状的一种方法。

熔合 CV 并不会把两个 CV 子对象合并到一起。它们被连接在一起，但是保留截然不同的子对象，可以随后取消熔合。

熔合的 CV 如同一个单独的 CV 一样，直到取消熔合。熔合后的 CV 类似一个单独的点，但是也应用了统一 CV 多样性的属性。（请参见 NURBS 概念和 CV 曲线。）熔合的 CV 对曲线产生很大的影响。在熔合的 CV 附近，曲线可能会变得更尖锐，如果多于两个 CV 熔合在一起，甚至可能形成角。

熔合的 CV 会以明显的颜色显示。默认设置为“紫色”。（使用“颜色”面板可以更改这一颜色，“颜色”面板可以在“自定义用户界面”对话框中找到。）

取消熔合

将熔合的 CV 取消熔合。

优化

通过添加 CV，对曲线进行细化。

警告:使用“细化”添加 CV 时，曲线上的所有 CV 都会失去动画控制器。

当将鼠标移动到 CV 曲线之上时，会添加 CV 预览，并且它们的位置会以蓝色显示。

提示当使用细化将 CV 添加到曲线上时，最好进行重新参数化。请参见编辑曲线子对象。
删除

删除选定的 CV。

插入

将 CV 插入到曲线中。单击“插入”，然后单击曲线上想要插入新 CV 的地方。插入 CV 类似于使用 CV 细化，只是曲线中的其它 CV 并不会移动。这就意味着当插入时，曲线的形状会更改。

插入 CV 并不会将动画移除出曲线，这与细化不同。

提示当使用插入将 CV 添加到曲线上时，最好进行重新参数化。请参见编辑曲线子对象。
延伸

扩展 CV 曲线。从曲线端点拖动，可以添加新 CV，扩展曲线。

警告:使用“延伸”添加点时，曲线或曲面上的所有点都会失去动画控制器。
移除动画

从选定的中移除动画控制器。

显示晶格

启用此选项后，会在 CV 曲线周围显示控制晶格。禁用此选项后，控制晶格不显示在视口中。默认设置为启用。

选定 CV 数

此文本字段会显示当前选中了多少个 CV。

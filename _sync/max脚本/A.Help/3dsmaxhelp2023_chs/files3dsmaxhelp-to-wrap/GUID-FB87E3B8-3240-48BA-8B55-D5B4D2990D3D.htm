使用 Vault 管理资源

如果要将 Vault 用作资源管理系统，则在第一次访问 Vault 时需要先登录到 Vault 中，否则无法检入和检出文件。

从 3ds Max 启动并登录 Vault：

  1. 在 3ds Max 中，选择“文件”菜单 “参考”“资源追踪”。
  2. 在“资源追踪”对话框上，确保未在“选项”“禁用资源追踪”下禁用“资源追踪”。
  3. 在“资源追踪”对话框中，选择“服务器”“登录”。
  4. 在“登录”对话框中，输入以下内容：
    身份验证 身份验证的类型。默认值=Vault 帐户
    用户名 库帐户的名称。默认值= Administrator
    密码 与库帐户相关的密码。默认值=<left blank>
    服务器 安装库服务器的计算机名称。如果服务器安装在与客户端相同的计算机上，则使用“localhost”。如果此操作无法执行，则请咨询系统管理员。
    Vault 位于指定服务器的库数据库的名称。默认值为“Vault”。单击“浏览”按钮以从服务器的可用数据库列表中进行选择。默认值=Vault
    注意:如果您希望在下一次会话中自动登录，请启用“下一次会话自动登录”复选框。

    在所有的字段中输入这些值，然后单击“确定”。

    注意:您可以使用系统管理员为您提供的帐户值，而不使用默认值。如果您遇到问题，请咨询系统管理员。

要在使用 Vault 时设置本地工作文件夹，请执行以下操作：

在 Autodesk 库中使用这些文件时，需要设置一个工作文件夹。当您使用这些文件时，工作文件夹将会存储它们（在将文件检出库和将它们检入库之间）。工作文件夹既可以是本地计算机的文件夹，也可以是网络上的文件夹。通常，系统管理员会在网络上设置工作文件夹。

注意:如果系统管理员已在网络上设置了工作文件夹，则会发布消息通知此事。尽管您可能需要映射已为工作文件夹设置的驱动器位置，但是您不需要设置工作文件夹。有关详细信息，请咨询您的系统管理员。
  1. 在 3ds Max 中，选择“服务器”“登录”，然后登录到 Vault。
  2. 选择“服务器”“选项”。
  3. 浏览到您要使用的文件夹，然后单击“确定”进行确认。

要协调本地文件和库文件，请执行以下操作：

为使本地文件和库中的文件达到最佳的协调性，应使数据库中的文件夹结构和工作文件夹及其子目录中的结构保持一对一的对应关系。

创建 Vault 文件的本地文件夹结构，然后使用“选项”将本地结构的顶部映射到 Vault 结构的顶部（即，Vault($)）。

  1. 在本地驱动器上创建一个工作文件夹，用以容纳通过库与其它用户进行协调的所有文件。例如，可以将工作文件夹命名为 My_Vault。
  2. 打开“资源跟踪”对话框，如有必要登录到跟踪数据库。
  3. 在“资源追踪”对话框中，选择“服务器”菜单 “选项”。
  4. 在“库选项”对话框中，单击“更改”按钮，然后使用“浏览文件夹”对话框选择在步骤 1 中创建的文件夹。
    提示也可以使用“浏览文件夹”对话框创建该文件夹。

    现在，从 Vault 中打开文件时，如有必要，将 Vault 中存放文件的文件夹结构复制到本地文件夹中。同样，在将文件添加到库中时，如有必要，将本地结构复制到库中。例如，如果从 Vault($)\max_files\characters\ 中打开了一个角色网格，并且该文件使用在 Vault($)\max_files\characters\face_maps\ 中存储的位图，则该角色网格存入本地 My_Vault\max_files\characters\ 中，而位图文件也存入本地 My_Vault\max_files\characters\face_maps\ 中。

排列、选择和编辑 OpenFlight 场景

  • “工具”面板 “工具”卷展栏 “更多”按钮 “工具”对话框 Flight Studio Flight Studio 卷展栏 “修改层次/属性”按钮 “层次浏览器”对话框

层次浏览器是非模式对话框：在视口中和 3ds Max 界面中的其他位置编辑对象时，可以保持打开层次浏览器。

要查看层次，请执行以下操作：

  1. 导入 OpenFlight 文件。
  2. 打开 Flight Studio 工具。
  3. 单击“修改层次/属性”按钮。

    此操作将打开“层次浏览器”。

  4. “层次”窗口显示名为场景根的单个节点。要展开场景图形，请单击场景根左侧的加号 (+) 图标。

要选择节点及其所有子级，请执行以下操作：

  1. 在“层次浏览器”工具栏中打开“子树”。
  2. 通过在“层次”窗口中单击节点图标可以高亮显示节点。

    现在，将选中节点及其所有子节点。

若要将层次浏览器选择复制到 3ds Max 视口中，请执行以下操作：

  1. 高亮显示“层次”窗口中的一个或多个节点。
  2. 单击 （选择集）。

若要将 3ds Max 视口选择复制到层次浏览器中，请执行以下操作：

  1. 在 3ds Max 中选择一个或多个对象。
  2. 单击 （选择获取）。

要更改属性的值，请执行以下操作：

  1. 在“层次”窗口中，通过单击节点图标高亮显示节点。
  2. 在“节点属性”窗口中，单击“属性”名称右侧的“值”字段。
  3. 通过键入或从下拉列表中选择来更改属性值。
  4. 按 Enter 键。

要更改节点名称，请执行以下操作：

  1. 在“层次”窗口中，通过单击节点图标高亮显示节点。
  2. 单击节点名称的文本。

    名称将成为可编辑文本字段。

  3. 键入名称更改，然后按 Enter 或单击编辑字段之外的位置。

要在层次中移动节点，请执行以下操作：

  1. 在“层次”窗口中，通过单击节点图标高亮显示节点。
  2. 将节点拖到将变为父节点的另一节点的名称处。
  3. 通过释放鼠标放置节点。

    现在，所拖动的节点即为放置于其上的节点的子节点。如果父节点有多个子节点，则所移动的节点将成为最后一个子节点。

要创建一个新节点，请执行以下操作：

  1. 在层次中，通过单击节点图标高亮显示节点。

    所选节点将是新节点的父节点。（要创建顶级节点，请首先高亮显示场景根。）

  2. 从“层次浏览器”工具栏右端的下拉列表中选择节点类型。
  3. 单击“创建”。

本节内容

  • “层次浏览器”对话框（Flight Studio 工具）

    使用“层次浏览器”对话框，可以排列、选择和编辑其节点的 OpenFlight 场景及属性。

  • “切换属性”对话框（Flight Studio 工具）

    通过“切换属性”对话框，可以编辑选定“切换”节点的遮罩数据。

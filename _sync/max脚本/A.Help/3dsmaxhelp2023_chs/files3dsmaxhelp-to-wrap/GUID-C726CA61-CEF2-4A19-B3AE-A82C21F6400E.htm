“笔划首选项”对话框

使用“笔划首选项”对话框可以保存笔划集并设置其他笔划属性。

  • 绘制“笔划首选项”笔划。默认情况下，笔划为反 L，先从下到上垂直绘制，然后从左到右水平绘制。
  • 建立一个笔划，以访问“笔划首选项”对话框 “使用笔划”。
注意:该对话框只有通过绘制其笔划才可使用（默认情况下，是一个倒转的 L，从下到上垂直绘制，然后从左到右水平绘制）。可以在“查看和编辑笔划”对话框中重定义“笔划首选项”笔划。
注意:默认情况下，只有将笔划指定给“笔划首选项”对话框，然后使用笔划访问该对话框，“笔划首选项”对话框才可用。参见以下步骤。

过程

要访问“笔划首选项”对话框，请执行以下操作:

  1. 设置鼠标左键或鼠标中键的笔划。有关如何执行此操作的信息，请参见笔划。
  2. 要定义笔划，请按住 Ctrl 键，同时拖动一个倒转的 L 形状（从下到上垂直拖动，然后从左到右水平拖动）。

    该笔划的名称为 JGAB。

  3. 在“定义笔划”对话框上，从列表中选择“笔划首选项”。
  4. 单击“确定”关闭此对话框。
  5. 拖动一个倒转的 L 形状来访问“笔划首选项”对话框。

界面

“当前笔划集”组

显示当前集中的笔划名称和编号。

  • 要创建一个新集，请在字段中输入一个新名称，然后单击“保存”。
  • 要选择不同的集，请从列表中选择它，然后单击“确定”。
保存
保存显示在此列表中的集。
删除
删除显示在此表中的集。

“查看笔划默认显示顺序”组

指定在“查看笔划”对话框中最初列出的是命令还是笔划。

  • 命令名按命令名显示指定的笔划（例如，“播放动画”）。
  • 笔划名按笔划名显示指定的笔划（例如，HK）。
显示栅格时间 (ms)：
完成一个笔划时，笔划分析栅格显示在视口中所需的时间，以毫秒表示。设置为 0 可隐藏栅格。默认值为 300（大约为 1/3 秒）；范围设置为 0 到 2000。
显示范围时间 (ms)：
笔划范围显示在视口中所需的时间，以毫秒表示。范围从 0 到 2000。将其设置为 0 以禁用此功能。默认设置为 300（大约为 1/3 秒）。

应用于“第一点”上的笔划显示一个小 X。应用于笔划边界框中条目的笔划显示边界框。应用于圆形范围的笔划显示一个填充在笔划方形边界框内的圆形。窗口选择显示为实体。交叉选择显示虚体。

笔划点大小
视口中绘制的 Xs 的大小（以像素表示），可以使笔划形状直观化。默认设置为 4；范围为 3 至 20。

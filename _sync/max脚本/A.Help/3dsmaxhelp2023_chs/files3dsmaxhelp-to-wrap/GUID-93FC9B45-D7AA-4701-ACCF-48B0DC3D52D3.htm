属性承载器修改器

“属性承载器”修改器在“修改”面板上提供了一个用户界面，您可以向其中添加自定义属性。

  • 创建或选择对象。 “修改”面板 “修改器列表” “对象空间修改器” “属性承载器”
  • 创建或选择对象 “修改器”菜单 “动画” “属性承载器”

“属性承载器”是一个空的修改器，本身不含有用户界面；界面仅包含向其指定的属性。大体上，“属性承载器”是参数收集器的一个简化版本，它仅用于收集自定义属性并显示在“修改”面板而不是浮动对话框上。

过程

示例：要在“属性承载器”修改器中收集不同的自定义属性，请执行以下操作：

在开始执行此步骤之前，应熟悉“参数关联”对话框的基本使用方法。

  1. 在一个空白场景中添加一个小长方体。使其一边为 20.0 单位。
  2. 按顺序应用“锥化”修改器和“属性承载器”修改器。

    在修改器堆栈中,应高亮显示“属性承载器”修改器。

  3. 从“动画”菜单中，选择“参数编辑器”。
  4. 在“参数编辑器”，“属性”卷展栏上，作如下设置：
    • 添加类型为“选中对象的当前修改器”
    • 参数类型为浮点型
    • UI 类型为滑动
    • 名称为长方体高度
  5. 在“浮点型 UI 选项”卷展栏中，保持所有默认设置。
  6. 在“属性”卷展栏上单击“ 添加”。

    在“修改”面板上显示“自定义属性”卷展栏，包含新的“长方体高度”滑块。

  7. 添加另一个属性：
    • 添加类型为“选中对象的当前修改器”
    • 参数类型为整型
    • UI 类型为微调器
    • 名称为长方体高度分段
    • “整数 UI 选项”“范围从”1 到 50。
  8. 添加两个“浮点型/微调器”属性并命名为锥化量和锥化曲线。

    现在，属性承载器修改器有四个自定义的属性，但是它们不起作用因为没有连接。使用参数关联来将它们彼此连接。

  9. 关闭“参数编辑器”对话框。
  10. 在活动视口中，右键单击长方体并选择“连线参数”。从显示的下拉菜单中，选择“修改对象”“长方体(对象)”“高度”。

    会出现一个橡皮筋似的虚线将鼠标光标与长方体之间相连接。

  11. 不能将此“关联”与自定义属性直接连接，因此只需在视口的空白部分左键单击来打开“参数关联”对话框。

    左边的层次列表（Box01）展开到高亮显示的长方体高度参数。

  12. 在右侧，展开此路径：“对象”“Box01”“修改对象”“属性承载器”“自定义属性”，然后单击“长方体高度”以使其高亮显示。
  13. 单击 （“双向连接”，双头箭头），然后单击“连接”。
  14. 同样， 将左边的“高度分段”与右边的“长方体高度分段”连接。
  15. 关闭“参数关联”对话框。
  16. 再次右键单击长方体，选择“参数关联”，选择“修改对象”“锥化”“数量”，然后左键单击打开“参数关联”对话框。

    量参数在对话框的左边高亮显示。

  17. 在右边单击“属性承载器”的“锥化量参数”并连接它们。
  18. 将左边的“曲率”与右边的“锥化曲线”相连接，然后关闭对话框。

    所有参数现在都连接完毕。

  19. 在“自定义属性”卷展栏改变值来进行试验。

    属性承载器修改器允许改变长方体的创建参数和锥化修改的设置而不必在修改器堆栈中来回切换。通过这种方法可以在一个方便的位置随意的对在对象修改器堆栈中的许多来自不同层次的不同参数进行访问，甚至它们可以是来自不同对象的。

    可能会出现不能将锥化设在曲线内部。重新打开“参数编辑器”，单击“编辑/删除”，然后修改“锥化曲线”是它允许负值，这样可以解决这个问题。改变会立即生效，无需重新关联。

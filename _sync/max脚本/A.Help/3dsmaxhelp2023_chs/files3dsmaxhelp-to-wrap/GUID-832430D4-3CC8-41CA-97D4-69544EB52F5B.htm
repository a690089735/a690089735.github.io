“选择”卷展栏（投影修改器）

“投影”修改器的“选择”卷展栏管理子对象选择。

  • 选择对象。 “修改”面板 “修改器列表” “对象空间修改器” “投影” “选择”卷展栏

界面

框架
单击可启用“框架”子对象层级。

“框架”是法线投影的曲面。在“框架”子对象层级中，可以通过转换框架顶点的子对象选择来手动调整框架。

面
单击可启用“面”子对象层级。

“面”子对象层级可以为不同曲面的部分指定不同的源几何体。

元素
单击可启用“元素”子对象层级。

元素子对象层级可以为单个元素指定不同的源几何体。（元素是相邻面组。）

收缩
通过取消选择最外部的子对象缩小子对象的选择区域。如果不再减少选择大小，则可以取消选择其余的子对象。
扩大
朝所有可用方向外侧扩展选择区域。

使用“收缩”和“增长”，可以从当前选择的边上添加或移除相邻元素。“收缩”和“增长”在任意子对象层级上起作用。

忽略背面
启用后，选择子对象将只影响朝向您的那些对象。禁用此选择后，无论可见还是面对，都可以在鼠标光标下选择任一子对象。如果多个子对象位于光标下，可以重复单击循环切换。同样，禁用“忽略背面”后，无论面对的方向如何，区域选择都包括了所有的子对象。默认设置为禁用。
取得堆栈选择
单击并从修改器聚集子对象选择，该修改器在堆栈上的“投影”修改器下方。
选择平滑组
要通过平滑组值进行选择，可以使用微调器设置平滑组的数量，然后单击“选定平滑组”。
选择 MatID
要通过材质 ID 进行选择，可以使用微调器设置 ID 数量，然后单击“选定 MatID”。
  • 子材质下拉列表当将多维/子对象材质应用到低分辨率对象上时，列表会显示子材质的名称和数目，该子材质被指定为对象的面或者元素。当材质 ID 已经选定，相应的子材质会出现在列表上的字段中。
清除选择
启用该选项后，每个堆栈、平滑组或者材质 ID 的选择会替换上一个选择。禁用此选项后，会将每一个新的选择添加到上一个选择集中。默认设置为启用。

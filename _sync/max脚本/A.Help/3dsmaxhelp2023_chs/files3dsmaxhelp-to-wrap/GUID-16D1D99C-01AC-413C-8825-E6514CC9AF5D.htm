“HD IK 解算器”卷展栏

“HD IK 解算器”卷展栏显示在“层次”面板上。

本节内容

  • “对象参数”卷展栏（HD 解算器）

    使用“对象参数”卷展栏可以设置整个层次链的 IK 参数。

  • 位置/方向/绑定到跟随对象（HD 解算器）

    使用这些组框中的控制将 HD IK 链中的对象绑定到世界或跟随对象上。使用各种选项修改绑定的行为。

  • 优先级（HD 解算器）

    您可以使用三个关节优先级控件来改变计算 IK 解决方案的方式。

  • 复制、粘贴和镜像关节参数（HD 解算器）

    使用“对象参数”卷展栏中的“复制”和“粘贴”按钮，在 IK 控制器之间复制和粘贴关节设置。还可以将关节设置从非 IK 控制器复制到 IK 复制器，但是不能从 IK 复制器复制到非 IK 控制器。利用 HD IK 解算器层次，可以粘贴到选择的多个关节。

  • “滑动关节”和“转动关节”卷展栏（HD 解算器）

    在反向运动学中，操作关节的方法是在一根或更多的轴上允许运动，在其余的轴上限制运动。在“关节”卷展栏中，使用“激活”复选框，可以设置对象是否能围绕给定的轴移动（滑动）或旋转。关节最多可能有六根轴：三根位置轴，三根旋转轴。通过设置轴活动的轴，就可以限制关节的运动。

共享目录

您可以共享目录所在计算机中的目录。从而可以使网络上的其他计算机也能够访问该目录。下面是一些简要说明。有关详细信息，请参见 Windows Vista、Windows XP 或 Windows 2000 文档。

下一步

管理器和服务器程序的初始设置

过程

要共享目录，请执行以下操作：

  1. 转到包含要共享目录的计算机。
  2. 在 Windows 资源管理器中，右键单击要共享的目录，然后从右键单击菜单中选择“共享”。
  3. 如果使用 Windows XP 或 Windows 2000，则在“共享”选项卡上选择“共享此文件夹”选项。
  4. 如果使用 Windows NT4，则在“共享”选项卡上选择“共享”选项。
  5. 使用默认的“共享名称”。
  6. 单击“权限”，然后确保权限设置为“全员/完全控制”。单击“确定”可退出“权限”对话框。
  7. 单击“OK”以接受更改。
    注意:如果计划使用 10 台以上的渲染服务器，则所有场景位图的输出路径和位置均应位于运行 Windows XP 或 windows 2000 服务器的系统上，因为 Windows XP 专业版和 Windows 2000 专业版具有 10 个并发连接的限制。

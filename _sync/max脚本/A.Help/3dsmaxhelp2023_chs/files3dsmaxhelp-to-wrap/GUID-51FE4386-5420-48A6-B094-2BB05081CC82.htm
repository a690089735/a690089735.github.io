锚定 VRML97 辅助对象

使用“锚定”辅助对象可以指定场景中的单击播放触发器。该触发器将链接到场景中当前现有的对象上。从而可以向其他 HTML 页面、VRML97 世界添加链接，或替代 VRML97 世界中的摄影机。

  • “创建”面板 “辅助对象” “VRML97” “对象类型”卷展栏 “锚”按钮
  • “对象”菜单 “多个辅助对象” “锚”

过程

要将“锚定”设置为跳转到另一个 VRML 世界，请执行以下操作：

  1. 单击“锚定”按钮，然后在“顶”视口中单击并拖动来创建其图标，可以添加“锚定”辅助对象。
  2. 场景中的拾取触发器对象将是浏览时查看者单击的对象。
  3. 选择“超链接跳转”并指定跳转的 URL。
  4. 当用户单击“触发器对象”几何体时，浏览器将用指定的 URL 替换当前场景。

界面

“锚定”卷展栏包含以下选项：

拾取触发器对象
指定将称为此锚的触发器的几何体。单击此按钮，然后选择几何体。
描述
可以输入文本描述或消息，当鼠标移到已定义锚动作的对象的上方时，它们会出现在浏览器的状态栏中。

“超链接跳转”组

超链接跳转
创建跳转到某 URL 的锚。
URL
指定用于“超链接跳转”的位置。使用“书签”列表，或手动输入一个位置。如果 URL 指向另一个 VRML97 world（.wrl 文件），您可以将“#CameraName”附加到 URL 的末尾，使浏览器使用名为“CameraName”的视口作为初始视图。
参数
可以为超链接跳转指定其他浏览器参数。有关该字段的用法，请参见 VRML97 规范。
书签
可以从书签列表中选择 RUL 位置。单击“导入列表”可导入在浏览器中定义的书签列表，或在列表中手动输入新 URL。

“摄影机”组

设置摄影机
创建跳转到当前 VRML97 世界中给定摄影机的锚。
摄影机
指定用于“设置摄影机”的摄影机的名称。
图标大小
确定场景中辅助对象的大小。

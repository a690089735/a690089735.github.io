“缩放”工具

缩放工具调整模型当前视图的比例。

使用“缩放”工具时，您可以通过以下方式更改模型的缩放比例：

  • 单击 如果单击轮子上的“缩放”工具，那么当前的视图会以 25% 的因子放大。如果正在使用“完整导航”轮子，那么必须在“视口配置”对话框的“SteeringWheels”面板上启用“增量放大”。
  • Shift + 单击 如果单击轮子上的“缩放”工具之前按住 Shift 键，那么当前的视图会以 25% 的因子缩小。
  • Ctrl + 单击 如果单击轮子上的“缩放”工具之前按住 Ctrl 键，那么当前的视图会以 25% 的因子放大。
  • 单击并拖动 如果您单击“缩放”工具并按住指点设备上的按钮，那么可以通过上、下拖动来调整模型的放大倍数。
  • Ctrl+单击并拖动 当使用“完整导航”轮子时，您可以控制“缩放”工具使用的目标点。通过按住 Ctrl 键，“缩放”工具可使用“中心”工具定义的中心点。
注意:当从“完整导航”轮子调用“缩放”工具时，Ctrl + 单击可以缩放当前轴点附近的位置，而非当前光标位置。只有在“视口配置”对话框的“SteeringWheels”面板上启用“增量放大”，单击与 Shift+单击才可执行增量缩放。

缩放约束

使用“缩放”工具更改模型的放大倍数时，只能放大焦点或者不能在超出模型范围的位置进行缩小。放大和缩小的方向由“中心”工具设置的中心点控制。

注意:与“视图对象”轮子上的“缩放”工具不同，“完整导航”轮子上的“缩放”工具不受约束。

过程

使用单击可缩放视图：

要将该功能和“完整导航”轮子一起使用，您必须启用“增量放大”选项。

  1. 显示具有“缩放”工具的轮子。
  2. 单击“缩放”楔形体。

    模型的放大倍数会增加并且可在离模型较近的范围内进行缩放。如果按住 Shift 键的同时单击“缩放”楔形体，那么模型会缩小，或者按住的是 Ctrl 键可放大模型。

  3. 单击“关闭”退出轮子。

通过拖动缩放视图：

  1. 显示具有“缩放”工具的轮子。
  2. 单击“缩放”楔形体。按住指点设备上的按钮并垂直拖动可以放大和缩小。
  3. 释放指点设备上的按钮以返回到轮子。
  4. 要退出轮子，请右健单击。

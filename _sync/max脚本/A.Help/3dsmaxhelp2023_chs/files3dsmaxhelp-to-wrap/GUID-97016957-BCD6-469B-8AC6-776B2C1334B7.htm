“权重解算器”对话框（蒙皮修改器）

使用这些对话框可以调整“体素”和“热量贴图”这两种“权重解算器”方法的设置。

重要信息:在调整这些设置后，请确保单击“应用”。直接关闭对话框没有任何效果。

对于上述两个解算器，前两项设置的作用相同。

衰减（两个解算器）
确定关节混合的平滑度。值越低衰减越平滑，值越高变换越急促。

左侧：衰减 = 0.0；居中：衰减 = 0.3；右侧：衰减 = 1.0

“衰减”值与上面的插图中所示相同，该图直观呈现了顶点权重

最大影响（两个解算器）
可影响蒙皮网格中每个顶点的最大骨骼数。值 0 视为 1，值 -1 允许解算器确定要使用的骨骼数。

此设置相当于“高级参数”卷展栏上的“骨骼影响限制”设置，但仅适用于受权重解算器影响的顶点。

最大分辨率（仅限体素解算器）
要使用的体素解决方案的密度。值越大，解决方案的精度越高，所花的时间也就越长。
使用缠绕数值（仅限体素解算器）
解决方案的精度。值越高可以降低求解速度。
应用
对蒙皮网格中的选定节点运行解算器。如果未选择任何顶点，则将这些设置应用于所有顶点。

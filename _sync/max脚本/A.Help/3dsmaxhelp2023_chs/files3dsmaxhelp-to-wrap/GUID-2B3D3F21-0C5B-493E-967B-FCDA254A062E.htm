拖放子对象材质指定

可以将某一材质应用于选定的可渲染子对象（如网格中的各面）。在“材质编辑器”中，可以使用“将材质指定给选定对象”。还可以将材质从“材质编辑器”或“材质/贴图浏览器”拖动到选定面。这可以立即创建一个新的“多维/子对象”材质。

还可以禁用将材质拖放到子对象。为此，请转至“首选项”对话框 “常规”选项卡，并在“子材质”组中，关闭“自动指定”。默认情况下，此复选框处于启用状态。

如何创建多维/子对象材质

根据已向选定子对象应用了什么材质，多维/子对象材质采用以下三种方式创建：

  • 未应用材质

    如果选定面未应用材质，则创建新的多维/子对象材质。拖动的材质会变为新材质中的子材质。如果材质 ID 已存在，则会予以保留。

  • 已应用现有材质（并非多维/子对象材质）

    将创建新的多维/子对象材质，并将其应用于选定的面。现有的材质将移动到该多维/子对象材质，并成为第一个子材质。未选定的面的材质 ID 为 #1，选定面的材质 ID 为 #2，拖动的材质成为多维/子对象材质的一部分。不会保留现有的材质 ID。

  • 已应用多维/子对象材质

    如果现有多维/子对象材质已在场景中应用多次，该材质会被复制，新的副本将应用于选定的面。

    如果多维/子对象材质只在场景中应用一次，则使用现有材质。拖动的材质将被添加到现有的多维/子对象材质。

    如果拖动的材质已成为该多维/子对象材质的一部分，那么选定的面将接收相应的材质 ID 号。如果所有选定面具有相同的材质 ID 号，并且没有任何未选定面使用了该编号，则使用此编号，同时新材质将替代此 ID 号的原有子材质。否则，将为这些面指定新材质 ID 号，并且此 ID 号用于拖动的材质。这种情况下，将保留所有现有材质 ID。

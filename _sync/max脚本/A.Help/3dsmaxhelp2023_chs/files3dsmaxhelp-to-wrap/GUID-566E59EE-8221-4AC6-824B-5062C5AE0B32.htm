几何体文件格式

使用“文件”菜单上的“导入”和“导出”命令，可以与其他 3D 建模程序共享 3D 几何体。

3ds Max 可以导入和导出各种文件格式，如本部分中所述。

本节内容

  • 实体对象

    3ds Max 会在您导入某些文件格式（如 SAT 文件）时创建实体对象。

  • Adobe Illustrator (AI) 文件

    Adobe Illustrator 是一款基于矢量的图形应用程序。3ds Max 可以导入和导出 Illustrator 文件。

  • Alias (WIRE) 文件

    3ds Max 与 Alias 的配合相得益彰，非常适用于基于设计数据创建真实照片级渲染和动画，以及创建基于多边形的几何体以重新集成到 Alias 场景中。

  • Alembic (ABC) 文件

    Alembic 是视觉特效和动画专业人员常用的计算机图形交换文件格式，它可对复杂的动画场景进行提取，获得一组非程序性、应用程序无关的烘焙几何体结果。 它可以处理具有变化拓扑的动画，有助于流体等模拟。

  • 导出到 ASCII

    选择将“ASCII 场景导出 (*.ase)”作为“另存为”类型，导出的文件是场景的 ASCII 表示方式。出现一个对话框，用于指定包含哪些场景组件，以及输出方式。这对于编写需要解析的 ASCII 文件非常有用。

  • CATIA V4（MODEL、DLV4、DLV3、DLV、EXP、SESSION、MDL）文件

    CATIA® 是 Dassault Systèmes 开发的一种计算机辅助设计软件。 您可以将 CATIA V4 文件导入到 3ds Max 中。

  • CATIA V5 (CATPRODUCT, CATPART) 文件

    CATIA® 是 Dassault Systèmes 开发的一种计算机辅助设计软件。 您可以将 CATIA V5 文件导入到 3ds Max 中。

  • AutoCAD (DWG) 文件

    本部分中的主题介绍如何与 AutoCAD 和其他使用 DWG 格式的程序共享文件。

  • AutoCAD 交换 (DXF) 文件

    DXF 文件用于在 AutoCAD 及其他支持该文件格式的程序中导入和导出对象。最常见的情况是，dxf 文件用于从支持 DXF ，但不支持 dwg 文件的 CAD 程序导入和导出建模数据。

  • 3D DWF 文件

    使用 3D DWF 发布，可以导出三维模型的 Design Web Format™ （.DWF™）文件，这样导出的效果几乎与渲染的场景有相同的视觉精度。DWF 文件是高度压缩文件。您可以与没有 3ds Max 的团队成员轻松共享这些文件。

  • Autodesk Inventor (IPT、IAM) 文件

    IPT 和 IAM 是用于部分(IPT) 和集合(IAM) 的固有 Autodesk Inventor® 文件格式。您可以将两种文件格式导入到 3ds Max。

  • Autodesk Revit (RVT) 文件

    除了使用文件链接管理器，还可以将 RVT 文件导入到 3ds Max。

  • Autodesk FBX 文件

    无论使用哪种软件供应商或计算机平台，FBX 文件都可以在大多数 3D 软件之间提供互操作性。您可以使用 3ds Max 导入和导出此格式的文件。

  • 导出为 glTF
  • IGES 文件

    可以使用 3ds Max 导入和导出 IGES（初始化图形交换规范）文件。

  • JT Open Program (JT) 文件

    JT 文件是一种用于产品可视化、协作和数据共享的 open CAD 格式。您可以将 JT 文件导入到 3ds Max 中。

  • LandXML (XML, DEM) 文件

    本节中的主题介绍如何与 3ds Max 一起使用土地开发数据。土地数据通常来自市政工程项目，可以使用几种不同文件格式。

  • 运动分析文件 (HTR/HTR2, TRC)

    本部分中的主题介绍如何使用 HTR（层次平移旋转）和 TRC 格式的运动捕捉文件。

  • OpenFlight (FLT) 文件

    OpenFlight® 格式常用于视觉模拟系统。可以导入和导出 OpenFlight 文件。使用 Flight Studio® 工具，还可以创建和编辑 OpenFlight 文件中的对象和属性。

  • Pro/ENGINEER（PRT、ASM）文件

    Pro/ENGINEER® 是一种计算机辅助设计软件应用程序。 可以导入 Pro/ENGINEER 文件。

  • SAT 文件

    SAT 格式功能处理导入和导出精确几何边界表示 (BREP) 数据，它也称为 ACIS 实体，对于修剪曲面，作为 3ds Max“实体对象”的实体模型和集合。

  • 3D Studio 网格（3DS、PRJ）文件

    本节中的主题介绍了如何从 3D Studio ® 处理网格文件。

  • SketchUp (SKP) 文件

    使用 SketchUp 导入器可以从 SketchUp 中直接导入场景文件。

  • SolidWorks（SLDPRT、SLDASM）文件

    SolidWorks® 是一款计算机辅助设计 (CAD) 和计算机辅助工程 (CAE) 软件应用程序。3ds Max 可以导入 SolidWorks 零件 (SLDPRT) 和部件文件 (SLDASM)。

  • STEP（STP、STEP）文件

    产品数据交换标准 (STEP) 是一种 ISO 标准交换格式，用于在 CAD/CAM 系统之间传输图形数据。STEP 和 STP 文件均可导入到 3ds Max 中。

  • Stereolithography (STL) 文件

    本节中的主题介绍如导入和导出 stereolithography (STL) 文件。

  • SHP 文件

    SHP 是 3D Studio (DOS) 图形文件格式。您可以将这些文件导入到 3ds Max。

  • UG-NX (PRT) 文件

    NX 是一种实体建模软件包。您可以将 NX 文件导入到 3ds Max 中。

  • VIZ 渲染（DRF）文件

    DRF 是用于 VIZ Render 的文件格式，它是早期版本的 AutoCAD Architecture 附带的渲染工具。DRF 文件类型类似于 Autodesk VIZ 先前版本中的 MAX 文件。

  • VRML 文件

    本节中的主题介绍如何在 3ds Max 中使用 VRML（虚拟现实建模语言）文件。

  • Wavefront (OBJ) 文件

    3ds Max 可以导入和导出基于文本的 (ASCII) Wavefront 格式 OBJ。OBJ 文件包含几何体描述，而关联 MTL 文件格式包含材质描述。

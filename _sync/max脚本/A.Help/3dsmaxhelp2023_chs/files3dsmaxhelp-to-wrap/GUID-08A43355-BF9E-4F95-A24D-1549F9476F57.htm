高级木材

使用高级木材贴图生成逼真的三维木材纹理。

  • “材质/贴图浏览器” “贴图” “常规” “高级木材”
注意:仅当活动渲染器支持时，高级木材贴图才会显示在浏览器中。

界面

“预设”卷展栏

预设
从预设中选择以应用标准木材纹理（例如橡木、松木和胡桃木），以及各种磨光。

“常规”卷展栏

“总体”组

缩放
设置木质纹理的整体比例。
轴
设置局部轴（X、Y 或 Z）用于对齐木材的颗粒。
使用场景单位
使用场景单位来计算比例。
粗糙度
设置要发送到材质的粗糙度级别。最大值设置为 1。

“早期木材”组

颜色
设置早期木材的颜色，这种木材在春天开始生长，颜色较浅。早期木材与晚期木材之间的明暗差异有助于定义木材的外观。

或者，您可以使用贴图来定义颜色。

使用 Perlin 噪波
对早期木材应用增强真实感的视觉噪波。
使用凹槽粗糙度
如果启用，则仅为早期木材设置粗糙度级别。最大值设置为 1。

“晚期木材”组

强度
为晚期木材设置颜色强度，这比早期木材深。对于大多数用途，可以使用“强度”设置，而不是选择特定颜色。最大值设置为 4。
使用颜色
启用对晚期木材使用特定颜色。
颜色
设置晚期木材的自定义颜色。
使用 Perlin 噪波
对晚期木材应用增强真实感的视觉噪波。
使用凹凸
如果启用，则会从晚期木材生成凹凸贴图。请注意，必须将高级木材的凹凸输出连接到材质的凹凸贴图输入，此设置才能生效。最大值设置为 1。

“全局”组

使用漫反射 Perlin 噪波
对漫反射颜色应用 Perlin 噪波。使用此选项可为木材颜色添加非常精细的细节。
比例(Z)
设置噪波的比例。最大值为 1。
漫反射 Lobe 权重
设置木材表面的漫反射。最大值为 1。

“权重和比率”卷展栏

“权重和比率”组

纤维余弦
如果启用，则使用余弦函数以使颗粒看起来更不平行且更不规则，并加入较大波纹。为获得更加逼真的效果，此选项通常与“纤维 Perlin 噪波”一起启用。
纤维 Perlin 噪波
启用 Perlin 噪波以使颗粒看起来更不平行且更不规则，并加入较大波纹。为获得更加逼真的效果，此选项通常与“纤维余弦”一起启用。最大值为 1。
增长 Perlin 噪波
启用 Perlin 噪波以随机化增长环的厚度，就像树在多年生长过程中经受各种温度后发生的真实情况。这些环可以在对象的两端看到，很像一系列嵌套圆柱体的顶部（当沿着它们的长度切割成颗粒时）。
晚期木材比例
设置每个环的晚期木材比例。最大值为 1。
早期木材锐度
设置早期木材颜色的各边的锐度。最大值设置为 1。
晚期木材锐度
设置晚期木材颜色的各边的锐度。最大值设置为 1。
圆环厚度
设置增长环的厚度。值越低，表面纹理越细密，而值越高，颗粒之间的间隔越大。最大值设置为 1。

“气孔”卷展栏

启用了早期木材气孔的未加工松木（右）

“气孔”组

使用气孔
用于在木材表面产生瑕疵，显示为增长环上的圆点和沿颗粒的条纹。
气孔类型
设置气孔的生成位置：早期木材、晚期木材或两者。
气孔半径
设置各个气孔的半径。值越低（如 0.04），真实感越强。最大值设置为 2。
气孔单元尺寸
增加生成的气孔的密度。最大值设置为 3。
颜色强度
设置气孔的颜色强度。最大值设置为 5。
气孔深度
设置各个气孔的深度以用于凹凸贴图。最大值设置为 1。

“光线”卷展栏

启用了光线的未加工松木（右）

“光线”组

使用光线
如果启用，则生成带状髓线，根据设置，可能显示为从增长环中心向外辐射的线条以及颗粒中的短垂直线。
颜色强度
设置光线颜色的强度。最小值设置为 0.5（浅色）；最大值设置为 3（深色）。
分段长度
设置光线线段的长度。最大值设置为 10。
切片数
设置要以辐射状生成的光线近似数量。其他光线设置可能会增加或减少可见数量。最大值设置为 400。
椭圆 Z2X
设置用于生成光线的单元的深度。最大值设置为 40。
椭圆半径 X
设置用于生成光线的单元的半径。最大值设置为 1。

“坐标”卷展栏

这些设置用于相对于应用贴图的对象的体积移动贴图。有关详细信息，请参见“‘坐标’卷展栏（三维）”主题。

列（“材质资源管理器”的“材质”面板）

“材质资源管理器”的下部“材质”面板的主窗口可以包含若干列，这些列包含了有关当前所选材质的信息。有些类型的列包含可编辑单元。

  • 观察点 (POV)“视口标签”菜单 “扩展视口” “材质管理器” “材质”（下部）面板
  • 默认菜单：“渲染”菜单 “材质资源管理器” “材质”（下部）面板
  • Alt 菜单：“材质”菜单 “工具(材质设置)” “材质资源管理器” “材质”（下部）面板

右键单击列标签时显示的菜单

右键单击“材质资源管理器”的下部面板中的列标签时，3ds Max 会打开一个菜单，该菜单包含了若干用于控制列显示的选项：

  • 排序选项 用于选择以何种方式对列进行排序。

    另外，还可以通过单击列标签来循环切换这些选项。

  • 配置列 显示一个用于配置在窗口中显示哪些列的对话框。
  • “最佳拟合”选项 右键单击一个列的标签后，选择“最佳拟合”可调整该列的大小，选择“最佳拟合（所有列）”可调整所有列的大小。相应列将根据其内容的宽度调整大小。
提示右键单击一个列标签并选择“最佳拟合（所有列）”是查看当前窗口布局中信息的很好方式。

列

名称

显示材质、贴图和子材质的名称。可以编辑此列中的单元，方法是：单击一个单元，然后再次单击或拖动它以高亮显示文本，再为材质或贴图键入新名称。

注意:即使隐藏“名称”列，层次也会始终显示在下部面板窗口的左侧。

默认列。

属性

对于贴图，会显示此贴图分配到的材质通道的名称。例如，“漫反射颜色（反射比）”。

对于材质，此列中的单元是空白的。

默认列。

纹理大小

对于位图，会显示贴图的尺寸；例如，“512*512”。

对于材质和程序贴图（不论是 2D 还是 3D），此列中的单元都是空白的。

默认列。

UV 坐标

对于 2D 贴图，此列中的单元对应于 2D 贴图的“坐标”卷展栏中的“贴图列表”。可以编辑这些单元，方法是：单击相应的单元以打开一个下拉列表，使用此下拉列表便可以在多种贴图类型之间进行选择。

对于材质和 3D 程序贴图，此列中的单元显示“无”。

贴图通道

对于 2D 和 3D 贴图，此列中的单元显示“贴图通道”值。可以编辑此列中的单元，方法是：单击一个单元，然后将其拖过相应的值。此值高亮显示时，可以键入新的贴图通道值，也可以单击此单元中显示的微调器箭头来更改贴图通道值。（还可以通过拖动微调器的向上箭头或向下箭头按钮，更快地更改此值。）

对于 3D 程序贴图，除非“坐标”“源”值设置为“显式贴图通道”，否则不显示贴图通道值。

对于材质，此列中的单元是空白的。

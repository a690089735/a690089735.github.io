SAT 文件

SAT 格式功能处理导入和导出精确几何边界表示 (BREP) 数据，它也称为 ACIS 实体，对于修剪曲面，作为 3ds Max“实体对象”的实体模型和集合。

Autodesk Inventor 中的吹风机实体

SAT 格式对象可能来自 Autodesk 程序（如 Revit 和 Inventor）以及类似 Rhino 和 Catia 之类的第三方软件。

导入 SAT 文件时，生成的对象为实体对象，并且可以在“修改”面板上调整这些对象的设置。

您也可以使用“创建”面板上的实体对象工具处理导入的实体对象并将 3ds Max 基本体转换为实体对象格式。

本节内容

  • “SAT 导入”对话框

    使用 SAT 格式将 BREP（边界表示）对象导入 3ds Max，同时保留其实体几何体格式。此类对象通常来自诸如 Autodesk Revit 和 Autodesk Inventor 等程序，称为“实体对象”。

  • “SAT 导出”对话框

    使用 SAT 导出格式保存 3ds Max 中的实体对象，同时保留其实体几何体格式。接下来，此类对象可重新导入诸如 Autodesk Inventor 之类的程序中。

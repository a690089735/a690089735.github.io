“随机放置难度”对话框

如果不使用“散布对象”对话框重叠，character studio 很难放置克隆的对象时，将会显示此对话框。该对话框中显示的文本显示了 character studio 尝试放置特定对象的次数，并询问您是否要重试。

界面

重试

单击此按钮可以强制重试 N 次，其中 N 在“还要尝试的次数”字段中设置。

跳过这一次

停止尝试放置当前对象，而继续放置下一个对象。

退出尝试

中止“生成位置”进程；将不会放置更多的对象。

“将轨迹映射到轨迹”卷展栏

此卷展栏包含三个列表窗口。左（当前）和右（引入）显示节点/控制器层次，如在“轨迹视图”中一样。

  • 默认菜单：“动画”菜单 “加载动画” “打开”对话框 “编辑贴图”
  • 默认菜单：“动画”菜单 “加载动画” “打开”对话框 “加载运动” “XML 动画”对话框 单击“是”。
  • Alt 菜单：“动画”菜单 “工具(动画设置)” “加载动画” “加载 XML 动画文件”对话框 “编辑贴图”
  • Alt 菜单：“动画”菜单 “工具(动画设置)” “加载动画” “加载 XML 动画文件”对话框 “加载运动” 单击“是”。

因为“贴图节点”组功能应用于高亮显示的轨迹，可以使用标准的高亮显示的方法。单击可高亮显示某个条目，按住 Ctrl 键并单击可高亮显示多个条目，按住 Shift 键并单击可高亮显示某个范围内的条目。此外，在当前窗口和即将出现的窗口中，您可以右键单击打开菜单，这可使您高亮显示所有轨迹（全选），反选当前的高亮显示（反选），并对所有轨迹禁用高亮显示（全部不选）。

此外，可以使用右键单击菜单进行展开并使用其名称旁边的 - 或 + 图标塌陷任何轨迹。

界面

当前列表

使用相同层次显示在场景中显示选定对象的动画轨迹，如在“轨迹视图”中一样。未指定轨迹使用黑色字符，而指定轨迹使用灰色字符。

状态

此只读字段显示控制器的数量和贴图节点的数量。

贴图列表

显示已经贴图的轨迹。

<-

将传入列表中高亮显示动画轨迹指定给当前列表中的高亮显示动画轨迹。然后该指定将出现在“映射”列表中，与相应的当前列表条目相反。

如果两个轨迹不包含比较的数据，单击按钮时不会发生任何情况。

->

移除高亮显示的贴图列表条目。

引入列表

使用与“轨迹视图”中相同的层次显示效果，在加载的 XAF 文件中显示动画轨迹。未指定轨迹使用黑色字符，而指定轨迹使用灰色字符。

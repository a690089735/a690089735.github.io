影响区域修改器

使用影响区域修改器可以在对象曲面形成膨胀或缩进。

利用影响区域，可以变换选中的顶点，也可以围绕选中的顶点在不同程度上变换该区域中的顶点。观察修改器效果的最简单方法是通过带有大量细分的小的扁平对象。

应用“影响区域”后，它会指定一个包含通过一条线连接两点的箭头形状的 Gizmo。箭头的底部是起始点。箭头的长度和方向定义了顶点移动的量。箭头底部的任意带有衰退距离的顶点都会向箭头方向转换。

因为不能在晶格上直接选中点，此修改器并不依赖于输入对象的拓扑。可以将其应用于任意可渲染对象。然而，可以通过使用选择修改器（比如“网格选择”或“体积选择”）将选中的子对象传送于堆栈的上部来限制其效果。

注意:“影响区域”对简单动画效果来说是很有用的，特别是需要使用交互式参数时。不过，为了能够精确地进行建模调整，您可能希望在可编辑网格、可编辑多边形、可编辑面片和 NURBS 对象中使用“软选择”功能集的扩展功能。“软选择”也是“编辑多边形”、“多边形选择”、“体积选择”和“HSDS”等修改器的一部分。

  2. 将平面的长和宽设置为 50 单位。

  4. 在“参数”组中，将“衰退”设置为 50。
  5. 调整参数来实现不同的效果。

AVI 文件

AVI（音频-视频隔行插入）格式是电影文件的 Windows 标准格式。.avi 文件名扩展名表示 Windows AVI 电影文件。

3ds Max 会创建您在进行动画预览时创建的 AVI。也可以将最终输出渲染到 AVI 文件。尽管 3ds Max 通过渲染单帧 TGA 文件或直接渲染到数字磁盘记录器可以得到较高品质的输出，但您在渲染 AVI 文件时仍可获得较好结果。

AVI 文件可以多种方式用作 3ds Max 的输入，例如：

  • 作为“材质编辑器”中的动画材质
  • 作为对位的视口背景
  • 作为在 Video Post 中进行合成的图像

界面

当选择 AVI 作为输出格式时，单击“渲染输出文件”对话框上的“渲染”或“设置”可显示“视频压缩”对话框。

压缩器

使用下拉列表来选择要用于压缩文件的编解码器（压缩器/解码器）。可以使用系统上安装的任何编解码器。

或者，可以渲染未压缩的帧，然后使用外部应用程序来压缩动画。视频文件压缩非常复杂，有很多方面需要注意。

注意:使用 64 位版本的 3ds Max 时，默认压缩器为 DV 视频编码器。此编码器支持的唯一输出分辨率为 720 x 480。要使用其他输出分辨率，请将“压缩器”设置为其他选项。
压缩质量

仅对某些编解码器可用。可用时，使用滑块来指定所需的质量。设置的质量越高，文件就越大。

主帧比率

仅对某些编解码器可用。可用时，使用此设置来指定用于在帧之间进行比较的 delta 关键帧之间的间隔，并生成中间帧。间隔太大将使用 AVI 文件的整体质量下降。

设置

仅对某些编解码器可用。单击此按钮可查看编解码器特定的其他任何选项。这些选项是厂商指定的，并且编解码器之间有所不同。

“附加选项”对话框

当附加两个或更多已经指定了不同材质的对象时，会打开“附加选项”对话框。该对话框提供了合并两个对象中的子材质和材质 ID 的三种方法。

  • 选择要指定材质的可编辑网格对象、多边对象或面片对象。 “修改”面板 “附加”按钮 选择要指定不同材质的对象。

界面

匹配材质 ID 到材质

修改附加对象的材质 ID 数目，使得它们不大于指定到这些对象上的子材质的数目。例如，如果只为一个长方体指定了两个子材质，并且将该长方体附加到另一个对象上，那么长方体就只有两个材质 ID，而不是在创建时所指定的六个。

匹配材质到材质 ID

通过调整所获得的多维/子对象材质的子材质数目，来保持附加对象的原始 ID 指定不变。例如，如果附加两个长方体，它们都只指定了一个材质，但是它们的默认指定为 6 个材质 ID，这样就产生了含有 12 个子对象的多维/子对象材质（六个含有一个长方体材质的实例，六个含有另一个长方体材质的实例）。当保持几何体中的原始材质 ID 非常重要时，请使用此选项。

注意:要使实例子材质唯一，请在“轨迹视图”中高亮显示其轨迹，然后应用“使唯一”按钮功能。也可以通过精简材质修改器中的“使唯一”工具逐一使对象唯一。
不修改材质 ID 或材质

防止调整结果子对象材质上的子材质数目。请注意，如果一个对象上的材质 ID 数目大于其多维/子对象材质上的子材质数目，那么在附加操作后结果面材质指定可能会不同。

精简材质和 ID

影响“匹配材质 ID 到材质”选项。启用该选项后，在对象上不使用的复制子材质或子材质将从附加操作所得到的多维/子对象材质上移除。默认设置为启用。

提示

  • 大多数情况下，使用第一个选项（匹配材质 ID 到材质），并保持“精简材质和 ID”处于启用状态。这样可以保持对象出现的同时，额外的子材质或 ID 最少。
  • 当需要保持原始材质 ID 指定时，可以使用第二个选项（“匹配材质到材质 ID”）。
  • 尽量避免使用第三个选项，除非需要为了与以前的项目兼容而重复 3ds Max 版本 1 中的附加选项。
  • 保持“精简材质和 ID”处于启用状态，除非您有未指定的子材质，并且需要为以后的指定保留该子材质。
  • 使用“附加”前，先执行“编辑”菜单 “暂存”。

解决顶点颜色或贴图的问题

默认情况下，使用平面贴图附加对象，且顶点通道设置为白色。在某些情况下，这可能会导致现有贴图出现不希望的效果或顶点颜色更改。要更改默认行为，请执行下列任一操作：

  • 将 DefaultAttachMappingClear = 1 添加到 .ini 文件。
  • 在 MaxScript 中运行以下调用：meshINI.SetAttachDefaultClearMapChannel on/off

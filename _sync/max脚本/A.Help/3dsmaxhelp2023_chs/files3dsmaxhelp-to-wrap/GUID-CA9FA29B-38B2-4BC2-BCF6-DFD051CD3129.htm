对齐到光标

使用“对齐到光标”可以将选定关键点移到当前时间并保持偏移。使用“移动”、“滑动”或“缩放”选择要对齐的关键点。

  • “轨迹视图” 在“关键点”窗口中，选择要对齐到当前帧的关键点。 “关键点”菜单 “对齐到光标”

“对齐到光标”可用于提取一组分散的关键点，并将它们移到相同的时间位置，同时在每个轨迹内保持选定关键点的时间偏移。也就是说，每个轨迹内的第一个选定关键点移动到当前时间，这些轨迹上的后续关键点也移动相同的距离和方向，同时保持原始时间偏移。

“对齐到光标”可以在“摄影表 - 编辑关键点”模式和“曲线编辑器”模式中使用。

“将关键点与光标对齐”忽略“捕捉帧”的状态，并且始终使用时间滑块设置的确切时间。

过程

要将所有关键点移到当前帧，请执行以下操作：

  1. 打开“曲线编辑器”或“摄影表 - 编辑关键点”模式。
  2. 将时间滑块拖到希望关键点对齐的时间处。
  3. 使用 （移动关键点）、（滑动关键点）或 （缩放关键点）高亮显示一个或多个关键点。
  4. 从“关键点”菜单选择“对齐到光标”。

    每个轨迹中最左侧的选定关键点将移到当前时间。同一轨迹上任何选定关键点与最左侧关键点的相对距离保持不变。

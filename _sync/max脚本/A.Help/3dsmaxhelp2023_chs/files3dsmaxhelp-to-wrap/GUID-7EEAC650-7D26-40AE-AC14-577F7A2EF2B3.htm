创建 Interactive 明暗器

注意:在 3ds Max 2018.4 Update 中，Stingray 基于物理的明暗器被重命名为 Interactive。

Interactive 基于物理的明暗器 (PBS) 遵循物理学定律和能量守恒。通过它，您可以使用粗糙度、法线和金属贴图来平衡散射/反射和微曲面详图/反射率。

要使用 Interactive 基于物理的明暗器创建明暗器图形，请执行以下操作：

  1. 将 DirectX 明暗器材质应用于对象。
  2. 在“DirectX 明暗器”卷展栏上，从下拉列表中选择“Interactive”。
  3. 也可以在“DirectX 明暗器”卷展栏上，单击“打开 ShaderFX”。

    Interactive 明暗器与 ShaderFX 网络使用相同的图形编辑器。

    默认情况下，ShaderFX 编辑器打开一个标准的预设 Interactive 网络。Interactive 输出节点是右侧的“标准基础”节点。

  4. 像编辑常规 ShaderFX 网络一样编辑明暗器网络。
    注意: 尽管您可以像创建 ShaderFX 明暗器网络那样通过 ShaderFX 编辑器创建 PBS 明暗器网络，但无法从先前的 .sfx 文件保存 .sfx 文件并在以后的文件中使用它，或反之亦然。尽管 ShaderFX 编辑器界面类似，但可用节点集不同，因此不可互换。

本节内容

  • Interactive 明暗器的“标准基础”节点

    “标准基础”节点是 Interactive 明暗器树的根，该节点提供明暗器的 DirectX 输出。

  • Interactive 明暗器的标准预设明暗器图形

    为了避免重复创建相同的明暗器图形和降低游戏的速度，可以创建并保存预设明暗器图形，然后在下次创建基于物理的新材质时重用此预设。

  • Interactive 明暗器“参数”卷展栏

    在材质编辑器中，当前 Interactive 明暗器的“参数”卷展栏让您可以更改设置，无需改变明暗器图形。

  • 使用 ShaderFX 编辑器构建新的 Interactive 明暗器

    您可以使用 ShaderFX 编辑器窗口和可用的 ShaderFX 节点构建基于物理的明暗器。

  • 示例：创建 Interactive 明暗器网络以混合两个纹理

    此示例演示了如何使用 Interactive 材质和 ShaderFX 节点创建明暗器网络，以允许您通过绘制顶点混合两个纹理。

  • 为 Interactive 节点创建新的默认输入值
  • 将顶点颜色添加到 Interactive 明暗器网络

    您可以将顶点颜色通道添加到 Interactive 网络，在 3ds Max 中绘制顶点，然后将对象及其关联的 Interactive 明暗处理网络作为 FBX 文件导出到游戏引擎。

  • 非法操作和调试 Interactive 明暗器网络

    ShaderFX 在 Interactive 图形的工作区域底部显示各种错误消息。

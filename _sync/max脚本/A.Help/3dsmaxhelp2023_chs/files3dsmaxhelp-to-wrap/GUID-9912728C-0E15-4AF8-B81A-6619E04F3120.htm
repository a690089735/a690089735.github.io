各向异性高光

各向异性高光适用于创建头发、玻璃或磨沙金属的模型。

  • “材质编辑器” “标准”材质 “基本参数”卷展栏（标准材质）（针对“各向异性”明暗器 “反射高光”组）
  • “材质编辑器” “光线跟踪材质” “光线跟踪基本参数”卷展栏 “明暗处理: 各向异性” “反射高光”组
注意:对于光线跟踪材质，“反射高光”组中将显示“高光颜色”组件。并且，不属于当前明暗器的高光控件将标记为“N/A”。

过程

要增加或降低高光的大小，请执行以下操作：

  • 更改“光泽度”值。

    高光曲线的宽度和高光预览将更改。光泽度为 0% 时，曲线采用其最大宽度。光泽度为 100% 时，曲线会非常狭窄。

要增加或降低高光的强度，请执行以下操作：

  • 更改“高光级别”的值。

    高光曲线的强度和高光预览将更改。在 0% 高光级别上，没有高光。在 100% 高光级别上，曲线采用没有超载的最大高度。如果值大于 100%，则曲线将超载：曲线将变得更宽，且较宽的区域采用最大高光强度。

    高光曲线的形状影响材质高光颜色区域和漫反射颜色区域之间的混合。曲线越陡峭，存在的混合就越少，反射高光的边缘越尖锐。

要调整高光的形状（各向异性），请执行以下操作：

  • 更改各向异性的值。

    白色高光曲线的宽度和高光预览将更改。各向异性为 0% 时，高光曲线相同并且高光为圆形，与在 Blinn 和 Phong 明暗处理中一样。各向异性为 100% 时，白色高光曲线和高光会非常狭窄。

要调整高光的方向，请执行以下操作：

  • 更改方向的值。

    高光预览显示方向的更改。高光曲线的显示不更改。

界面

高光级别

影响反射高光的强度。随着该值的增大，高光将越来越亮。对于标准材质，默认设置为 0；对于光线跟踪材质，默认设置为 50。

单击贴图按钮可指定“高光级别”贴图。这个按钮是一个快捷方式：还可以在“贴图”卷展栏中指定高光级别贴图。

光泽度

影响反射高光的大小。随着该值增大，高光将越来越小，材质将变得越来越亮。默认值为 25。

单击贴图按钮可指定“光泽度”贴图。此按钮是一个快捷方式：也可以在“贴图”卷展栏上指定光泽度贴图。

各向异性

控制高光的各向异性或形状。值为 0 时，高光为弧形。值为 100 时，高光非常狭窄。高光图的一个轴发生更改以显示该参数中的变化。默认值为 50。

单击按钮可指定“各向异性”贴图。此按钮是一个快捷方式：还可以在“贴图”卷展栏上指定各向异性贴图。

方向

更改高光的方向。示例窗显示方向的更改。该值以度数表示，范围从 0 到 9,999。默认设置为 0。

单击贴图按钮可指定方向贴图。此按钮是一个快捷方式：还可以在“贴图”卷展栏上指定方向贴图。

高光图

这两条相交曲线显示调整“高光级别”、“光泽度”和“各向异性”值的影响。如果降低“光泽度”，曲线将变宽；如果增加“高光级别”，曲线将变高。随着调整“各向异性”，白色曲线将发生更改以显示高光如何变宽或变窄。

特性

“属性”子菜单上的命令会显示或设置当前 3ds Max 场景的属性。

  • “文件”菜单 “属性” “属性”子菜单

本节内容

  • 摘要信息

    “摘要信息”显示关于当前场景的统计信息。

  • 文件属性

    使用“文件属性”可以输入有关场景的信息，以后可以通过 Windows 来读取这些信息，它们使用诸如 Windows 资源管理器这样的程序来查看场景文件的属性。

子对象

子对象是对象几何体的单个成员或子集。很多对象都有各种可供独立使用的子对象。例如，可编辑网格对象的子对象层级为顶点、边、面、多边形和元素。要访问子对象，请转至“修改器”面板。显示修改器堆栈时，单击加号按钮以显示对象的层次，然后从该层次中选择子对象层级。在子对象层级，您可以选择子对象、变换选择和应用修改器等。

右侧：顶点子对象选择

联机参考中的很多主题都涉及到子对象。要查看与子对象有关的主题的列表，请按照术语“子对象”进行搜索（包括关键字中的问号）。

子对象 ID

每个子对象（例如顶点、边和面）都有一个唯一 ID，选择子对象时其 ID 将显示在“修改”面板的“选择”卷展栏的底部。

所选择的单个子对象的类型和 ID 显示在“选择”卷展栏的底部（添加了高亮显示）。

当同时选择多个边时，某些功能（例如“对象绘制”“填充”）将使用此 ID。

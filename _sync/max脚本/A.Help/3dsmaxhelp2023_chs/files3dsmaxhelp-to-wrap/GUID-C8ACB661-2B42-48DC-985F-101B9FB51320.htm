材质频率操作符

“材质频率”操作符允许将材质指定给事件，并指定每个子材质在粒子上显示的相对频率。通常，此材质是“多维/子对象”或其他复合材质，可通过为最多 10 个不同的子材质（或材质 ID）中的每一个设置一个百分比来指定频率。“粒子流”根据这些百分比按随机序列将 ID 指定给粒子。还可以采用其他使用子材质的材质，如“双面”和“顶/底”。

  • 粒子视图 在某个事件中单击“材质频率”，或将“材质频率”操作符添加到粒子系统中，然后选择它。

界面

该用户界面出现在“粒子视图”对话框右侧的“参数”面板中。

指定材质
启用时，该操作符将一个材质指定给多个粒子。默认设置为启用。
[按钮]
使用此按钮可将材质指定给操作符。单击此按钮，然后使用“材质/贴图浏览器”选择材质。或者，将材质从“材质编辑器”示例窗拖至此按钮。将材质指定给操作符后，其名称将出现在此按钮上。
指定材质 ID
启用时，该操作符会为每个粒子定义一个材质 ID 数目，然后启用其他参数。默认设置为启用。

通常，此选项应保持启用状态。粒子流将材质 ID 用于复合材质，以了解将那个子材质指定给粒子。

在视口中显示
启用时，在视口中材质会显示为应用于粒子上。
子材质 ID 偏移
将材质继承与多维/子对象材质一起使用时，“材质频率”操作符会将此值添加到“材质 ID”值以创建子材质 ID。粒子流将使用此 ID 根据材质中的子材质 ID 确定继承材质中的哪些子材质，然后将这些子材质应用于其事件中的粒子。
子材质数目
显示指定材质中子材质的数目。
材质 ID #1–10
指定要分配相应材质 ID 的粒子的相对可能性。在要应用到粒子的材质中，为所有 ID 或子材质指定值。例如，如果使用包含五个子材质的“多维/子对象”材质，则需要为材质 ID #1–5 设置值。

该值并非绝对，而是相对于其他设置。继续上一示例，如果希望所有五个材质以相同频率出现，则需要为材质 ID #1–5 设置相同的非零值；实际值是多少无关紧要。另一方面，如果希望材质以递减的频率出现，则最好将较低的“材质 ID”设置设置为相对较高的值，如 100、80、50、33 和 10。在这种情况下，为每个粒子指定材质 ID 1 的可能性是指定材质 ID 3 的两倍，是指定材质 ID 5 可能性的十倍。

材质 ID 指定的实际序列是随机的，并且可以通过更改“唯一性种子”设置来改变。

“唯一性”组

“唯一性”设置可以改变指定 ID 的随机序列。

种子
指定随机化值。
新增特性
使用随机化公式计算新种子。

“颜色修正”贴图

“颜色修正”贴图为使用基于堆栈的方法修改并入基本贴图的颜色提供了一类工具。校正颜色的工具包括单色、倒置、颜色通道的自定义重新关联、色调切换以及饱和度和亮度的调整。多数情况下，“颜色调整”控件会对在 Autodesk Toxik 和 Autodesk Combustion 中发现的颜色进行镜像。

  • 材质编辑器 材质/贴图浏览器 贴图 标准 颜色修正

界面

重要信息:“颜色修正”贴图使用基于堆栈的处理方法，对界面顶部到底部末端的每个卷展栏都应用修改。首先应用“纹理”卷展栏的设置，然后将“通道”卷展栏的设置应用到“纹理输出”卷展栏，依此类推。顺序如下：
  1. “纹理”卷展栏
  2. “通道”卷展栏
  3. “颜色”卷展栏
  4. “亮度”卷展栏

因为此顺序是强制的，所以无法为此贴图的卷展栏重新排序。

“基本参数”卷展栏

[色样]

如果未指定贴图，则 3ds Max 使用此颜色。要更改颜色，请单击色样并使用“颜色选择器”控件。

[贴图按钮]

要指定贴图，单击此按钮，初始标签为“无”。使用材质/贴图浏览器指定贴图后，按钮标签将显示贴图的名称和类型。

提示如果使用“颜色修正”贴图替换现有贴图，则 3ds Max 会提示您选择是否将原贴图保存为子贴图。如果选择是，则原贴图将被放置于此示例窗中。

“通道”卷展栏

[通道操作]

选择要在贴图颜色通道中执行的初始操作：

  • 法线将未经改变的颜色通道传递到“颜色”卷展栏控件。
  • 单色将所有的颜色通道转换为灰度明暗处理。
  • 反转使用红、绿和蓝颜色通道的反向通道分别替换各通道。

    每个通道的反向通道计算方法是由最大值减去一个值，如果是浮点颜色，此值是 1.0，如果是 8 位通道，此值是 255。例如红色更改为青色（绿色 + 蓝色）；绿色更改为洋红（红色 + 蓝色）；蓝色更改为黄色（红色 + 绿色）。

  • 自定义允许您使用卷展栏上其余控件将不同的设置应用到每一个通道。
提示可以使用预设通道操作中的任一种操作（普通/单色/反转）作为自定义设置的起始点。选择“预设”，然后选择“自定义”。之前的设置仍然处于活动状态并可以更改。
红/绿/蓝/Alpha

允许您在每个通道上指定通道操作。仅在启用“自定义”后可用。否则这些字段显示当前设置，例如 RGB 通道均为单色。

使用下拉列表为每个通道选择以下替换值或通道：

  • 红/绿/蓝/Alpha使用选中的通道替换通道。例如，如果设置蓝 = 红，则每个像素的蓝分量会采用其红分量的当前值。
  • 红（反转）/绿（反转）/蓝（反转）/Alpha（反转）使用选中通道的反向通道替换通道。例如，如果设置蓝 = 红（反转），则每个像素的蓝分量会采用该像素红分量当前值的反转值。

    每个通道的反向通道计算方法是由最大值减去一个值，如果是浮点颜色，此值是 1.0，如果是 8 位通道，此值是 255。例如红色更改为青色（绿色 + 蓝色）；绿色更改为洋红（红色 + 蓝色）；蓝色更改为黄色（红色 + 绿色）。

  • 单色将颜色通道转换为灰度。要确定通道的灰度值，3ds Max 将每个像素的红、绿和蓝通道值相加，然后除以 3。例如，如果 RGB 值为 0.5、0.4 和 0.0，则像素的每个通道的单色值为 0.3。
  • 1将通道设置为最高可能值；效果为最大限度的打开。例如，如果某一 24 位或 32 位贴图中的像素原始颜色为 R=50、G=75 和 B=100，若设置绿色 = 1，则结果为 R=50、G=255 和 B=100。
  • 0将通道设置为最低可能值；效果为全部关闭。例如，如果某一像素的原始颜色为 R=50、G=75 和 B=100，若设置绿色值为 0，则结果为 R=50、G=0 和 B=100。

“颜色”卷展栏

此卷展栏提供三种用于总体颜色转换的控件。这些控件对“通道”卷展栏的输出起作用。要使用原始贴图，请确保将“通道”卷展栏设置为“普通”。

色调切换

使用标准色调谱更改颜色。使用滑块或数值控件确定如何重新映射贴图中的颜色。要重置为 0，请右键单击滑块。范围从 -180 到 180。

此控件与 Autodesk Combustion 和 Autodesk Toxik 中的“色调切换”控件功能相同。

饱和度

贴图颜色的强度或纯度。降低“饱和度”值会移除颜色，使图像趋向灰度；提升其值，将增强颜色。要修改此值，请使用滑块或数值控件。要重置为 0，请右键单击滑块。范围为 -100 至 100。

此控件与 Autodesk Combustion 和 Autodesk Toxik 中的“饱和度”控件产生相同效果。

色调染色

根据色样值色化所有非白色的贴图像素。灰度值（包括黑色和白色）无效。

强度

“色调染色”设置的程度影响贴图像素。范围为 0 至 100。

“亮度”卷展栏：“标准”

“亮度”卷展栏上的“标准”选项提供以下两个易用控件：

亮度

贴图图像的总体亮度。要修改此值，请使用滑块或数值控件。要重置为 0，请右键单击滑块。范围为 -100 至 100。

对比度

贴图图像深、浅两部分的区别。要修改此值，请使用滑块或数值控件。要重置为 0，请右键单击滑块。范围为 -100 至 100。

“亮度”卷展栏：“高级”

此“高级”控件与 Autodesk Toxik“照片实验室”功能中提供的控件具有相似的功能。使用此工具可以在贴图中模拟摄影机曝光以及照片显影效果。可以逐步更改曝光以提亮或变暗贴图，在亮度中提供可感知的相对统一的更改。“照片显影”调整可以生成具有不同颜色分布的图像。

[曝光方法]

从下拉列表中选择表示曝光的方法：

  • 增益将像素颜色值乘以此值。
  • F 制光圈在摄影中，增加 1 将增大一倍亮度，并增加因子为 2 的增益。
  • 打印机灯光一个可定义的设置（参见“每次停止的打印机灯光”），通过用“每次停止的打印机灯光”设置的值 (N) 增加此值可增大一倍亮度（N 打印机灯光 = F 制光圈）
RGB/R/G/B

可以同时为所有三个颜色通道更改设置，也可以单独为每一个通道更改设置。此外，可以使用复选框切换单个通道的设置。

Gamma/对比度

Gamma 校正量可以通过对比度或常用 Gamma 指数表达。增加 Gamma 指数可减少对比度。

轴

应用的 Gamma 校正大约等于轴值。即，等于轴值的像素值将不会发生改变。如要使用 Gamma 校正更改贴图的对比度而不需要影响特定的亮度层级时，此功能会非常有用。

提升/偏移

提升仅是一种添加到所有像素值的统一偏移（为不同颜色分量的不同偏移）。提升通常作为过程的最后一步使用，也可用于控制贴图的总体亮度。

每次停止的打印机灯光

当使用“打印机灯光”曝光方法时，此设置可确定打印机灯光数等于一个 F 制光圈；即，所需数值将增大一倍曝光或减半曝光。

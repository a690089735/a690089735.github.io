对齐

对齐可以将当前选择与目标选择进行对齐。

  • 主工具栏 “对齐”（位于“对齐”弹出按钮上）
  • 默认菜单：“工具”菜单 “对齐” “对齐”
  • Alt 菜单：“编辑”菜单 “变换” “对齐”
  • 键盘 Alt+A

步骤 界面

沿轴对齐对象：

左：X 位置，居中

右上：Y 位置，最小

右下：Y 位置，最大

选择要对齐的对象后，单击“对齐”（在“对齐”弹出按钮上），然后选择要与第一个对象对齐的其他对象。这样可以打开标题栏上有目标对象名称的“对齐”对话框。执行子对象对齐时，“对齐”对话框的标题栏会显示为“对齐子对象当前选择”。

可以将源对象边界框的位置和方向与目标对象的边界框对齐。

可以对任何可变换的选择使用“对齐”工具。如果显示三轴架，则可以将该三轴架（以及其代表的几何体）与场景中任何其他对象对齐。可以使用此方法对齐对象的轴点。

您可根据参考，在 XRef 场景中使用“对齐”弹出按钮上的所有对齐工具（对齐到视图除外）。

执行子对象对齐时，“当前对象”选项和“匹配比例”框处于禁用状态。如果要对齐子对象的方向，请先在主工具栏上切换到“局部”变换模式，以便可以将三轴架与子对象选择正确对齐。

“对齐”弹出按钮上的其他对齐工具有快速对齐、法线对齐、放置高光、对齐到摄影机和对齐到视图。

要将对象与点对象对齐，请执行以下操作：

  1. 创建一个点辅助对象，并将其放置到场景中的目标位置。根据需要，旋转该点辅助对象以调整最终方向。
  2. 选择源对象。
  3. 执行下列操作之一：
    • 在主工具栏上，单击“对齐”。
    • 在“工具”菜单上，选择“对齐”“对齐”。如果使用 Alt 菜单系统，请选择“编辑”菜单 “变换”“对齐”。

    3ds Max 将显示“对齐”光标。它带有一对十字线。

  4. 将光标移到点对象上并单击。

    3ds Max 将打开“对齐当前选择”对话框。如有必要，将该对话框移开，以便可以看到活动视口。

  5. 在“对齐位置”组中，启用“X 位置”。

    选定的源对象变为与点对象的 X 轴对齐。

  6. 启用“Y 位置”和“Z 位置”。

    源对象将移动，以便其中心位于点对象上。

  7. 在“对齐方向”组中启用“X 轴”、“Y 轴”和“Z 轴”重定向该对象，以与该点的坐标匹配。

要按位置和方向对齐对象，请执行以下操作：

  1. 选择源对象（要转为与目标对象对齐的对象）。
  2. 执行下列操作之一：
    • 在主工具栏上，单击“对齐”。
    • 在“工具”菜单上，选择“对齐”“对齐”。如果使用 Alt 菜单系统，请选择“编辑”菜单 “变换”“对齐”。

    3ds Max 将显示“对齐”光标。当光标在有资格的目标对象上时，它将显示十字线。

  3. 将光标定位到目标对象上并单击。

    将显示“对齐当前选择”对话框。默认情况下，该对话框中的所有选项处于禁用状态。

  4. 在“当前对象”和“目标对象”组中，选择“最小”、“中心”、“轴点”或“最大”。

    这些设置会在每个对象上确立成为对齐中心的点。

  5. 启用“X 位置”、“Y 位置”和“Z 位置”的任意组合开始对齐操作。

    源对象变为沿参考坐标系的轴与目标对象相关联。在给定“当前对象”和“目标对象”设置的前提下，设置上述三个选项可以尽可能精密地移动对象。

  6. 在“对齐方向”组中，启用“X 轴”、“Y 轴”、“Z 轴”的任意组合。

    源对象会相应地重新对齐。如果对象方向已相同，则启用该轴没有任何作用。如果按方向对齐两个轴，则第三个轴将自动定向。

要将 Gizmo 与其他对象对齐，请执行以下操作：

  1. 显示子对象选择的 Gizmo 层级。
  2. 执行下列操作之一：
    • 在主工具栏上，单击“对齐”。
    • 在“工具”菜单上，选择“对齐”“对齐”。如果使用 Alt 菜单系统，请选择“编辑”菜单 “变换”“对齐”。
  3. 单击以选择视口中的目标对象。（可以选择包含 Gizmo 的同一对象，以将 Gizmo 与自身对象的某部分对齐。）
  4. 使用“对齐”对话框中的可用设置调整 Gizmo 变换。

要将几何体的子对象选择与其他对象对齐，请执行以下操作：

  1. 执行下列操作之一：
    • 将对象转化为可编辑网格，然后在任意层级执行子对象选择。
    • 应用“网格选择”或“多边形选择”修改期、选择子对象、应用变换修改器（“网格选择”修改器自身不允许变换。）、然后激活变换修改器的 Gizmo 子对象层级。
  2. 执行下列操作之一：
    • 在主工具栏上，单击“对齐”。
    • 在“工具”菜单上，选择“对齐”“对齐”。如果使用 Alt 菜单系统，请选择“编辑”菜单 “变换”“对齐”。
  3. 在视口中，单击目标对象。
  4. 使用“对齐”对话框控件执行对齐操作。

界面

“对齐位置”组

X/Y/Z 位置

指定要在其上执行对齐的一个或多个轴。启用所有三个选项可以将该对象移动到目标对象位置。

“当前对象”和“目标对象”组

指定对象边界框上用于对齐的点。可以为当前对象和目标对象选择不同的点。例如，可以将当前对象的轴点与目标对象的中心对齐。

  • 最小将具有最小 X、Y 和 Z 值的对象边界框上的点与另一个对象上选定的点对齐。
  • 中心将对象边界框的中心与另一个对象上的选定点对齐。
  • 轴点将对象的轴点与另一个对象上的选定点对齐。
  • 最大将具有最大 X、Y 和 Z 值的对象边界框上的点与另一个对象上的选定点对齐。

“对齐方向（局部）”组

这些设置用于在轴的任意组合上匹配两个对象之间的局部坐标系的方向。

该选项与位置对齐设置无关。可以不管“位置”设置，使用“方向”复选框，旋转当前对象以与目标对象的方向匹配。

位置对齐使用世界坐标，而方向对齐使用局部坐标。

“匹配比例”组

使用“X 轴”、“Y 轴”和“Z 轴”选项，可匹配两个选定对象之间的缩放轴值。该操作仅对变换输入中显示的缩放值进行匹配。这不一定会导致两个对象的大小相同。如果两个对象先前都未进行缩放，则其大小不会更改。

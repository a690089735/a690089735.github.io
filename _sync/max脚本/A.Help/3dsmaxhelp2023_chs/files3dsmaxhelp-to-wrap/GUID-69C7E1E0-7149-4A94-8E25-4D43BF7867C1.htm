曲面贴图修改器（世界空间）

“曲面贴图 (WSM）”修改器将贴图指定给 NURBS 曲面，并将其投影到修改的对象上。将单个贴图无缝地应用到同一 NURBS 模型内的曲面子对象组时，曲面贴图显得尤其有用。它也可以用于其他类型的几何体。

  • 进行选择。 “修改”面板 “修改器列表” “世界空间修改器” “曲面贴图(WSM)”

NURBS 曲面的贴图会按该 NURBS 曲面的法线方向投影到其他几何体上，如果修改的对象位于该 NURBS 曲面的另一侧面上，那么就按法线相反的方向进行投影。

过程

要使用曲面贴图世界空间修改器，请执行以下操作：

  1. 创建用于投影的 NURBS 曲面，对它进行变换以便它包裹想要进行贴图的对象。
  2. 使用 材质编辑器将贴图材质指定给 NURBS 曲面。
  3. 选择要进行贴图的对象。
  4. 使用 “材质编辑器”将相同的材质指定给要进行贴图的对象。
  5. 应用“曲面贴图”世界空间修改器。
  6. 在“参数”卷展栏中，启用“拾取 NURBS 曲面”，然后单击视口中的 NURBS 投影曲面。

    3ds Max 就会按 NURBS 曲面的法线方向将纹理投影到修改的对象上。

    提示要微调网格上贴图的放置，可以使用 NURBS 曲面的“编辑纹理曲面”对话框。

界面

“源纹理曲面”组

这些控件用以选择要进行投影的 NURBS 曲面。

拾取 NURBS 曲面

拾取用于投影的 NURBS 曲面。单击可启用该按钮，然后在活动视口中单击 NURBS 曲面。

曲面

在拾取 NURBS 曲面之前显示“<无>”；在拾取 NURBS 曲面之后显示该曲面的名称。

“贴图通道”组

这些控件用于选择要使用的贴图通道。

输入通道

在投影之前选择要使用的 NURBS 曲面贴图通道。

输出通道

在投影之后选择要使用的修改对象的贴图通道。

“更新选项”组

这些控件用以选择如何更新显示在视口中的贴图。如果在视口中显示贴图处于禁用状态，这些控件无效。

  • 始终在贴图更改时更新视口。
  • 手动仅在单击“更新”时才更新视口。
更新

更新视口。只有在选择了“手动”后，此选项才可用。

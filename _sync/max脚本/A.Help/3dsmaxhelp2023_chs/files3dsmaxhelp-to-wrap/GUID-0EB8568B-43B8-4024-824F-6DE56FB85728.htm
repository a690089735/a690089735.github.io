光圈

在摄影术中，光圈控制通过摄影机镜头的灯光量。

在大多数情况下，虹膜用于控制开口。这些各种各样的设置称为 f 制光圈。f 制光圈越小，开口越大。标准 f 制光圈值为 f1.8、f2.8、f4、f5.6、f8、f11 和 f16。

图 A = f1.8

图 B = f5.6

图 C = f11

光圈也可用于控制景深。景深是用于聚集于场景中固定点（称为焦平面）的技术。焦平面周围的区域保持聚焦状态，同时图像的其余部分模糊显示。光圈越宽（设置的 f 制光圈越小），越模糊。

下一图显示场景中景深的效果。焦平面设置得很短，因此椅子处于聚焦状态，而使背景模糊显示。

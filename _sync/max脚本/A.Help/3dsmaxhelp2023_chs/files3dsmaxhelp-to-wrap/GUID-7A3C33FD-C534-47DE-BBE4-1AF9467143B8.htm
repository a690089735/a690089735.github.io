动画分层

要在 CAT 中管理层，请使用“层管理器”卷展栏。

CAT 中的分层概念

“层管理器”卷展栏可用于 CATRig 中的每一块骨骼。要访问该卷展栏，请选择装备中的任何骨骼，然后转到 “运动”面板。

层应用于整个 CATRig

通过 CAT 的分层系统，每当添加新层时，此新层都会应用于 CATRig 中的每块骨骼。不存在手臂层或手部层等。当添加新层时，每块骨骼都会获得该新层，您可以选择不应使用新层的 CATRig 部位。此过程称为羽化权重。

所有基本层都包含标准的最大值控制器

CAT 中的层系统只是用于保存和管理动画控制器的机制。它支持所有 3ds Max 动画控制器，其中包括所有不同的关键帧控制器（例如，Bezier、TCB、线性、四元数和 Euler）和程序控制器（脚本和表达式控制器及约束）。

本节内容

  • 层管理器 (CAT)

    层管理器是 CATLayer 系统的界面。当使用 CAT 设置动画时，此卷展栏用作分层工作流程的中心。您可以在其中添加层、移除层、上移和下移层、调整全局或局部层权重并设置动画等。

  • 层类型

    CAT 分层系统的多功能性基于它对多种不同层类型的支持。每个层类型都具有不同功能，且用途不同。

  • CATMode

    CAT 提供两种基本操作模式：设置和动画：

  • 混合层

    混合层通常非常简单，如同将层堆栈中层的“全局权重”值制作成动画。不过，有时您可能需要混合各个肢体或禁用某些元素（如腿），以便使用堆栈中的当前层下的其他层的动画。此过程称为羽化权重。例如，对上半身使用关键帧动画，而对腿使用 CATMotion。

  • 羽化权重

    当添加新层时，每块骨骼都会获得该新层，您可以选择您不希望使用新层的 CATRig 部位。此过程称为“羽化权重”。羽化权重是通过编辑各装备元素的局部权重实现的。

  • 层变换 Gizmo

    使用层变换 Gizmo 可在空间中变换动画。

  • 范围视图

    范围视图是 CAT 的非线性动画 (NLA) 视图；通过范围视图，您可以同时移动某个层中的所有关键帧，并重新设置这些关键帧的时间。其优点包括：能够更改动画计时和速度，并修剪动画末尾。

  • 层时间扭曲

    通过“时间扭曲”曲线，您可以使用一根样条线曲线来控制动画层的速度。例如，这使您能够轻松实现矩阵式的子弹时间效果。

  • 复制和粘贴动画层

    可复制和粘贴动画层（包括在多个绑定之间复制和粘贴）。还可以复制和粘贴实例层，这意味着不同绑定可以引用同一动画层。

  • 折叠动画层

    使用“折叠层”按钮可将层堆栈中的所有当前动画转化到一个层。CAT 会使用一致采样将您的所有动画向下绘制到一个层中。

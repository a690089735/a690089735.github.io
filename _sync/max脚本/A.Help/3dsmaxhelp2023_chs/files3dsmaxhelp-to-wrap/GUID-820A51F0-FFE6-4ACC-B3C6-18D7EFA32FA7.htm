在线部署 3ds Max Batch

在线部署 3ds Max Batch：

  1. 在线以管理员身份登录到虚拟机。
  2. 安装 3ds Max。
  3. 在 Windows 资源管理器中，导航到 3ds Max 的安装位置，此后称为 <maxroot>。
  4. 删除 <maxroot>/atf_converter 文件夹和其中的所有文件。
  5. 导航到 <maxroot>/stdplugs，然后删除 IwATFExport.dle 文件。

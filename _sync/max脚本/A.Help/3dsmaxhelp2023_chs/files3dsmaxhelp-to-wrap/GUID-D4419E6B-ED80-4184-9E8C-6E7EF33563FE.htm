Civil View 对象类别

Civil View 具有自己的方法来引用 3ds Max 中不同类型的对象。

该方法有助于将土木工程可视化模型的内容组织到易于管理的层次中。“Civil View 浏览器”面板和“Civil View 对象属性”面板广泛使用了这些类别。

Civil View 几何体导入面板

1. 导入的路线

导入的路线可以用作 Civil View 中创建的其他对象的参考路径。永久链接将保留到源数据。

导入的路线包含基于桩号的几何数据。此数据与用于在源 Civil Design 应用程序中设计路线的路径的水平和垂直剖面实体相关。

源文件类型：VSP3D 文件

2. 导入的图形

导入的图形可以用作 Civil View 中创建的其他对象的参考路径。永久链接将保留到源数据。

源文件类型：GENIO、MX®、12D®、VSP3D、LandXML 或 DXF™ 文件

3. 导入的曲面

导入的三角化地面曲面可以用作可视化模型的基本地面曲面。永久链接将保留到源数据。

源文件类型：GENIO、MX、12D、VSP3D、LandXML 或 DXF 文件

4. 导入的管网

导入的管网用作一个容器，用于管理导入的管道对象和结构对象。管网可以用于对地下设施（例如排水系统和供水系统）进行建模。

源文件类型：VS3PD 文件

5. 导入的点

导入的点可以用作放置 Civil View 中创建的其他实例化对象的参考点。永久链接将保留到源数据。

源文件类型：VSP3D 文件

对象放置样式 (OPS) 编辑器

6. 车辆

每个面都具有内置材质通道 ID 的静态车辆网格对象，允许自动放置材质。

会对在 Civil View 中创建的每个车辆应用一种随机车身颜色，甚至可以控制模拟的车辆前灯。

源文件类型：Civil View 对象 INI 文件

7. 家具

具有永久可编辑参数的静态或参数化对象。

源文件类型：Civil View 对象 INI 文件

8. 树

使用 Civil View 树对象插件和漫反射/不透明度位图，可以快速轻松地对树进行建模。

源文件类型：Civil View 对象 INI 文件

9. 标志

参数化 Civil View 标志对象插件可用于多种标志和栏柱配置。道路标记符号在 Civil View 中也分类为路标，并且也可以快速轻松地进行建模。

源文件类型：Civil View 对象 INI 文件

10. Civil View 摄影机

可以平滑轻松地对 Civil View 摄影机设置动画，以生成驶过或飞越动画。摄影机的方向锁定在路径的方向，但也可以在任何相对方向平移。

源文件类型：Civil View 对象 INI 文件

11. Civil View 基本体

具有预先指定的材质的 3ds Max 基本体对象（圆柱体、长方体、四棱锥等）。

源文件类型：Civil View 对象 INI 文件

扫描对象样式编辑器

12. 扫描对象样式编辑器

扫描对象可用于生成一系列沿一个或多个参照图形“放样”或“扫描”的曲面。

源文件类型：扫描对象样式

道路标记样式编辑器

13. 道路标记样式编辑器

可以沿现有图形覆盖道路标记“功能区”。可以随时更改与这些道路标记对象关联的大量参数。

源文件类型：道路标记样式

主 Civil View 下拉菜单中的“创建森林”选项

（仅当安装 Forest Lite 或 Forest Pro 时）

14. 森林

一种创建大量树对象而对场景复杂性影响较小的方式。

仅当 Civil View 与 iToo Software 发行的 Forest Pro 或 Forest Lite 插件结合使用时才可用。

轨道对象样式 (ROS) 卷展栏

15. 轨道对象

特殊 Civil View 参数化轨道对象。用于对公路护栏、围栏、噪声挡墙和铁轨进行建模。

源文件类型：轨道对象样式

建筑对象样式 (BOS) 编辑器

16. 建筑对象

特殊 Civil View 参数化建筑对象。用于从导入的楼层面积图形对简单三维纹理建筑几何体进行建模。

源文件类型：建筑对象样式

通道场对象

  • “创建”面板 “几何体” “流体” “设置”卷展栏 “模拟视图” “解算器参数”面板 “通道场对象”

使用通道场对象可修改流体模拟中体素或粒子通道的值，例如液体密度或泡沫。

“通道场参数”卷展栏

“影响”组

液体
启用时，通道场将影响模拟的液体组件。
泡沫
启用时，通道场将影响模拟的泡沫组件。
模式

设置要修改粒子还是体素上定义的通道。一些通道仅适用于粒子或体素，而另一些可同时适用于两者。

影响通道
设置要修改的特定通道。

可用通道如下：

  • 年龄
  • 空气距离
  • 涡流
  • 曲率
  • 密度
  • 水滴
  • 膨胀速率
  • 位置
  • 粘附带宽
  • 粘附强度
  • UV
  • 速度
  • 涡度
使用边界
启用时，仅在由对象的边界图形指定的体积内应用效果。
注意:不同于发射器、碰撞对象、导向和泡沫遮罩的边界控件，您可以使用长方体或球体之外的其他图形来定义体积。
反转
启用时，将在边界外部（而非内部）应用效果。

“覆盖值”组

覆盖值
启用时，将通道中的每个值设置为介于“覆盖”属性块中“最小值”和“最大值”之间的均匀分布随机值。
最小值
设置最小随机值。
最大值
设置最大随机值。

“增量”组

增量
启用时，为通道中的每个值添加正或负增量。增量是介于“覆盖”属性块中“最小值”和“最大值”之间的均匀分布的随机值，随着时间的推移而增加，每个帧以增量为基数发生变化。
最小值
设置最小随机值。
最大值
设置最大随机值。

“缩放”组

缩放
启用时，将通道中的每个值设置为介于“缩放”属性块中“最小值”和“最大值”之间的均匀分布随机值。
最小值
设置最小随机值。
最大值
设置最大随机值。

“钳位”组

钳位
启用时，将通道中的每个值钳制在“钳位”属性块中的“最小值”和“最大值”之间。
最小值
设置最小钳位值。
最大值
设置最大钳位值。

操作步骤

要将通道场对象添加到模拟，请执行以下操作：

  1. 从“辅助对象”面板中，选择“体积”并将体积辅助对象添加到场景中。
  2. 从“模拟视图”窗口的“液体属性”面板中，单击“通道场”卷展栏中的“拾取”以从视口中拾取体积辅助对象。或者，可以单击“添加”，然后从对话框中选择辅助对象。
  3. 从“模拟视图”窗口的“解算器参数”面板中，在“通道场对象”下选择所添加的体积辅助对象。将显示“通道场参数”卷展栏。
  4. 在“通道场参数”卷展栏中，为通道场配置各种设置。

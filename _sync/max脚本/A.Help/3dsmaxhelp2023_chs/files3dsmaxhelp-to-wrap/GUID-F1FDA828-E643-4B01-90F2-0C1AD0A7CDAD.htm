“平移/平行移动”、“2D 平移缩放”和“穿行”弹出按钮

在“透视”视口和“摄影机”视口中，此弹出按钮允许您在“穿行”导航、“2D 平移缩放”或“平移/平行移动”之间进行选择。

  • 平移或平行移动

    “平移或平行移动”按钮的使用取决于您所在视口的种类。

    • 透视视口（平移）
    • 摄影机视口（平移摄影机）
  • 2D 平移缩放模式

    2D 平移缩放模式可用于平移或缩放视口，而无需更改渲染帧。（仅限于 Nitrous 视口。）

  • 穿行

    “穿行”按钮是启用穿行导航的方法之一。

正交视口或者聚光灯视口不显示此弹出按钮。聚光灯视口不提供穿行导航，但会显示“平移和 2D 平移缩放”弹出按钮。正交视口仅显示“平移”按钮。

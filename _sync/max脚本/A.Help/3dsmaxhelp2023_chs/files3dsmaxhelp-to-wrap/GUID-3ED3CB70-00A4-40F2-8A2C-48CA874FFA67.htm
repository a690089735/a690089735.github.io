3D DWF 文件

使用 3D DWF 发布，可以导出三维模型的 Design Web Format™ （.DWF™）文件，这样导出的效果几乎与渲染的场景有相同的视觉精度。DWF 文件是高度压缩文件。您可以与没有 3ds Max 的团队成员轻松共享这些文件。

使用 3ds Max 附带的 Autodesk Design Review 程序可以在 Web 上轻松发布和查看 DWF 文件。

通过 DWF 还可以快速查看模型，不必花时间设置动画或渲染它们，因为您可以使用 DWF 查看器中的“环绕”功能来预览模型。例如，场景的 DWF 文件可以让您和其他人快速查看一个交互模型以及每个对象的属性。

3D DWF 文件的接收者可以使用 Autodesk® Design Review 进行查看和打印。该工具是 3ds Max 安装程序的一个可选部分。有关使用查看器的更多信息，请参见 Autodesk Design Review 帮助系统。

带有 3D DWF 导出的功能支持

  • 2D 和 3D 样条线。导出样条线时，不需要启用“渲染”卷展栏上的“在视口中启用”设置。
  • 3D 几何体
  • 具有漫反射贴图通道、UV 贴图通道和材质 ID 的纹理贴图材质。
    注意:启用了“真实世界比例”的材质在 Autodesk Design Review 程序中显示更为精确。
  • 程序材质，但仅作为粗糙的低分辨率的近似。应该对所有具有程序材质的对象禁用“真实世界贴图大小”。
  • 可以正确导出具有透明度的材质，Autodesk Design Review 程序可以识别它们的不透明度值。即使当您盘旋视图时，对象在具有透明材质的其他对象后面通过，也仍然可见。

3D DWF 导出的限制

材质和环境

  • 环境背景不导出，Autodesk Design Review 程序使用自己背景颜色设置。
  • 不导出诸如“雾”这样的环境效果。
  • 使用反射贴图，如“平面镜”的材质在导出后不产生反射。
  • 不支持双面材质。但是，可以通过启用“渲染设置”对话框上的“强制双面”来避免这样的问题。
    注意:启用“强制双面”会影响整个场景。这会降低 Design Review 的性能，因为它必须处理场景中所有对象的双面显示。
  • 不导出第三方提供的材质。

    具有不支持材质的对象在查看器中显示它们的漫反射颜色。AutoCAD 建筑材质以环境光颜色进行显示。

  • “导出”并不支持所有的材质参数，及时带有支持的材质。因此导出到 Design Review 中的材质看起来可能不象渲染图像中的样子。

灯光

  • 不能导出场景照明。

摄影机和动画

  • 导出已命名的摄影机视图。导出器在场景中创建每个摄影机的 DWF 视图。您可以从 Autodesk Design Review 程序中的“视图”面板选择这些视图，但是摄影机不会以其他方式呈现为场景中的对象。
  • 不支持动画，但可以发布导出时的帧。

自旋操作符

“自旋”操作符给事件中的粒子指定角速度，并且可选设置角速度的随机变化。每事件每粒子应用自旋一次，除非使用的是“速度空间跟随”选项；不过，该设置可以设置动画。

  • “粒子视图” 在某事件中单击“自旋”操作符，或向粒子系统中添加“自旋”操作符，然后选择它。

要仅指定粒子的方向，请使用“旋转”操作符。

界面

该用户界面出现在“粒子视图”对话框右侧的“参数”面板中。

自旋速率
旋转速率以每秒度数为单位。
变化
自旋速率可以改变的最大值，以每秒度数为单位。实际变化只为每个粒子随机计算一次。

“自旋轴”组

“自旋轴”下拉列表
“自旋轴”设置包括可以在随机的或特定的轴上应用自旋的选项，并且可选设置自旋轴的随机变化。默认设置为“随机 3D”。
  • 随机 3D让每个粒子围绕任意的、随机的 3D 轴自旋。
  • 世界空间在世界坐标空间中指定自旋轴。

    使用 X/Y/Z 设置可指定自旋轴。

  • 粒子空间在每个粒子的局部坐标空间中指定自旋轴。

    使用 X/Y/Z 设置可指定自旋轴。

  • 速度空间粒子方向的坐标空间由刚进入事件的粒子方向确定。

    Z 轴与 X 轴垂直并朝上，尽量考虑世界空间的同时，将 X 轴与粒子方向对齐。

    使用 X/Y/Z 设置为所有粒子指定方向。

  • 速度空间跟随粒子方向的坐标空间由整个事件中的粒子方向确定。使用此选项可允许粒子在事件期间更改方向以保持自旋轴与粒子方向对齐。

    Z 轴与 X 轴垂直并朝上，尽量考虑世界空间的同时，将 X 轴与粒子方向对齐。

    使用 X/Y/Z 设置为所有粒子指定方向。

X/Y/Z
使用这些项可设置自旋轴。不能和“随机 3D”选项共同使用。默认值为 0,0,1。范围为 –1.0 至 1.0。

要指定一个轴，请将相应的参数设置为任一非零值，并将其他参数设置为 0。负值将翻转轴，并反转旋转的方向。为一个以上的轴指定一个非零值时，这些数值会发挥作用，在此情况下它们产生的效果是相互关联的。例如，要让自旋轴定向于正向 X 轴和正向 Y 轴之间一半的位置，应该将 X 和 Y 轴设置为相同的正数。而实际的值无关紧要。同样，要将自旋轴设置为从 X 轴至 Y 轴的 30 度（角度的 1/3），您要将 Y 值设置为 X 值的两倍。例如，X 设置为 0.2 且 Y 设置为 0.4，或者 X 设置为 0.5 且 Y 设置为 1.0。

散度
定义自旋轴方向的变化范围（以度为单位）。实际偏离是在此范围内随机计算得出的。不能和“随机 3D”选项共同使用。默认值为 0。范围从 0 到 180。

“唯一性”组

“唯一性”设置会影响自旋速率变化的随机性、带有“随机 3D”选项的自旋轴和带有其他选项的“分散度”。

种子
指定随机化值。
新增特性
使用随机化公式计算新种子。

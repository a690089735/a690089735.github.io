加载和保存 STP 文件

足迹 (STP) 文件保存足迹，但不保存躯干关键点。

STP 文件格式是 ASCII 格式，使开发者能够编写程序，从而为 Biped 运动生成足迹文件。随 ..\cstudio\docs 目录中的 character studio 一起提供的联机文档 stp.rtf 描述了 STP 格式。

过程

要加载足迹数据，请执行以下操作：

  1. 选择要通过已保存足迹制作动画的 Biped，然后转到 “运动”面板。
  2. 在“Biped”卷展栏中，确保未处于“体形模式”，然后单击 （“加载文件”）。
  3. 在对话框中，选择足迹文件 (.STP) 作为要加载的文件类型。
  4. 选择要加载的足迹文件，然后单击“确定”。

足迹便加载到了 Biped，并且生成了新的默认关键点以匹配足迹。

要保存足迹数据，请执行以下操作：

  1. 选择要保存其足迹的 Biped，然后转到 “运动”面板。
  2. 在“Biped”卷展栏中，确保未处于“体形模式”，然后单击 （“保存文件”）。
  3. 在文件对话框中，选择足迹文件 (.STP) 作为要保存的文件类型。
  4. 输入足迹文件的文件名，然后单击“确定”。

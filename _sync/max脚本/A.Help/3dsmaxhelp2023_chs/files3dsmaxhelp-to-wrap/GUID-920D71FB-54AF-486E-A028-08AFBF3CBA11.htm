为 Interactive 节点创建新的默认输入值

注意:在 3ds Max 2018.4 Update 中，Stingray 基于物理的明暗器被重命名为 Interactive。

在 Interactive 图形的 ShaderFX 编辑器中，创建一个节点，例如“添加”。单击节点上的任何输入属性可在“属性”面板上显示其设置。系统已对该节点指定了一个默认值，本例中为 0 0 0 0（float4 值）。

您可以将此默认值更改为以下任何数据类型。属性端口还会随其默认值更改为不同的数据类型而更改颜色。

  • int（深蓝色端口）
  • float（绿色端口）
  • float2（黄色端口）
  • float3（浅蓝色端口）
  • float4（品红色端口）

通过设置输入默认值，可以不提供外部连接而执行操作。

示例：更改“添加”节点的输入默认值：

  1. 在自定义图形中，添加一个“数学”“添加”节点。
  2. 单击第一个值输入。“属性”面板会显示初始默认值 0 0 0 0。
  3. 将默认值字符串更改为 1 0 1。

    “添加”节点现在显示新值 (1.0 0.0 1.0)，并且输入套接字为浅蓝色，表示输入类型为“float3”。

  4. 单击第二个值输入。将默认值字符串更改为 1 0 –1。
  5. 在“添加”节点上，单击“渲染样例” 图标。预览样例现在显示为红色，因为节点已添加了两个 float3 值 (1, 0, 1) + (0, 0, –1) = (1, 0, 0)。

    这是设置值而不连接任何外部输入节点的一种方法。但是，只要将节点连接到输入，您指定的默认值即将被清除并替换为新的连接。

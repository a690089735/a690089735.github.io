设置渲染软件

如果已经配置了渲染网络上计算机的 TCP/IP，则可以准备加载 3ds Max。

计划用于网络渲染的每个系统上都需要安装 3ds Max。在所有系统上安装 3ds Max 之后，至少有一台计算机需要获得授权。这是您可以交互式运行并用于提交供网络渲染作业的 3ds Max 副本。

有关安装 3ds Max 的详细信息，请参考“安装指南”。

注意:使用扫描线渲染器、旨在操作专用渲染服务器的系统无需获得 3ds Max 的授权。

下一步

设置目录

本节内容

  • 设置目录

    在网络渲染过程中，使用公用目录（在网络上共享的目录）可以访问所有渲染服务器需要的文件。您可以组织和共享这些目录，并在必要时进行安装。

  • 共享目录

    您可以共享目录所在计算机中的目录。从而可以使网络上的其他计算机也能够访问该目录。下面是一些简要说明。有关详细信息，请参见 Windows Vista、Windows XP 或 Windows 2000 文档。

  • 装载目录

    您可以将目录装载到驱动器盘符，作为使用 UNC 名称的替代方式。例如，在混合的 UNIX/XP/2000 网络中，您可能需要装载输出目录。

  • 使用“配置项目路径”

    仅渲染机器不需要任何形式的授权。但是，您不能使用未经授权的 3ds Max 版本访问“配置项目路径”对话框，来指定服务器搜索位图文件的替代位置。

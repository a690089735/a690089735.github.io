Maya 2023 Open Source Components

This document provides information about customizable open source components used in Maya 2023.

If you need to customize any open source libraries for your project, ensure that you use the versions listed here.

OSC:

  • Alembic: updated to version 1.8.3
  • Boost: updated to version 1.76
  • OpenColorIO: updated to version 2.1.1
  • OpenEXR: updated to version 3.1.3
  • OpenImageIO: updated to version 2.3.10
  • OpenSSL: updated to version 1.1.1k
  • PTex: updated to version 2.4.1
  • PyMEL: updated to version 1.3
  • Python: updated to version 3.9.7
  • TBB: updated to version 2020u3

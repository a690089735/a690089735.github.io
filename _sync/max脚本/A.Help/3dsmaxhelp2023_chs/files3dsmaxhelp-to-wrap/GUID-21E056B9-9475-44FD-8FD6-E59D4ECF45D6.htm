渲染静止图像或动画

要渲染静态图像，请执行以下操作：

  1. 激活要进行渲染的视口。
  2. 单击主工具栏 “渲染设置”。

    将打开“渲染设置”对话框，其中“公用”面板处于活动状态。

  3. 在“公用参数”卷展栏上，选中“时间输出”组，确保选择“单个”选项。
  4. 在“输出大小”组中，设置其他渲染参数或使用默认参数。
  5. 单击“渲染”按钮。

    默认情况下，渲染输出会显示在“渲染帧窗口”中。

    提示若要不使用该对话框渲染视图，请单击主工具栏 “渲染产品”或按 F9 键（按上次设置渲染）。

要渲染动画，请执行以下操作：

  1. 激活要进行渲染的视口。
  2. 单击主工具栏 “渲染设置”。

    将打开“渲染设置”对话框，其中“公用”面板处于活动状态。

  3. 在“公用参数”卷展栏上，转到“时间输出”组，然后选择时间范围。
  4. 在“输出大小”组中，设置其他渲染参数或使用默认参数。
  5. 在“渲染输出”组中，单击“文件”。
  6. 在“渲染输出文件”对话框中，指定动画文件的位置、名称和类型，然后单击“保存”。

    通常将显示一个对话框，用于配置所选文件格式的选项。更改设置或接受默认值，然后单击“确定”继续执行操作。

    将启用“保存文件”复选框。

  7. 单击“渲染”按钮。
    注意:如果已设置一个时间范围，但是未指定保存到的文件，则只将动画渲染到该窗口。这可能是一个耗费时间的错误操作，因此将就此发出一条警告。
提示一旦采用这种方式渲染动画，无需使用对话框，您就可以通过单击主工具栏 “渲染产品”或按 F9 键（按上次设置渲染）再次渲染该动画。

本节内容

  • 产品级渲染与 ActiveShade 渲染

    对全尺寸详细图像进行产品级渲染。使用 ActiveShade 进行交互式预览渲染。

  • 缺少贴图坐标

    在某些情况下，渲染器将检测需要应用贴图坐标的对象。

  • “渲染设置”对话框

    使用“渲染”可以基于 3D 场景创建 2D 图像或动画。从而可以使用所设置的灯光、所应用的材质及环境设置（如背景和大气）为场景的几何体着色。

  • “渲染进度”对话框

    显示渲染操作的状态。

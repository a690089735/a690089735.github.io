导入预设

3ds Max FBX plug-in包含两个导入工作流程预设：

  • Autodesk Media and Entertainment
  • Autodesk Architectural (Revit)

默认的预设取决于您当前所用应用程序的版本。如果使用的是 3ds Max，则默认预设为“Autodesk Media and Entertainment”。如果使用的是 Revit，请使用“Autodesk Architectural (Revit)”预设。

选择预设

您可以在“FBX 插件”“预设”部分的菜单中选择预设。

“预设”区域

编辑预设

如果更改预设，则预设字段将显示“用户定义”，以表示该预设与其默认状态不同。您可以在临时文件中找到这些“用户定义”状态，在下次使用插件时，这些设置将成为默认设置。

还可以使用“编辑”“保存预设”将您的设置保存为自定义预设。请参见创建自定义预设和编辑预设。

Autodesk Media & Entertainment 预设

Autodesk Media and Entertainment 预设包含适用于大多数动画工作流的最佳设置。在 UI 中您可以看到所有可用的选项，因此可以对它们进行访问和自定义。

FBX Media & Entertainment 预设

Autodesk Architectural (Revit) 预设

“Autodesk Architectural (Revit) 预设”是 FBX UI 的简化版本。它拥有建筑工作流程的最佳设置，可将 Autodesk Revit 应用程序的数据转换为 Autodesk 3ds Max 可用的数据。预设中除了“预设”和“统计信息”部分之外没有其他任何选项。此预设包含从 Revit 到 3ds Max 的工作流所需的所有基本设置，但是您还可以编辑预设，使它包含您所需的任何其他设置。请参见编辑/保存预设。

在此预设中，“动画”选项和“平滑组”选项处于禁用状态，“天空和太阳”选项处于激活状态，而“文件内容”选项设置为“添加到场景”。

FBX Autodesk Architectural 预设

编辑/保存预设

通过“编辑预设”按钮，可使用定义的选项设置来创建和保存自定义预设。

编辑/保存预设选项

通过FBX 导入器 UI 中的“编辑”按钮访问编辑/保存预设选项。

3ds Max FBX plug-in将自定义预设存储在一个文件中，以便您可以与其他用户共享。详细信息请参见存储预设。

有关创建和编辑预设的详细信息，请参见创建自定义预设和编辑预设。

“IK 显示选项”卷展栏（HI 解算器）

以下是用于在视口中启用和禁用各种 Gizmo 的控件。“目标”和“末端”效应器、旋转角度操纵器向量控制柄以及 IK 解算器显示本身都位于此卷展栏中。

  • 创建骨骼链 添加 HI 解算器。 “运动”面板

过程

要在单个链中查看多个 IK 解算器，请执行以下操作：

  1. 在包含多个 IK 解算器的链中，在视口中 选择一个目标。
  2. 在“IK 显示选项”卷展栏中，启用“IK 解算器显示已启用”。
  3. 选择第二个目标并重复该过程。

    无论是否被选中，都仍然显示每个目标。

界面

“末端效应器显示”组

控制 IK 链中的末端效应器的外观。

启用
启用或禁用末端效应器显示。默认设置为禁用状态。
大小
控制视口中的末端效应器 Gizmo 的大小。

“目标显示”组

控制 IK 链中的目标的外观。

启用
启用或禁用目标显示。默认设置为启用。
大小
控制视口中的目标 Gizmo 的大小。

“旋转角度操纵器”组

控制 IK 链中的旋转角度操纵器的显示。启用此控件，然后启用“操纵”模式以显示旋转角度操纵器

启用
启用或禁用旋转角度操纵器。默认设置为禁用状态。
大小
控制视口中的操纵器控制柄的大小。
长度
控制视口中的操纵器的长度。

“IK 解算器显示”组

控制 IK 解算器显示的外观，对于其他解算器，显示起始关节和末端关节之间绘制的线条。如果希望同时查看多个链，请启用此控件。

对于当前选择的链，IK 解算器显示始终处于启用状态。

启用
启用或禁用 IK 解算器显示。默认设置为禁用状态。

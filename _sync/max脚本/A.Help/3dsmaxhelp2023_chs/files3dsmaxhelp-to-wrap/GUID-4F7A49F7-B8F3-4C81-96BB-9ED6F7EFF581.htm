为 CATMotion 参数设置动画

一般情况下，最好不要对单个 CATMotion 参数设置动画，而是采用通过对权重值设置动画来简单地混合 CATMotion 层。这使得您需要管理的关键帧大为减少。

这意味着，如果需要，可以在 CATMotion 编辑器中对所有参数设置动画。使用标准模式，通过在视口中设置关键点或使用轨迹视图可以完成此操作。

在轨迹视图中，CATMotion 数据保存在 CATParent 的相应 CATMotion 层中。

控制器完全镜像 CATMotion 编辑器层次。每个控制器都可见，每个预设层都存在。

标准装备的轨迹视图中的 CATHierarchy

提示 访问特定 CATMotion 控制器的动画参数的最简单方法是在 CATMotion 编辑器中单击该控制器的面板上的“轨迹视图”按钮。

为第二个结的骨盆抬起值设置动画：

  1. 在 CATMotion 编辑器中，访问“骨盆组”“骨盆”“抬起”。
  2. 单击“曲线编辑器”。此时打开轨迹视图，显示所选控制器的全部参数。
  3. 展开 Key2Val。
  4. 根据需要，插入和编辑动画关键帧。
提示 CAT 的一个主要优点是可以将任意浮点型控制器（例如，浮点列表控制器）指定给任意参数。
提示 动画设置偏移控制器（OffsetRot 和 OffsetPos）是一种在运动循环中添加额外操作同时保持辅助运动（如抬头等）的方法。执行此操作的其他（通常更好）方法是直接在调整层中设置动画。

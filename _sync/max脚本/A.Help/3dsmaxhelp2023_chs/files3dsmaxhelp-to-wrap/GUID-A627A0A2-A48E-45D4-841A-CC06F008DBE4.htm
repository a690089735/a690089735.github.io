将对象发送到 Print Studio

您可以使用 Print Studio 直接从 3ds Max 中打印 3D 对象，Print Studio 是随 Autodesk SPARK 技术和 EMBER 3D 打印机一起提供的 3D 打印应用程序。该应用程序仅适用于 64 位计算机。有关详细信息，请参见 http://spark.autodesk.com。

若要安装 Print Studio，请执行以下操作：

  1. 选择要导出的对象。
  2. 选择“文件”菜单 “发送到 Print Studio” “以新场景发送”。

    如果未安装 Print Studio，则会显示一个消息框。

  3. 单击“下载”按钮以下载安装程序。
  4. 运行安装程序以安装 Print Studio。

若要将 3D 对象发送到 Print Studio，请执行以下操作：

  1. 选择要导出的对象。

如果未选择对象，则会发送场景中的所有对象。

  1. 确定模型方向，将 Y 轴或 Z 轴作为上方向轴。Print Studio 默认情况下使用 Z 轴向上导入对象，但可以在 Print Studio 中的“文件”>“首选项”中将其设置为 Y 轴向上。
  2. 选择“文件”菜单 “发送到 Print Studio” “以新场景发送”。
  3. 当 Print Studio 打开时，选择是要打印到 USB 或网络打印机，还是将对象保存到文件中稍后打印。
  4. 选择支持的 3D 打印机的型号：

    •Autodesk Ember

    •Dremel 3D Idea Builder

    •Makerbot Replicator® 2

    •Printrbot Play

    •Printrbot Plus

    •Printrbot Simple

    •Type A Machines Series 1

    •Ultimaker B.V. Ultimaker 2

“文件”菜单

“层”对话框的“文件”菜单包含用于保存和加载绘制的位图的命令。

  • 默认菜单：“工具”菜单 “视口画布” 激活任意绘制工具或单击“层对话框”。 “层”对话框 菜单栏 “文件”菜单
  • Alt 菜单：“材质”菜单 “创建/编辑贴图” “绘制位图(视口画布)” 激活任意绘制工具或单击“层对话框”。 “层”对话框 菜单栏 “文件”菜单
将 PSD 另存为

打开用于将当前纹理另存为分层的 PSD 文件的文件对话框。

将位图（展平的）另存为

打开一个文件对话框，以便将当前纹理的展平版本以该对话框中“另存为类型”下列出的任意格式保存。分层的纹理在 3ds Max 中保持不变。

将位图加载到当前层

打开一个文件对话框，以便选择要加载到活动层的位图文件，该文件将替换活动层的当前内容。如果选择 PSD 文件，则会打开“PSD 输入选项”对话框，并提供塌陷层或加载单个层的选项。

提示这是在不使用“材质编辑器”的情况下，将位图应用于材质贴图的简单方式。
从剪贴板粘贴

用 Windows 剪贴板的内容替换当前层内容。剪贴板内容必须是位图。

例如，可以在 Windows 画图中打开一个图像文件，选择其中的一部分，按 Ctrl+C 将所选内容复制到剪贴板，然后使用此命令将其粘贴到层。

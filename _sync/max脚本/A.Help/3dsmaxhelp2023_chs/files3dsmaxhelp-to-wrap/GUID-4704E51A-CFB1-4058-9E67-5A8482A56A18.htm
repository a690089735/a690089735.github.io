曲面变形修改器（对象空间）

“曲面变形”修改器工作方式与面片变形修改器一样，只是它使用 NURBS 点或 CV 曲面来应用曲面变形，而不使用面片曲面。

  • 选择对象。 “修改”面板 “修改器列表” “对象空间修改器” “曲面变形”
  • 选择对象。 “修改”面板 “修改器列表” “世界空间修改器” “曲面变形(WSM)”
  • 默认菜单：选择对象。 “修改器”菜单 “动画” “曲面变形”
  • 默认菜单：选择对象。 “修改器”菜单 “动画” “曲面变形(WSM)”
  • Alt 菜单：选择对象。 “修改器”菜单 “一致” “曲面变形”
  • Alt 菜单：选择对象。 “修改器”菜单 “一致” “曲面变形(WSM)”

“曲面变形”塑造蛇休息的方式。

过程

要使用“曲面变形”修改器，请执行以下操作：

  1. 选择对象。
  2. 从 “修改”面板 “修改器列表”中，选择“对象空间修改器”“曲面变形”。
  3. 在“参数”卷展栏上单击“拾取曲面”。
  4. 选择 NURBS 点或 CV 曲面。
  5. 通过调整“曲面变形”组中控件变形对象。

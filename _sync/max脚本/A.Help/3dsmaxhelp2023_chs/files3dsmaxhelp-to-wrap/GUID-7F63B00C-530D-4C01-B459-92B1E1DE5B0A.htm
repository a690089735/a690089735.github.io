“作业报告”对话框

您可以生成 ASCII 报告，其包含有关“作业队列”列表中选定的特定作业的详细统计信息。

  • Windows“开始”菜单 “程序” “Autodesk” “Backburner” “监视器” 高亮显示作业。 “作业”菜单 “报告”

界面

“标题”组

除了正文信息之外，请指定要包含在报告文件中的标题类型。

  • 长包含正文信息之外的信息，如作业名称、提交者、起始帧/结束帧、输出宽度/高度、像素纵横比和图像纵横比。
  • 短只在报告中包含列标题。

“记录分隔”组

指定字段之间使用的分隔符类型。例如，如果您使用制表符分隔符，则报告将正确导入到 Microsoft Excel 或 Access 应用程序中。

  • 制表符在报告中的字段之间插入制表符。
  • 空格在报告中的字段之间插入空格。
  • 逗号在报告中的字段之间插入逗号。
使用引号
使用双引号将每个字段括起来。

“输出文件”组

[文本字段]
指定报告的文件名。如需要，可以指定完整的路径。默认情况下，该路径是包含 monitor.exe 文件的目录。
提示如果使用快捷键图标来启动“队列监视器”，则可以在该快捷键的“属性”对话框的“启动位置”字段中指定报告输出的路径。
浏览
显示文件选择器，在此可以为报告指定文件路径。

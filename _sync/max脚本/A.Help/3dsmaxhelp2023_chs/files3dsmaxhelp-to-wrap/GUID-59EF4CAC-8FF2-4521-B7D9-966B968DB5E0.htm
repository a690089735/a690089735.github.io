曲线近似工具 (NURBS)

使用“曲面近似”工具无需转到 NURBS 面板即以更改近似并显示这些设置，而且尤其适用于在多个 NURBS 对象上一次更改这些设置。

  • “实用程序”面板 “实用程序”卷展栏 “更多”按钮 “实用程序”对话框 “曲面近似”

过程

要使用“曲面近似”工具，请执行以下操作：

  1. 在 “工具”面板上，单击“更多”按钮，然后从列表中选择“曲面近似”。
  2. 在“曲面近似”和“曲面显示”卷展栏上设置所需的选项。
  3. 选择要应用该设置的 NURBS 对象。
  4. 在“曲面显示”卷展栏上，单击“选定集”以应用该设置。

本节内容

  • “曲面近似”卷展栏

    “曲面近似”卷展栏上的控件与NURBS 曲面对象的曲面近似控件相同，但是有两个附加的按钮：“选定集”和“重置”。

  • “曲面显示”卷展栏

    “曲面显示”卷展栏上的控件与 NURBS 曲面对象的曲面显示控件相同，但是有两个附加的按钮：“选定集”和“重置”。

从 Revit Architecture 到 3ds Max 的工作流程

当在 Revit Architecture 中完成项目的初始设计、布局和建模后，您可以使用 Autodesk® 3ds Max® 生成高端渲染并添加最终细节。3ds Max 旨在实现与来自 Revit Architecture 的 FBX® 文件的互操作性，它保留了 Revit 项目中的模型几何体、灯光、材质、摄影机设置以及其他元数据。将 Revit Architecture 和 3ds Max 结合使用，设计师就可以扩展建筑信息建模过程以进行可视化设计。

您可以将 3D 视图从 Revit 项目导出到 FBX 文件中，然后再将其导入 3ds Max。然后，还可以创建增强的设计渲染以同 3ds Max 客户一同分享。

通过 FBX 文件格式可将渲染信息传递给 3ds Max，这些信息包括灯光、渲染外观、天空设置，以及 3D 视图的材质分配。通过在导出过程中保留该信息，Revit Architecture 可维持较高的精度，并减少 3ds Max 中所需的工作量。

注意: 使用 Revit 创建时存在一些限制。有关这些限制的产生原因及其解决方法的信息，请参见故障诊断和限制。

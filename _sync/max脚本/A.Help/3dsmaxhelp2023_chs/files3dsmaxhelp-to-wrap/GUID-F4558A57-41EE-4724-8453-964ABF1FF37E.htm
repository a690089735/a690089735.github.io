加载自定义 UI 方案

在“加载自定义 UI 方案”对话框上，可以指定要加载的自定义 UI 方案的基本文件名。

  • 默认菜单：“自定义”菜单 “加载自定义用户界面方案”
  • Alt 菜单：“自定义”菜单 “自定义” “加载自定义用户界面方案”

要保存自定义 UI 方案，请使用“保存自定义 UI 方案”对话框。

有关保存和加载自定义用户界面的信息，请参见保存和加载自定义 UI 界面。

也可以使用自定义 UI 与默认设置切换器加载自定义 UI 方案。

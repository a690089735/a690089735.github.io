使用层资源管理器

层资源管理器是在使用以下任意命令时打开的场景资源管理器版本。在此模式下，您可以查看对象、层和层层次，而不是对象层次或组。使用层资源管理器可以创建和嵌套层，然后在层之间移动对象。

  • 主工具栏 “切换层资源管理器”
  • 层工具栏 “切换层资源管理器”
  • 默认菜单：“工具”菜单 “层资源管理器”
  • Alt 菜单：“场景”菜单 “管理场景内容” “层资源管理器”
  • “场景资源管理器” “视图”工具栏 打开下拉列表。 层资源管理器
注意:这些命令打开一个具有专门为管理层而配置的列的场景资源管理器版本。此外，可以通过将任何场景资源管理器设置为“按层排序”模式来使用该场景资源管理器管理层；这不会更改列设置。

创建新层:

创建新层时，默认情况下，3ds Max 将按顺序对其进行命名：Layer001、Layer002，依此类推。您可以通过直接键入的方式来重命名新层，也可以通过双击标题然后再键入名称来重命名。3ds Max 随机指定颜色给所有新层。可以接受默认设置，也可以指定其他颜色。

注意:默认层的名称为“0（默认）”，无法重命名。
  1. 在主工具栏上，单击“切换层资源管理器”。

    “层资源管理器”对话框将会打开，并列出场景中的所有层和对象。

  2. 此时可以选择要添加到新层的对象。创建后，它们将自动添加到新层。
    注意:如果在创建层时有选择一个层，则新层将成为该高亮层的子层。如果选择了多个层，则新层将添加到最近选择的层中。
  3. 在“工具”工具栏上，单击“新建层”。

    3ds Max 将在列表中创建一个新层。

    新层处于活动状态 ()，因此随后创建的任何对象都将自动添加到此层。

激活层：

活动层接收所有新创建的对象。一次只能激活一个层。

注意:对于其他与层相关的活动（例如更改属性），无需更改活动层。对于多层活动（例如更改属性），只需选择要修改的层。
  • 在“层资源管理器”对话框中，单击要激活的层名称旁边的非活动层图标 ()。

    图标会变为蓝色 ()，以指示相应层处于活动状态，因此随后创建的任何对象都将自动添加到此层。

使用另一种方法激活层：

  • 在“层”工具栏 “层”列表中，选择一个层。

    选择的层将成为当前层。

在层之间移动对象：

  1. 使用“层资源管理器”对话框选择要移动的对象。您可以选择任何层中的任何对象。
  2. 将选择的任何对象拖动到列表中的某个层。鼠标光标图标类似于两个重叠的矩形时（如下图所示），释放鼠标按键。

    拖动的所有对象现在即属于目标层。

嵌套层：

只需通过将层拖动到其他层即可创建层级，这被称为“嵌套”层。父层控制任何子层的隐藏/冻结属性，但其他可由层成员继承的设置（例如“显示”和“渲染控制”）则仅由直接父层控制。

注意:虽然默认图层图层 0 可以包含其他图层，但不能嵌套在其他图层内。
  1. 要嵌套多个层，请首先选择要移动的层。如果要嵌套单个层，则无需这样做。
  2. 将一个层（选择的某个层或任一层）拖动到列表中的另一个层。鼠标光标类似于两个重叠的矩形时（见上图），释放鼠标按钮。

    拖动的层现在嵌套在目标层中。

从层次中移除层：

  1. 选择一个或多个嵌套层。
  2. 单击鼠标右键并从四元菜单中选择“取消嵌套”，或拖动到左侧或下方的空白空间。

    选择的层将移动到层级的顶层。

隐藏或取消隐藏层：

隐藏层还会隐藏任何成员对象，以及属于嵌套在隐藏层中的层的对象。

可见层或对象由 图标表示，隐藏的层或对象由 或 图标表示。

  1. 要隐藏或取消隐藏多个层，请首先使用“层资源管理器”对话框选择要影响的层。如果要隐藏或取消隐藏单个层，则无需执行此步骤。
  2. 单击任何一项左侧的 或 图标，可以隐藏或取消隐藏。

冻结或解冻层：

冻结层还会隐藏任何成员对象，以及属于嵌套在冻结层中的层的对象。

在编辑与特定层相关联的对象时，如果还要查看其他层上的对象而不进行编辑，那么冻结层非常有用。在冻结层上不能编辑或选择对象；但是，如果层处于打开状态，该对象仍然可见。可以激活冻结层，也可以向冻结层添加新对象。

解冻的层或对象由“冻结”列中的 图标表示，冻结的层或对象由 或 图标表示。

  1. 要冻结或解冻多个层，请首先使用“层资源管理器”对话框高亮选择要影响的层。如果要冻结或解冻单个层，则无需执行此步骤。
  2. 在“冻结”列中，单击任何一项的 或 图标，可以冻结或解冻。

    列表中的图标外观将进行切换。

将颜色指定给层：

可以使用“层资源管理器”为层指定颜色。例如，可以将红色指定给名称为 HVAC 的层，以帮助识别场景中的机械设备。

  1. 在主工具栏上，单击“切换层资源管理器”。
  2. 如果需要，请在布局中添加“颜色”列（请参见场景资源管理器列）。
  3. 单击要改变其颜色的层的色样，然后使用“颜色选择器”对话框选择一种新颜色。

    这将更改层的颜色。对于层中要继承此颜色的对象，其颜色必须设置为“按层”。要执行此操作，请参见以下过程。

允许按层控制对象属性：

您可以设置由包含层控制的四个不同类别的对象属性：“线框颜色”、“显示属性”、“渲染控制”和“运动模糊”。您可以一次性切换所有四个类别的“按层”选项，也可以单独切换每个类别的选项。

  1. 在“层资源管理器”中，高亮显示要调整的对象。
  2. 执行下列任一操作：
    • 右键单击，然后从四元菜单的“层”象限中选择“按层”。这将同时切换所有四个属性类别的“按层”选项。
    • 右键单击，然后从四元菜单中选择“属性”（右下角）。使用“对象属性”对话框控件为每个类别单独切换“按层”/“按对象”。

重命名层：

您可能希望重命名一个层来更好地定义其内容在场景中的使用方式。在 3ds Max 会话期间，可以随时重命名除“层 0”以外的任何层。

  1. 在主工具栏上，单击“切换层资源管理器”。
  2. 在“层资源管理器”对话框上，选择要重命名的层。
  3. 再次单击层的名称以激活文本输入模式（或双击以高亮显示整个名称）并输入新名称。

删除层：

在 3ds Max 会话期间，可以随时删除空层。不能删除默认层（0（默认））或包含对象的层。如果删除激活的层，则会使默认层处于激活状态。

  1. 在主工具栏上，单击“切换层资源管理器”。
  2. 在“层资源管理器”对话框列表中，选择一个或多个层，然后右键单击并从四元菜单中选择“删除”。

打开“层属性”对话框选择层：

  1. 在主工具栏上，单击“切换层资源管理器”。
  2. 在“层资源管理器”对话框列表中，选择一个或多个层。
  3. 单击右键，然后从四元菜单中选择“属性”。

本节内容

  • 关于场景层

    层就像透明的叠加，可以用来组织和组合不同类型的场景信息。您创建的对象拥有一些常用属性，包括颜色、渲染性和显示。层也具有这些属性，其中包含的对象也可继承这些层的属性。

  • 层资源管理器

    “层资源管理器”是一种显示层及其关联对象和属性的“场景资源管理器”模式。您可以使用它来创建、删除和嵌套层，以及在层之间移动对象。还可以查看和编辑场景中所有层的设置，以及与其相关联的对象。

  • “层属性”对话框

    “层属性”对话框类似于“对象属性”对话框。您可以在此处更改渲染、运动模糊并显示一个或多个选定层的设置。另外，您也可以更改高级照明设置或隐藏/冻结一个或多个选定层。

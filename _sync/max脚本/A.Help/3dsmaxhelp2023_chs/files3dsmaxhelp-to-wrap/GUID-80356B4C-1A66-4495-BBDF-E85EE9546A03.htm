手动设置优先级（HD 解算器）

一些模型和设置动画的运动不能完全对齐“子->父”或“父->子”优先级。在这种情况下，您可以基于关节手动将优先级值指定给 IK 链中的任何对象。

例如：

  • 以灵活的轻关节和带有阻力的重关节的组合设置模型动画。想像这样一个模型：重铁球与一定长度的链子链接在一起。将链子的优先级值设置为高于铁球的优先级值可以模拟球对运动的惯性阻力。
  • 设置某些关节必须在其他关节之前移动的运动。想像一下，高尔夫运动员的手臂，在摆动高尔夫球棒时肘部保持锁定。这可以通过将肘部的优先级设置为低于腕部和肩部的优先级完成。

高优先级值在低优先级值之前计算。将按照“子->父”顺序计算相等的优先级值。

优先级计算只考虑相对 IK 值。这意味着三个对象的优先级值为 0、30 和 200 的 IK 链与优先级更改为 1、2 和 3 之后的 IK 链拥有相同的解决方案。

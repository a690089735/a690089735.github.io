Detailed Description

Descriptor for a complete sampler state.

This class encapsulates a complete Sampler state.

Examples:
apiMeshShape/apiMeshGeometryOverride.cpp, customSpriteShader/customSpriteShader.cpp, fileTexture/fileTexture.cpp, hwPhongShader/hwPhongShader.cpp, simpleNoiseShader/simpleNoiseShaderOverride.cpp, viewDX11DeviceAccess/viewDX11DeviceAccess.cpp, viewRenderOverride/viewRenderOverrideOperations.cpp, viewRenderOverridePointLightShadows/viewRenderOverridePointLightShadows.cpp, and viewRenderOverrideShadows/viewRenderOverrideShadows.cpp.

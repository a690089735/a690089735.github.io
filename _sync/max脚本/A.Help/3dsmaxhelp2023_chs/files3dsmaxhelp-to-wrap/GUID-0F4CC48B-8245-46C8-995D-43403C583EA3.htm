贴图和明暗器

使用贴图通常是为了改善材质的外观和真实感。也可以使用贴图创建环境或者创建灯光投射（请参见“高级效果”卷展栏）。

贴图可以模拟纹理、应用的设计、反射、折射以及其他的一些效果。与材质一起使用时，贴图可增加细节而不会增加对象几何体的复杂度（但置换贴图除外，它可以修改几何体）。

访问贴图类型

使用材质/贴图浏览器加载贴图或者创建一个特殊的贴图。在 Slate 材质编辑器中，默认情况下，该浏览器是一个始终可见的面板。在精简材质编辑器中，如果编辑器处于贴图级别，则单击贴图按钮 （获取材质）或“类型”按钮时，该浏览器将打开。

如何移除贴图

要从材质中移除贴图，请使用“Slate 材质编辑器”。在活动视图中，可以断开关联，删除关联，或者删除贴图本身。请参见移除贴图或贴图连接。

贴图和贴图坐标

每一个贴图都拥有一个空间方位。将带有贴图的材质应用于对象时，此对象必须拥有贴图坐标。此贴图坐标是以 UVW 轴表示的局部坐标。

多数对象都拥有“生成贴图坐标”切换。启用此选项可提供默认贴图坐标。如果对象有此切换功能，在进行场景渲染时，将自动启用此默认贴图坐标，或者使用在视口中显示贴图。

某些对象（如可编辑的网格）没有自动贴图坐标。对于此对象类型，使用 UVW 贴图修改器为其指定一个坐标。如果指定一个使用贴图通道的贴图，而没有对对象应用“UVW 贴图”修改器，此时，渲染器显示一个警告，其中列出需要使用贴图坐标的对象。也可以使用“UVW 贴图”来更改对象的默认贴图。

UVW 贴图坐标通道

每一个对象可以有 1 到 99 个“UVW 贴图”坐标通道。默认贴图（来自生成贴图坐标切换）始终为 UVW 1。UVW 贴图修改器可向上述任意通道发送坐标。

材质中的每个贴图都可以使用 UVW 通道（如果存在），其他贴图类型也可以使用 UVW 通道，无论是 2D 贴图还是 3D 贴图。

在创建中或修改参数中，可以设置由 NURBS 曲面子对象使用的贴图通道。

为 2D 贴图进行贴图

使用贴图通道、任何指定的顶点颜色或局部/世界坐标系统，可以将一个 2D 贴图放置在对象的表面上。也可以选择不同的环境贴图。请参见“坐标”卷展栏 (2D)。

为 3D 贴图进行贴图

使用贴图通道、任何指定的顶点颜色或局部/世界坐标系统，可以将一个 3D 贴图放置在对象的体积中。请参见“坐标”卷展栏 (3D)。

贴图的噪波

随机噪波值增加了贴图的复杂度但是却使其看起来更加自然。对于 2D 贴图，请参见“噪波”卷展栏 (2D)。对于 3D 贴图，可以指定一个独立的“噪波”贴图。请参见噪波贴图。

本节内容

  • 贴图和贴图坐标

    贴图坐标指定如何在几何体上放置贴图、调整贴图方向以及进行缩放。

  • 真实世界贴图

    真实世界贴图是另一种贴图范例。真实世界贴图的想法是简化应用于场景中几何体的纹理贴图材质的正确缩放。该功能可以创建材质并在“材质编辑器”中指定 2D 纹理贴图的实际宽度和高度。将该材质指定给场景中的对象时，场景中出现具有正确缩放的纹理贴图。

  • “输出”卷展栏

    应用贴图并设置其内部参数后，可以通过调整它的输出参数来确定贴图的渲染外观。

  • “缺少贴图坐标”对话框

    尝试使用其贴图使用非通道 1 的贴图通道的材质渲染一个或多个对象时，将显示“缺少贴图坐标”对话框。（通道 1 除外，因为在向对象指定贴图材质时，会自动启用通道 1 贴图坐标。）只有已切换为“在视口中显示贴图”的贴图才被列为缺失。

  • UVW 移除工具

    使用“UVW 移除工具”可以将贴图坐标或材质从当前选定对象中移除。

  • 将贴图和环境用作材质组件

    使用材质的颜色组件（如漫反射颜色）中的贴图可将贴图的颜色应用于该组件。将贴图应用于其他类型的材质组件会具有不同的效果，例如，使材质看起来凹凸不平，控制材质的透明度，控制高光的外观等等。也可以将环境用作贴图以模拟反射或折射，或将背景提供给场景。

  • 2D 贴图

    2D 贴图是二维图像，它们通常贴图到几何对象的表面，或用作环境贴图来为场景创建背景。最简单的 2D 贴图是位图；其他种类的 2D 贴图按程序生成。

  • 3D 贴图

    3D 贴图是根据程序以三维方式生成的图案。例如，“大理石”拥有通过指定几何体生成的纹理。如果将指定纹理的大理石对象切除一部分，那么切除部分的纹理与对象其他部分的纹理相一致：

  • 合成器贴图

    合成器专用于合成其他颜色或贴图。在图像处理中，合成图像是指将两个或多个图像叠加以将其组合。

  • 颜色修改器贴图

    使用“颜色修改器”贴图可以改变材质中像素的颜色。

  • 反射和折射贴图

    这些贴图将创建反射和折射。

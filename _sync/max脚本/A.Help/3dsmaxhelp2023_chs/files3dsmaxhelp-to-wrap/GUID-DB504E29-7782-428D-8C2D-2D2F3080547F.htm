轴转化

FBX 导入器在 UI 上未提供“轴转化”选项。仅导出器中的 UI 显示此选项。这是因为在导入时 3ds Max FBX plug-in自动计算和应用“轴转化”。

自动轴转化的作用是什么？

本导入器查看主应用程序中使用的 up 轴。在 3ds Max 中，通常为 Z-向上。然后导入器将它与传入文件中的 up 轴进行对比。如果两轴匹配，则导入器不应用转换。如果它们不匹配，则导入器会自动转换导入数据，使它匹配主应用程序的 up 轴。

如果上方向轴不匹配，则导入过程的时间会变长，因为 3ds Max FBX plug-in必须将每个根对象旋转 90 度，才能与主机应用程序的上方向轴匹配。对于从 Maya 或 MotionBuilder 导入 3ds Max 的工作流尤其如此。

注意: 导入器仅对传入场景的根元素应用轴转化。如果导入时必须对根对象上的动画进行转换，则会对这些动画曲线重采样以应用适当的轴转化。为了避免对这些动画曲线进行重采样，请确保在导出到 FBX 前，在源场景（源应用程序）中添加一个根节点（虚拟对象）作为动画对象的父对象。

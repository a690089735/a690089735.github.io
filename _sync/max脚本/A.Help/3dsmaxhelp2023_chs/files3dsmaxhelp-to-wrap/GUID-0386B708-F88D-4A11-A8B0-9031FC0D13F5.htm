“交互式 IK”和“应用式 IK”

交互式 IK 和应用式 IK 是 IK 解算器方法的备选方法。

提示交互式 IK 和应用式 IK 是从 3ds Max 的最早版本开始就具有的功能。建议您首先熟悉 IK 解算器方法，仅当您发现 IK 解算器无法满足您的需求时才使用交互式 IK 或应用式 IK。

本节内容

  • 使用交互式 IK 设置动画

    启用“交互式 IK”和“自动关键点”按钮后，在关键帧上调整模型的位置，并在关键帧之间进行 IK 解决方案插值。由于 IK 解决方案对多个对象和对象间的关节均发生作用，因此使用了 IK 对象的插值动画通常与没有使用 IK 对象的动画不同。

  • 使用应用式 IK 设置动画

    应用式 IK 需要将 IK 结构的一个或多个部分绑定在一起，以设置跟随对象的动画。绑定之后，可以选择运动学链中的任何对象，然后单击“应用 IK”。该操作将会为动画的每一帧计算 IK 解决方案，并为 IK 链中的每个对象放置变换关键点。

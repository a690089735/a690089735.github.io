Detailed Description

Introduced in 2022.0

2022.0:
Introduced in this version.

Matrix function set for dependency node data.

MFnFalloffData allows the creation and manipulation of falloff data function. Falloff data are created to define or call a falloff function. User can create its own falloff by subclassing MFalloffFunction

Examples:
exampleFalloff/smoothFalloffNode.cpp.
